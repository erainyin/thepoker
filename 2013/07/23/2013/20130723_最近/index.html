2013-07-22 最近 [星期一 北京 晴]

发表于 2013-07-23 更新于 2022-01-27 分类于 红桃 - 文章

最近在数着日子过。是入伏后的第几天，或是立秋前的第几天。总之日子过得很快，也似乎过得很慢。在炎热的光景里，不免让人心生无限慵懒和寂寥。在过去多少个这样类似的时光里，我总有一种天长地久的恐慌之感，这个夏天也不例外。我无所事事，为此心神不宁。

罗老师病了。我们通过网络表示了慰问。看罗老师的微博，情况显得非常严重。据说是国图夜以继日的加班让有着“万年花城桂冠诗人兼丰台星座小王子”之称的罗老师也倒下了，还是上医院检查并要回家养病的那种。罗老师夜观天象，说水星要是再逆行对他极为不利，对于罗老师这样眼里不揉沙子的人来说，甚至有一赌气死了算了的危险。网上遇到罗老师，得知他老人家目前情绪还比较稳定。比起前几日“没有家人陪伴不能下地”的骇人，现在的罗老师也能够聊几句天，并表示继续为我国的民主化进程在网端大声疾呼，奋力转载。 祝愿罗老师康健起来。

游泳是我继滑板后在体育界另一个痛。这个夏天，这个痛点又被狠狠地戳了一下。

我递过一个价值10元的纸条，对歪在躺椅上的一个大叔说：您好，我考深水证。大叔接过纸条，说：准备好了叫我。然后继续躺在那里，他发福的肚子令我印象深刻，白白的，好像里面藏了一个熟透的瓜。我说：我准备好了。而后我跳进冰冷的水中，他的目光开始和我的身体一起在水中慢慢游走。到了泳池中央的时候，他站在岸边朝我喊道：你别测了，你根本不会游泳。你这是在走啊。我点点头，对他说：被你发现了。 我考深水证的游泳馆拥有大概是专业长度的水道，这让整个场馆看起来非常巨大。泳池分为浅水区和深水区。浅水区水深1.2米，所以你能想象将近1米9的我在里面的话会是多大一个笑话——旁边还有无数路都走不利落的小崽子。所以我硬着头皮又多花了10块钱，以考深水证的名义走进了深水区，深水区1.4-1.8米，我很顺利的走了两个来回，以至于自信到觉得可以去考深水证了。但结果告诉我，取得深水证的条件之一并不仅仅是完成全程200米，还有重要一点是“不能靠走的”。

虽然深水证没有考下来，但我失败后他们倒没有马上撵我走。我安安稳稳地在水里泡了两个小时，我觉得这是游泳馆的一个盈利的策略——你不可能跟那些小崽子争着用一个小的可怜的浅水池吧，但没有深水证你就进不来深水区。所以你就多花钱考深水证吧。反正你也考不过，并且——欢迎补考哦亲。

热。/ 小萌跟我说了一种午后最妙的饮品搭配：冰棍+啤酒。/ 务农 / 地铁以及魔法。 / 期待去采蘑菇。
