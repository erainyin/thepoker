随笔140628 [星期六 北京 晴]

发表于 2014-06-28 更新于 2022-01-27 分类于 红桃 - 文章

看了一个怒清新的科幻片《Her》（她），探讨了人工智能可能达到的效果，以及人类与人工智能之间发展处情感的一种可能。人类在追求精神层面的更高需求的时候，物质的肉体产生的需求变成了阻隔。说不好是非物质的人工智能更自由，还是拥有所谓现实世界的人类本身更为自由。 不，这个片是在探讨孤独。彻头彻尾的孤独。把感情寄托给人工智能看似可行，但谁知到人工智能最后智能到了都能追求自己的美好未来，和其他人工智能谈上感情了。人类的感情又无法超脱躯壳和人工智能融为一体，由此产生的隔阂、引起的思考非常有趣。人类在对待人工智能的历史中，不能不面对情感透射的问题。一方面人类希望创造出和人类一样的智能生命／非生命体，另一方面，没有人喜欢看到一个和自己一样的人，更不要说能和自己一样思考的非生命体了——他们拥有和我们一样思考的能力，却没有肉体无法持久生存的缺点。由此看来《Her》的观点是乐观的，允许人工智能拥有学习的能力，不遵循“机器人三大定律”，这样势必导致人工智能的不可控。 无论怎样，片子里给人工智能配音的女人声音很好听，后来看了一下，说就凭这个没有任何画面的角色，Scarlett Johansson还获得了个最佳女主角。 ＊＊＊ 最近在读两本书《瓦尔登湖》和《查泰来夫人的情人》。一本书教你如何逃离人世的纷扰，逃离物质生活的牵绊，做一个纯粹的人，简单的人，脱离低级趣味的人。记得看过一句话介绍一本名著的文章，说《瓦尔登湖》写的就是“一个人离开了城市在湖边盖了个房子，居住了一段时间，然后又回到了城市”。我刚看到他在湖边修建好了房子过着一种贫穷人都不太能接受的寡淡清贫却精神世界丰富的生活，还不知道他最后怎么就又回到了城市里——要知道，他一直在说生活在这样一种靠自食其力的劳动就能拥有的简单却过得去的生活是多么的美好，多么另精神放松，拥有一种超脱世俗的优越感的生活是多么的好。难道他后来变卦了？或是什么让他改变了主意？真想赶快往后看看。 而另一本书则告诉你男人和女人是如何思考性这件事儿的，以及在查泰来夫人的时代，生活仿佛一潭死水的时候，一个认为生命之火也即将熄灭的女人是如何抗争的。目前的感觉是，查泰来夫人是个热爱生活的好女孩。 摘抄：“个人的青春是不自觉的浪漫主义，文学的浪漫主义是自觉的青春。”——木心
