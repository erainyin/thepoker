断舍离

发表于 2019-07-16 更新于 2022-01-27

最近终于把笔记本电脑给二手处理了。小萌一直在问，觉得笔记本电脑占地儿——即使我把电脑弄到了公司?。除了占地儿，还想处理了换成印着领袖的红纸，或者微信余额上多几个数字。

本来想去年底就给二手买了。不过犯了拖延症。又加上杂事繁多（其实主要还是懒）。到今年六月才真正付诸实施。

其实之前一大段时间也做了一些工作，主要是将电脑里的数据备份。真是一个繁重的工程，关键是之前还装了双系统，但后来windows用不了了，里面的数据只能硬拷出来。为此还专门买了一个新的移动硬盘。

整理完成后，照片什么的还都舍不得删除。一直等到快递来收电脑前的几个小时才狠心按下了删除键。电脑数据删除，用户注销，等到快递小哥取走之后，望着空荡荡的桌子，心里还真有那么一丝不舍呢。

电脑是委托京东二手回收的。其实他们也是与一个第三方“爱回收”合作。在网上估价910元，这个电脑当年2011年是花了13000，这么算来每年折旧大概1300。这台笔记本用的时间已经很长了。为我的工作做了不少贡献。回收的过程还比较顺利。就是中间出了个小插曲。京东估价910，结果快递过去后，给我回复说780。本着苍蝇也是肉的精神，我赶快联系客服，说明我这个电脑基本没什么问题。估价和电脑本身的描述相符合，为啥他们检测完估价就低了那么多呢！打电话过去后客服还比较客气，没多久给我回复，说他与领导申请了一下，把回收价提到了910。问我可以不可以，我当然可以了。

电脑换成了钱。桌子腾了地儿。然后小萌说，咱家的旧吸尘器，你也给处理了吧。
