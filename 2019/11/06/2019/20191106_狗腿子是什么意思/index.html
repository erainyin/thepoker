爸爸狗腿子是什么意思

发表于 2019-11-06 更新于 2022-01-27 分类于 红桃 - 文章

小猴回家的时候问我：爸爸，狗腿子是什么意思？

呃……你永远不知道孩子会问出什么问题，而且是毫无准备的情况下。

“这个…狗腿子就是指那些跟着坏人做坏事的人……”我搜肠刮肚，“比如…比如有些人怕坏人，但是他们又不敢反抗，反而与坏人走的近了成为同伙，一起做坏事。”回答的时候一边说一边想“大概是这样吧？这么说没什么漏洞吧？”

正所谓最好的防守就是进攻，我追问小猴一句：“谁跟你说的这个狗腿子啊”

“是我们在幼儿园看的那个聚宝盆的故事里讲得”哦哦哦，这我才有些如释重负，“里面就有那四个人一直跟着那个人，说他们是四个狗腿子。”

“那他们跟着的那个人是好人还是坏人啊？”“坏人。”小猴说。“你看，我没说错吧。”

——你真不知道小朋友从哪些渠道获得哪些信息。
