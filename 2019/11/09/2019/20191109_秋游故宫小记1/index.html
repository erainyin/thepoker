秋游故宫小记1

发表于 2019-11-09 更新于 2022-01-27 分类于 红桃 - 文章

“是时候了……”尹大爷慢慢地喝上一口咖啡，又看了看日历上的数字，“就定……这周六吧。”

十一月第三阵秋风刮过的时候，天上的云更高了，树叶掉落的速度如我们的期待一样，一夜之间，遍地金黄。

人们都知道金秋十月是北京最美的季节。景色虽美但人也非常多。而人们却因此经常忽略十一月这个真正适合游览京城的时间。这个时候初冬的寒风刮走了秋日最后一暖温存，但还算不上凛冽。同时被刮走的还有金秋北京美景最后的骄傲——香山红叶。随着游人的退去，一个干净冷酷的北京回来了。

还带来了“旅游淡季”我的朋友们。那些敏锐如尹大爷的人们笑了，翻了黄历，看了天象，选了日子，携上妻儿、干粮和水。手指轻轻滑过手机屏幕，点开嘚嘚打车。不多时，一辆黄绿相间的小车来了，坐上去，向东南一指——师傅，天安门。

天安门是北京内城的正门，而我们的目的地并不是广场，而是它守护的皇城。长安街，这条横贯北京东西的宽阔街道因为红绿灯之多，让人有三十到四十分钟的车程，您可以拿出本本儿，或弹开那什么百科，熟悉熟悉北京的历史，因为走过天安门的安检开始，你所看到的一切都将触及你历史知识的盲点。三四十岁的人了，已懂得如何在孩子面前掩饰无知的尴尬，无非是笨鸟先飞而已。

故宫位于北京正中心，是昔日明清两朝的皇家宫殿建筑群。是拥有世界之最Title的世界文化遗产，而一般游客可能不知道的是，故宫可不归园林局或者旅游局管，而是文化部直属的事业单位，所以整个故宫官方叫故宫博物院，英文Palace Museum，足见其规格之高。并且因为其博物馆的属性，对外一视同仁，导游证也没用也得买票，大有“官员人等致此下马”的气势。别说导游，如尹大爷这样见过世面的人走过了金水桥后，在天安门高墙巨大的压迫感之下，膝盖都有些发软想跪下磕几个。幸好看到孩子他妈给他眼色，乖乖，差点丢了当父亲的和祖国接班人的老脸，心理念叨着“大清早亡了，现在是新中国了”便挺直了腰板，朝军姿赛华表的武警哥哥点头致意。

然而武警哥哥并没有回礼。

军威如山，不怒自威，佩服佩服。尹大爷心中默念着穿过了大门。儿啊，这天安门前各有两座华表，一曰“望君归”，一曰“望君出”。意思是皇帝老儿你不要出去太久也不要宅太久，要平衡工作和生活。走进这天安门就算进了皇城，谁知前面不远处又一大门迎面而来，也是红墙黄瓦，上面“端门”二字清清楚楚。幸好没有脱口而出“这就是午门”，尹大爷清清嗓子，“儿啊，这是端门，端门后面就是……呃，咳……午门了。” 午门才是紫禁城的正门。此时距离排队安检进来已经过了40分钟。

对于故宫这样体量的景点，提前规划路线十分重要。常规路线是前后三大殿+御花园的中轴线路线——这是大多游客的选择。沿着从前面到天安门一路向北，沿着龙脉穿越北京的中心。如果还有体力，还可以继续后面景山北海+什刹海的皇家园林路线。不过对于北京故宫的老朋友，尹大爷更多的会关注天象，春日里必定选择东路，看绿树抽芽，和满地的芳草，秋日里可走走西路，看红墙黄瓦映衬着秋日的肃杀。他还知道故宫最美的景色属于两个极端的天气，一是瓢泼大雨的时候看太和殿广场的千龙吐水，一是冬日大雪纷飞之时享受雪锁皇城红白相应的绝美，二者皆是可遇不可求，能否看到全凭运气。所以，往来的游人们啊，不要感到遗憾，因为久居京城的尹大爷同你们一样，也一个都没见过。

因为是带着孩子，最终路线变得简单了很多。午门进，看内金水桥，进太和门，看太和殿广场，远眺太和殿。看中和殿，看保和殿。
