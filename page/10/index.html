种草拔草

发表于 2023-08-05 更新于 2023-12-20 分类于 红桃 - 文章

我儿时吃过几种非常美味的药。块状的板蓝根是其中之一。众所周知板蓝根是感冒发烧时的常用药，在当年非典时被奉为神药，虽然后来人们冷静下来说可能安慰剂成分更大些，但我们不能不对中国传统医学保持敬畏之心。我就是要说，板蓝根就是很好吃。

而且我小时候还吃过一款方糖造型的板蓝根，其形质模拟巧克力，而且比普通的板蓝根冲剂更甜美，应该是配料里加了足量的红糖。众所周知，红糖也入药，一起吞服，没毛病。

轻轻得知有这么一款好药后很羡慕我，但这毕竟是药。于是我给他推荐了另一款似药非药的好药——大山楂丸。大山楂丸是同仁堂出品的一款健胃消食的良药，以山楂为主料，辅以红糖，蜂蜜揉成一颗棕色大蜜丸。捧在手里分量压手，含在嘴里适口。和方块板蓝根、果味儿VC构成了童年药膳美味铁三角。

听我说完青青一直以为是我编的，一直在问真的假的。为此我打开同仁堂网站，官网上一些知名良药里赫然有着发光的四个大字“大山楂丸”。您瞅瞅，黑纸白字，有，并且可喜的是，现在还有卖的。

今天去中华民族园。门口有一个旅行团扎店的同仁堂药店。什么叫做来都来了，什么叫他还是个孩子，既然孩子来都来了，走过路过就不错过了。青青渴望得问道，妈妈我能去同仁堂里看一眼吗？当然可以。一问还真有，16元一盒。妈妈看到青青喜出望外的样子，还没让青青自己花钱买。

晚饭之前，青青拿出一粒，惊呼，果然大！一口塞进嘴里，咀嚼良久，惊呼，真好吃。

植物园野采

发表于 2023-08-03 更新于 2023-12-06 分类于 红桃 - 文章

北京大雨过后所有人都应对自然产生敬畏。大雨当天北京全境的公园都闭园了。即使几天后，植物园的樱桃沟景区也是不开放的。因为离山太近，据说香山上的积水还没流完，爬山大概率会变成溯溪。

但我们还是前往了植物园。之前对于植物园我有个很高的评价，就是「没有选择的选择」。是拖底的，实在不知道去哪里的时候去植物园走走肯定没错。植物园有山，有水，有自然，有人文，关键还离家近。

今天出来的时候已经到了下午，温度36度，湿度70%往上，体感温度超过了40度。虽然没有预报有雨，但天空被很厚重的云覆盖。这也就是湿度大的原因。我们的主要目标是想去抓几只蝗虫，最好是棉蝗。之前有在卧佛寺附近被我们称之为大王山的路段上抓到过巨大的螽斯（被我们亲切地称之为大王），这次也希望有所收获。植物园为了保护植物，整体上用了很多驱虫方法，所以能抓到的野生昆虫少了很多，不过上次还抓到了个体很大的很漂亮的白星花金龟，让人对这里还是抱有期待的。

这次我们走了一条之前没有走过的路线，绕澄镜湖走过碧桃园的东侧。目前碧桃园正在重修，围起了好大一圈，等走过去之后，我们发现已经绕到了景区东北侧，不过离大王山不远了。这一程我们抓到了一只中华大刀螳，还有几只知了。此时天非常闷热，我的衣服几乎湿透，整个旅程伴随着巨大的蝉鸣。这一切的感觉简直不能再夏天了。

就在快到大王山的时候，我们竟看到一只肥硕的野兔从树林间跳了出来，它似乎发现了我们，三下两下就蹦出去十来米，最终消失之前还回头看了看我们。这一切发生在短短的几秒钟之内。小萌看了惊呼，哇！那么大的兔子！我要捉一只！

大王山名不虚传，看来我们的判断没错，虽然没有巨大棉蝗，也许时机还未到。但我们发现这里有非常多的剑角蝗，就是那种通体绿色，名字很嚣张，但实际笨笨的蝗虫。他们弹跳力不错，但不怎么飞，随便从草丛里踢一脚，就会蹦出好几只。然后轻轻松松就能网住好几只。最终我们把它带回家，顺路窄了点干草。几个大哥也不拘束，看见有草猛猛吃。

钓鱼佬永不空军

发表于 2023-07-09 更新于 2023-07-18 分类于 红桃 - 文章

从网上学了如何制作钓鱼神器，爸妈儿子齐上阵，一人做了一个。厄尔尼诺你大爷四十度高温的下午前往园博园进行实验，看网上的教程是不是骗人的。本着钓鱼佬永不空军的宗旨，做好了下河徒手捞鱼的准备。但神器之所以称之为神器，从来都是结果导向。上鱼了，就是神器，没上鱼，就是狗屎。

执着的精神不仅能抵抗烈日，更能召唤友谊。甚至好久不见的李大班都带着弟弟莅临园博园，和我们一起等待渔获。钓鱼神器中放置了面包，坠入河底，一段时间之后，向上一提，还真有鱼上钩了。不大，但不少，几个来回居然收了十来条。要不是旁边有个抽水泵一直当当当响，惊了鱼群，定收得更多。

环球特种兵

发表于 2023-07-06 更新于 2023-07-11 分类于 红桃 - 文章

据说现在有种时髦的玩法，叫特种兵式旅游。这么时尚的玩法我们不能错过，今天孩子他妈就带着孩子前往环球影城一探，结果您猜怎么着，特种兵不止咱一家啊！今天可又是一个近40度的高温！厄尔尼诺你大爷发功，让你们这些特种兵再多点谈资。溜溜一天，早上6点就出门了，9点之前抵达环球影城排队，一直到晚九点。累成狗。

不过应了那句老话“来都来了”，于是结果就是该玩的都玩了。去之前还看着攻略想是不是升级个年卡，到时候想去随时去。结果有了这一天的体会，这么计划的在北京城高低几万人，不是，是几万户人。于是早上9点开门，刚排了40分钟才进门的小萌发我短信：今天可能得晚点回，因为不打算升级年票了，能玩多玩玩。人太他妈多了。

虽然第一个霸天虎过山车把小猴吓得够呛，用妈妈的话说：脸都吓白了，差点哭了。

端午

发表于 2023-06-23 更新于 2023-06-26 分类于 红桃 - 文章

我就不信治不了你！

今天北京红色高温预警，温度达到了43度。家里的温度还要高10度。现在是下午4点20，青青边哭边准备下棋。已经输了4盘棋，丢了几十分。妈妈的要求是把分追回来。已经将近五个小时了。

第一盘的时候下的非常不认真，一边下一边抠鼻子抠手抠脚，然后惨败。然而第一盘是一个1800分的高手，让对手赢了5分。第二盘本来有机会追回来，结果又没下好，还认输。一下分就拉开了20分。

然后就是大哭，妈妈大生气，俗话说胜败乃兵家常事，但故意认输就不能忍了。一气之下妈妈说，今天说什么也要把分追回来。青青发了半天脾气，终于回到棋盘上，下了一盘表现分一般但赢了的棋。本来以为第四盘会乘胜追击，结果，结果在比对方时间多20分钟的情况下被将杀。彼时我从卧室出来，都不敢出大气。屋内气氛极为严肃，只能听到风扇的声音。

完蛋，我心想。

又是大哭一场，青青说不想下棋了，再也不下了，妈妈怒气值也到了顶点，放狠话说，下！必须下！今天不把分追回来别睡觉！然后一边运气一边摆棋。青青大哭不解气，用手拍棋盘，弄乱棋子，说不下不下。

好！你要是不下，咱们就再也不下了。妈妈拿话激他。他当然是大哭着说不，你说的不对！就这样僵持了二十分钟，才逐渐平静下来。中间我也过去和稀泥，希望妈妈不要气到动手。

现在已经到6点12分了，还在下棋，妈妈的要求是下到表现分是75分以上。

最终闹到了晚上9点，一共下了12盘，4胜8负，丢了100多分，妈妈说明天要都赢回来。青青崩溃了。我在旁边溜溜一天旁听，听的我心惊胆战，头晕脑胀。想赶快过去吧这一天，太难熬了。

大风天

发表于 2023-04-11 更新于 2023-04-12 分类于 红桃 - 文章

北京的春天总让人感觉一体两面矛盾统一。

你永远不能相信某一种感觉会持续很久，比如就在清明节前两天，北京下了一场不大不小的春雨，春雨过后碧空万里。甚至有人说在城里都看到了雪山——那座横亘于北京西北的海坨山居然下了雪，而当天傍晚托高能见度的福，人们又拍下了落日余晖金光遍地的城市景象。与此同时，春花繁盛，清风和煦。

但不到3天后，北京大风。吹走了蓝天白云，吹来了黄沙漫天。这种场景总让我想起电影《双旗镇刀客》，这些风沙肯定是从西北而来，往东南而去，此去经年万里。
