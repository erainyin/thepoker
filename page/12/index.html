大觉寺野采

发表于 2022-09-04 更新于 2022-09-06 分类于 红桃 - 文章

上次去大觉寺回家的路上小萌就说，等秋天咱们一定还得再来一次，到时候肯定有大货。

这不一转眼，秋天到了。

虽然周六下了半天雨，到了周日早上天还是有些云，让人担心中午会下雨，我们只好带上了雨伞——什么叫做未雨绸缪啊。我们不仅带了雨伞，还带了登山杖，登山杖，又叫打狗棒，因为之前走香山八大处的时候被野狗尾随出了心理阴影，所以这次我们有备而来。山，在哪里。我们来了。

从家打车前往大觉寺，路虽然不近但很顺，下了六环路走一会儿就到了。除了费用有些小贵（算上高速费100来元），但正如那些奢侈品广告说的一样——大觉寺除了贵，没别的问题。而且大觉寺是真有货啊，上次来顺利抓到几只鹿角花金龟，引来不少路人艳羡的目光。这次我们决定重走上次的路线，非常好，这个路线除了一段不是很难的爬升，到了消防通道就是一马平川了。

说一下最后的收获，这个季节是螳螂和蝗虫爆发的季节。我们也捉到了4只大螳螂，一只非常肥硕的蝗虫，棉蝗还是稻蝗我分不太清楚，但是真肥。而且我们还发现了大量的蜥蜴，很多刚出生的漂亮的蓝尾石龙子在墙上爬来爬去，享受冬眠之前的温暖和舒爽，还有足量的食物。以及足量的游人。

八月杂记录

发表于 2022-08-29 更新于 2022-09-10 分类于 红桃 - 文章

8月总结

夏日的炎热终于在八月达到峰值，所幸在八月的末尾结束。能够让我们享受处暑之后的一些凉风的同时，看着仍然在和热浪斗争的南方的朋友们。能做的最多的就是关上空调了。极限炎热席卷，全球共此凉热。希望疫情退散，气温降低，世界和平。

8月28日 卡游北京旗舰店圣地巡礼！

我知道这店就在西红门宜家那边，但谁知道就在宜家的正对面。本来说是去宜家逛逛的，结果那巨大奥特曼，那醒目的黄色logo，还有店铺里闪耀的灯球——这！这！这！小猴飞奔进旗舰店大厅，爸爸看！这么多奥特曼卡！都是新款！

结果？结果当然是满载而归，青青花了一半的零花钱。

晚上小猴问我：“爸爸，我花了这么多钱，可我很开心。是不是我开心就好？”
“嗯，花钱如果能开心当然很重要。总比花了钱不开心要好吧。”

回家后赶快把这个消息广而告之，给奥特曼的迷弟们都种一把草。

8月27日 国象升级

小猴参加石景山区区赛，是六级升五级的比赛，比七轮，要积3.5分能升级。上午4轮，下午3轮。经过近一天的鏖战，终于以4.5分成功晋级。第一轮我们看到小猴对阵一个小个子，不太起眼的样子。我们心说，稳了。谁知第一盘就输了。更谁知，这位小个子最后横扫七轮，以七连胜的成绩拿下他们组第一名！真人不可貌相，海水不可斗量！提升棋艺的道阻且长，慢慢修炼吧。

不过升级成功总算是一件喜事。广发捷报后就直接带小猴去了物美小天才电话手表的营业柜台。购置了一款价值1580元的高级电话手表。大概半年多前小猴曾经在这个柜台前驻足一个小时，心理斗争要不要自己花钱提前买手机——那时他的零花钱还只够买500元的电话手表。而妈妈承诺的是他二年级的时候给他买一款电话手表，不用动用他自己的零花钱。今天得偿所愿，小猴欣喜之情溢于言表。

“这个手机怎么关机啊？”我问。
“我知道！”小猴抢过手机，把手罩住屏幕，果然屏幕就黑了。
“可以啊！”
“这不算什么，果果早都教我了。”

8月21日 大食代！

今天安排了周末去美术馆陶冶情操。美术馆正好有一带一路主题的美术展，能看到一些西域和丝绸之路沿线国家艺术家的作品。甚至还看到了题为赛里木湖玫瑰的画作，赛里木湖我们刚去过，那可是大西洋最后的眼泪，赛里木湖的玫瑰？不就是姆们嘛！

今天路过东方广场，第一次带小朋友吃了“大食代”。小朋友被众多餐饮摊位惊呆了，一时间不知道该吃啥好。

新疆归来

发表于 2022-07-14 更新于 2022-08-22 分类于 红桃 - 文章

新疆是一场梦一样的旅程。当经历了烈日暴雨狂风和林间温和的阳光之后，你会在某一个时刻梦到自己是一匹马，在无边的草原上吃草，或者奔跑。

在此之前，我对于平坦不屑一顾。有雪线的高山让人望而却步心声敬意，有江河的谷地让人赞叹自然的奇特。而平原的一望无际总给我带来绝望的感觉，周而复始，循环往复，就这样一直延续下去。直到来到了新疆。

2022年暑假第一天，我们前往新疆。

启程之前

启程之前能否出行的问题就一直在困扰我们。很久之前就确定了新疆的行程，报了团，交了钱，就看政策是否允许出京了。预计是7月15日开团，也就是孩子放假的第一天，而几个月前的北京正在经历一轮疫情，能否复课都要看老天爷，就连我们小区都经历过一次封控。这一折腾就到了六月，孩子们终于开学。我们虽然也复工，但也开始了常规化核酸。每天看着疫情的数据只希望北京的数字稳定地降下去，另外也希望政策宽容一些，新疆能接收我们这些游客，至于核酸，您随便捅千万别客气。

终于到了7月初得到了组团方的消息，说新疆那边已经有了入疆的游客。只需要北京健康宝绿码加上48小时核酸，以及通信大数据没有经过中高风险地区的记录就可以了。于是火速订机票——乖乖，真他娘不便宜。因为距离出发时间太近，折扣并不多，再加上今年的燃油附加税增加，导致成本增加很多。而且此行千里，来回四趟飞机，所以光机票就花了近2万。

总之还是那句话，能顺利出发就好。于是订票后的几天我们同样是心怀忐忑，并且时刻关注着疫情数据和政策。并且在最近的一个周末都没有出门溜达，生怕大数据上新增风险区的记录。终于在小猴放假前一天我们一切准备就绪。根据旅行社的介绍，整整准备了每人12套衣服，从短衣裤到秋冬的厚衣服几乎把大衣柜装进了箱子。青青也异常兴奋，说我要去新疆喽。

Day1

伊宁

出行之前我对新疆一无所知，甚至不知道落脚的第一站伊宁是何处何地，只在出发前一天草草看了看手机地图——原来如此之远，偏隅新疆以西，难怪从北京飞到伊宁连转机花了6个半小时之久。中间还落脚了克拉玛依机场——一座因油田尔诞生的城市。机场也非常小，从飞机下来我们就已经感受到了西北阳光的炽烈和晴朗。

伊宁机场出来，天特别蓝，阳光倾泻下来，对比度比北京高不知多少。炎热但并不闷热。从阳光下走到树荫下，能感受到通透的清爽感。北京刚刚入伏，即将进入桑拿天的状态。大家都非常兴奋，来新疆就算是避暑了。

接风的第一餐吃的就上了羊肉——出行之前我就期待过，这次去新疆，得吃点正经羊肉。这第一餐就上了胡萝卜炖羊肉——是真羊肉，还有两个颜色的胡萝卜。这个我懂，手抓饭都少不了红胡萝卜和黄胡萝卜。当然还有土豆和皮牙子，都安排了。当然，还有传说中的夺命大乌苏。

在上飞机之前我们被告知飞机上没有吃的，只有零食，所以买了一些零食上了飞机。谁知道中途不只有零食，还有正餐。算是惊喜了。下了飞机虽然不是特别饿但是新疆第一餐，不能含糊。大家吃的都很开心，不过我们也不敢吃太多，后面还有十来天，天天要是这个规格对肠胃是不小的挑战。

提前仨礼拜就碳水减半了，然并卵，今天一顿都吃回来了……
小萌如是说。

三泡台

一进餐厅就看到门口的茶，说是特色的「三泡台」。门口的玻璃茶壶里是复杂的组合：茶叶、杏脯、桂圆、红枣、菊花、枸杞还有黄澄澄的一根巨大的冰糖。小猴看了又惊又喜，到了座位就开始盘算，喝了三大杯，说好甜好甜。吃完了临走还去问服务员阿姨多少钱一袋。我们赶快拦住，这可是第一天，购物什么的咱们往后放放。

卡瓦斯

卡瓦斯几乎是每个新疆餐馆都售卖的饮品，有不同的口味。这种饮料在东北叫格瓦斯。喝起来就像加了气儿的啤酒味黑加仑。树莓和桑葚味的很好喝。一路走下来都有售卖的。无论对于不喝酒的穆斯林同胞，还是不能不喝酒的蒙古同胞，这个都是很好的日常饮料，尤其在炎热干旱的地方，喝上一杯抬头看天，天真她娘蓝，阳光真好。

上大巴前往酒店

这次出行的团友都是泡泡老师的老朋友，并且都是我们之前参团见过的小朋友及家长。也因此不用相互介绍破冰，小朋友们一上来就开始了交流，小猴拿出他的奥特曼卡，在餐后赏了一波卡。得知有小朋友也带了卡册后很高兴，没想到赏卡后还能换卡。现在唯一的愿望就是能到新疆的超市里转转，看看有没有奥特曼卡卖，最好是有不同语言的奥特曼卡。

时差

吃晚饭的时候是下午六七点，对于新疆本地人，这时候吃饭至多能算下午茶。因为新疆虽然也使用北京时间，单因为从北京向西跨越了两根经线，所以客观上是有大概两个小时时差。再加上伊宁纬度比北京向北了不少，日落就会更晚些。所以在这个季节，新疆要完全天黑，大概要到晚上12点以后。人家晚餐时间真正是九十点钟才开始。

Day2

酒店入住

补充一下酒店入住的流程。行程一开始导游就说，新疆是防疫的天花板，之前北京有疫情的时候，新疆是绝不允许北京人入境的。不管你是不是高风险地区来的。整个就熔断。这么坚持了几年，到了今年政府终于开放了一些跨省旅游，我们也托此福风，能得以成行。所以导游也提前打预防针，告诉我们此行中可能有若干关卡，查身份证查健康宝，查核酸。而且酒店都会严格按照要求进行核查和拍照。所以我们到达酒店后，也是所有核酸、报备的证明以及新疆本地健康宝都查了个遍。花了不少时间才顺利入住。

酒店早餐

小朋友早餐吃了一堆葡萄干炒米，我吃了一碗牛肉碱面，有个哈萨克族的酸辣面筋太好吃了……
小朋友们不用破冰，甩出几张凹凸曼卡，眼神就对上了。
小萌如是记录。

第二日的早餐是九点开餐，按照时差计算，实际也就是7点来钟。小猴装了一盘子葡萄干炒米，还有一块新疆油饼——话说这新疆油饼有点意思，也就有北京薄脆十分之一的大小，呈菱形，倒是有些厚度，吃起来，也不算外焦里嫩。这里按下不表——到后来我们才知道这油饼不简单，给我们新疆之行留下了最美好的记忆之一。

大西洋最后的眼泪

导游说：现在我们倒计时，等数到0的时候，我保证各位会欢呼起来。

于是在即将穿越最后一个隧道时，当阳光从洞口照入，一片蓝色映入眼帘，即使大巴在飞驰，但我们已被赛里木湖的壮美征服。天气非常给力，湖面上空一片碧蓝，但周围的远山上有隐约的云雾缭绕。不时还能看到飞鸟在辽阔的湖面上空翱翔，难怪导游保证大家都会惊呼。但我觉得惊呼不够，看到这样的美景，简直就是让人欢喜赞叹。

赛里木湖是新疆海拔最高、面积最大的高山湖泊，“赛里木”哈萨克语为“祝愿”，而蒙古族牧民则亲切地叫他“赛里木淖尔”，意思是“山脊梁上的湖”。赛里木湖古称“净海”，又被称为三台海子。湖面海拔2071米，东西长30公里，南北宽25公里，面积453平方公里，湖水清澈透底，透明度达12米。
链接：https://zhuanlan.zhihu.com/p/27278054

游览赛湖之前有两个点值得记录一下。一个是从伊宁到赛湖有几个小时的车程，车程的最后我们途经了著名的“果子沟大桥”。果子沟大桥是架在两座山之间的大桥，据导游介绍，此桥建造难度巨大，因此造价非常高，平均一米花了几百万，而且才刚全线通车没几年。果子沟大桥修建之前入过开到赛里木湖，需要走一条非常颠簸的国道，道阻且长。而今天我们能够舒舒服服地通过大桥到达塞湖实在感叹国家基础建设的超能力。

我们先是很远就看到了大桥，随着道路爬升盘旋到了大桥之上，从那个高度俯瞰让人两股战战，真是天堑变通途。而人们建造了这巨大的桥体就是因为前面的隧道终点就是绝美的塞湖。当我们看到广阔碧蓝的湖水的时候，觉得这一切的辛苦建造都是值得的，都是值得歌颂的。

又经过了半小时的车程，我们抵达了赛里木湖的东门。这短短的半小时车程，已经让我们饱览了美景，NONONO，这怎么能看够呢，从远处就能看见泛着不可思议蓝色光芒的湖面，然后是后面的雪山，我们在雪即将消失的季节抵达赛里木湖，面对那云母般光泽的雪色赞叹不已。

小插曲

本来我们预定的酒店是在赛里木湖景区里面的，也就是说，吃完了就可以在塞湖湖畔漫步，一直呆到满天繁星。但是因为特殊的原因，有一位重要的政治人物来访新疆。并且会到访赛里木湖——也因此导致在出行前三天我们得到消息，所有塞湖景区里的酒店都不能入住了，已经预定的客人都要迁到景区外面。我们也不能免俗，毕竟这位客人实在是太尊贵了——正如未来几天我们在电视看到的——新疆人民热烈欢迎习大大视察新疆。

拌面

游览赛里木湖之前简单吃了一顿，就在景区的餐厅里吃的拌面。

心心念念的京西古道行走

发表于 2022-07-10 更新于 2022-07-12 分类于 红桃 - 文章

终于，终于，终于在几周不见天日的阵雨天气之后，天放晴了。本来计划上周前往的京西古道终于可以去了。京西古道位于有着北京西大门之称的门头沟区，永定河畔有一片微微起伏的丘陵。在这片丘陵之上，有着百年之前骡马经过踩踏出的一条印满蹄涡的山路。

我们一直想找一个距离不太远的捉虫目的地，最好有山有水，道路又不太崎岖。小萌表示“我并不喜欢爬山，我就是喜欢找东西”。为此前一阵我们前往大觉寺，一天下来喜获甲虫数只，甚至捉到了颜值爆表的黄粉鹿花金龟。回来以后小萌表示“以后3cm以下的虫子我都看不上眼了”。从那以后就一直再想进山，希望在途中有只大兜甲撞到帽子上被我们捡回家。

大雨前植物园行走

发表于 2022-06-26 更新于 2022-07-11 分类于 红桃 - 文章

从前天开始，天气预报就说有大雨，昨天开始，天气预报开始蓝色预警，说有大暴雨。到了今天，天气预报精确到下午4开始降水概率达到50%，近期最强将于将于本日夜晚抵达京城。

我和小萌夜观天象，掐指一算，本来想去京西古道捉虫子，但下雨估计是去不了了，但还可以近郊走走，决定择日不如撞日，上午就前往植物园游览，植物园距离适中，进可上山，退可返城。于是早饭后就出发了。

什么叫去了就是赚了！什么叫做大自然的馈赠！诸位请看——


最给面子的是，您猜怎么着，下午4点的时候，天气预报取消了蓝色预警，说云变薄了。不过雨估计还是要下的，因为我们回家的时候，植物园已经闭园不让人进了。感恩。
