广西归来

发表于 2021-10-08 更新于 2022-01-27 分类于 红桃 - 文章

“每天清晨一碗粉，做个快乐的广西人”——小萌

螺蛳粉，广西料理の灵魂。身为一个精神广西人，从味蕾出发，抵达桂西。“太好吃了！”小萌边吃边感叹，“这个酸笋太香了”——青青也忍不住品尝了一口。将螺蛳粉的灵魂注入体内，从此成为了拥趸，“妈妈，您能给我再夹一根酸笋吗？”

“咱们选的路线是活神仙套餐吧！”——我

这时的北京气温正在十度上下，下着自夏末就几乎没有停过的小雨。而我躺在几千公里外的骄阳下，还嫌有些太晒了。屁股下面的竹筏顺水慢慢滑行，不是顺水，而是有个戴着斗笠的服务生在为我们撑船。“孩子”我对那个服务生示意，“能否帮我打开这瓶香槟？”“当然可以，先生。”服务生语气里显得有些不好意思——这本应是他分内的事儿，让客人提出了要求基本上就是一种失职——“砰”地一声，冰镇的香槟冒出凉气，倒入杯中，我拿了一块腰果布朗尼蛋糕，就着香槟慢慢吃着。周围一片绿色，百万年前这里还是海底深处，现在被雨水冲刷的山成为了美景，一条条河流蜿蜒而过，我们在上面慢慢漂着。以一种无所事事的状态和时间作对。真是他娘的惬意，当神仙也就不过如此了吧。

“我还没玩够呢，不想回去，呜呜~”——小猴

一觉醒来，天已经黑了，也许是在平流层，天空的黑显得更加纯粹。机舱内光线暗淡，头顶只有零星的阅读灯开着，晚餐已经结束了，饮料也发过了，返乡的航程过半，这个时候显得最为难熬。青青睁了睁眼，感觉怅然若失。这个早上他才刚从名为lux的酒店的大床上醒来。lux，奢侈也。奢侈到大床正对着巨大的落地窗，外面的群山上太阳慢慢升起，阳光从山后射出，告诉人们这就是丁达尔现象。光线穿过胶体形成的效果——而眼前一脉绿色似乎早已醒来，虫鸣，鸟叫，和这空间里眼前的一切告诉孩子们，珍惜眼前的时光吧。美好总是显得短暂。也正因此青青满怀委屈哭了出来，时间让他和这美好的景致分开，还有那个奢侈到能自动弹开加热的马桶盖的酒店。他的屁股已经适应不了冰冷的坐便器了。

“你们北京来的就是不一样，小朋友讲话都那么有水平！”——划船的广西大爷

水时快时慢，而影响水流速度的并不是水本身，而是地形。如果地形平缓，你能看到清澈见底的溪水，飘着若有若无的水草。而高低落差的水面下往往也有大小不一的石头，让水流湍急，充满旋涡。在一个个因旋涡推动下，我们的橡皮筏子摇曳不定，即使已过六旬的船夫大爷经验丰富，也是勉强支撑着才控制了船速和方向。直到再一次水速变为平缓，我们才抬头看这永远看不腻的景色。真是太美了，广西特有的卡斯特地貌形成的山，巨大而丰茂的植被，各种叫不出名字的花草树木，蝴蝶、蜻蜓四处飞舞。“妈妈，这就是绿水青山，就是金山银山吧。”青青情不自禁地说出了刚学会的口号——拖小学一年级德育老师的福，社会主义核心价值观小猴熟记于心，当然还有其他口号。这话是有感而发，让船夫大爷听了不免称赞了一番。

纪念

发表于 2021-09-12 更新于 2022-01-27 分类于 红桃 - 文章

“你还记得咱俩第一次约会我跟你说的第一句话是啥吗？”

11号晚上我打着哈欠准备上床睡觉的时候，小萌神秘兮兮地跟我说：明天就是12号了哦。我没有反应过来，12号怎么了？“9月12号了哦”小萌又说，还笑。搞得我有些慌张，难不成有什么今天要做的事情我没有做？还是又把脱下要洗的衣服放错了脏衣袋？

小萌说，你不记得啦？咱们第一次约会就是20年前的12号！哦原来是这样啊，我这记性太差了——“你还记得咱俩第一次约会我跟你说的第一句话是啥吗？”——小萌一句话给我问蒙了，日子都记不清，说过什么话更是想不起来了。

“我记得可清楚了！”小萌说，“因为，前一天美国被炸了😂。”

看来我的记性二十年了来都没怎么变，改变未来20年的911我正为次日到来的约会紧张呢。穿什么衣服？说什么话？要不要表现得风趣一些？还是装傻表现得单纯点？总之还是先睡觉吧，养精蓄锐。管他世界怎么变化呢。

谁成想，当年劫机炸楼的恐怖组织塔利班，20年后，居然建国了。

小萌的记录

别的日子都记不住了，但第一次约会的日子记得最清楚———因为头一天是『911』。

本来头一天晚上我巨紧张，在宿舍坐立不安。我们宿舍都是没有恋爱经验的吃瓜群众，有的不让我喝水说怕脸肿，有的说让我准备一些问题进行摸底排查…………正忙活着，就听楼道里有人大喊“美国被人炸了！！”当时我们俩的牵线红娘正好从水房洗漱回来，抱着脸盆顶着毛巾翻着白眼进来说“傻逼！以为美国是你们家呢，说炸就炸了！哼！！”……后来才知道，那位同学说的是真的。 于是第二天我见到未来老公第一个摸底排查问题就是“你知道美国被炸了么？”￼￼￼
