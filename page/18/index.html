冬日戏雪1

发表于 2019-11-30 更新于 2022-01-27 分类于 红桃 - 文章

2019年第一场雪，值得记录一笔。

11月30日晚开始，31日白天雪停了，于是整个北京银装素裹，是个户外的好天气。按照老规矩，我们吃好了早午餐，准备好了零食和水，还专门带了挖土三件套，一路风尘仆仆，来到了植物园。

植物园果然是不负众望，虽然我们抵达的时候已经下午了，但是仍有非常大面积的积雪。比起城里大量的供热和汽车尾气导致转瞬即逝的雪景，这里的雪不仅触手可得，还漫山遍野。我们进到公园深吸一口PM2.5为0的空气，望着蓝天和远处的群山，开心得像三个孩子。

什么时候来，都觉得那么的漂亮。

来到“大草坪”果然如我们所料——这个雪场简直就是我们独享！放好座椅，整理好营地，直奔主题开始玩雪！

我致力于给小猴撘一个雪房子——一个真正意义上的igloo，小萌则热衷于搞个大雪球——“我要让小猴坐在这个大雪球上！”果然没过多久，一个巨大的雪球已经滚了出来。小萌说，你把雪球推到山坡上，让小猴推下来，我给你们俩拍照！真是好主意！但我推起来才感觉自己承受着牛顿第三定律的羞辱，也就半米多高的山坡，居然举步维艰。

我的双臂承受着巨大的压力，脚下却因为湿滑的草走的不稳。终于经过一番努力，将雪球推上了坡，小萌一句“青青，推！”我的工作成果啊，咕噜噜地就滚下了山坡。

“哎呀，没照清楚，你再推一遍吧。”
“……”

又推了一次后我已经满身大汗，而后我和小猴合力将雪球推下山坡，不幸的是雪球因为太重，滚下去的时候被颠碎成了几块——看到碎块我松了口气——终于不用再往上推了。我可以安心搭我的雪房子了。

恭王府之行

发表于 2019-11-24 更新于 2022-01-27 分类于 红桃 - 文章

如果说到北京人都不太知道的北京景点，恭王府应该算一个。不过恭王府却是很多外地人都知道的北京景点，为什么呢？因为他们在这里花过钱——听过导游孜孜不倦地给他们分析过恭王府的三绝一宝，然后心满意足地买了一个轴儿带走，那里面是康熙御笔据说是天下第一的福字儿。

是不是天下第一不知道，但肯定是全天下创收最多的一个福字。大概十几年前我带着外国朋友参观的时候，导游就说刻着福字的石头马上就要保护性关闭了，现在是抢救性地拓印了若干张，大意是买一张少一张了，买一张您赚一张，数量有限，卖完为止。买了是你的福气，不买是你的损失……

可惜当时恭王府内外有别，福字拓印这个产品大多只针对国内游客，这种好事儿怎么能让外宾占了便宜。外国朋友一般会引导到大戏楼，听上一段那北京Opera，喝上一壶热气腾腾的WoolongTea。当笑容爬满外国友人的脸颊，那带着惊喜和感叹的语气说出：“太香甜啦，请问这是什么神仙饮料？”——当然是用英语，顺道还问哪里能买到的时候，导游的脸上也洋溢起了笑容，您这边请——我们特意为每一位远道而来的异国的朋友准备了精美绝伦且倍儿有面儿且价格平易近外国人的茶叶礼包，请把它带走并把您的微笑留下。告诉您的家人和朋友，这是来自遥远的东方，中华帝国的首都，北京的中心往北再往东的荷花市场旁边的Prince Kung’s Mansion.

一晃，十年过去了。看到来此地游览的朋友们还是笑着抱着福字的画轴，我就知道，他们不会这么便宜了那块石头的。或许我应该心怀向善……或许真是采用了什么牛逼技术，3D打印之类的，克隆了那块石碑用来拓印也未可知。

恭王府是属于闹中取静的那类景点。和国子监的太学孔庙有相似之处。只不过前者外面是皇家禁地，后者外面是市井街区。恭王府的地理位置很好，但正因为周围都是北京地标，反而让人容易忽略这个胡同中的王府。而比起恭亲王奕欣的府邸的名号，“满清第一贪官和珅的宅子”更让人能记住。

之前来的时候这里还不叫“恭王府博物馆”，只是“恭王府花园”——彼时前半部分宅邸被中国音乐学院附中占用，没有对外开放。后来2006年恭王府重新翻修，便把王府本府也纳入进来，重新修缮后全部对外开放。所以这次游览简直是惊喜连连，整体感觉就是王府真是深宅大院，灰墙绿瓦，非常规矩漂亮。后面的恭王府花园更是植被茂盛，山水错落。

小猴一路走来玩的很开心。先以冰糖葫芦开局、吃完后游览恭王府，尤其对佛楼门口的转经筒产生了浓厚的兴趣，还捡到了小臂长的豆荚硬壳，带回家作为纪念。从藏宝楼走过去，穿过西洋门就是后花园。这高低错落的假山呦青青看到兴奋地说：“妈妈我要爬山！”于是一通登高远眺，到高处又惊呼：“哇，妈妈，有湖！”这次第，只见山下是一汪池水，水中央有座观景亭，四周绿树环绕，楼宇错落，长廊相连，色彩非凡。之间其中一间殿宇门口人头攒动，“妈妈，那是什么地方？”小猴问道。“那里好像是……卖纪念品的商店。”

“妈妈，我还有多少零花钱？”

装逼储备：蝈蝈的学名叫什么？

请大声朗读此字——

蝈蝈的学名称之为：螽斯 (zhongsi)

据说蝈蝈叫可能出于两个原因，一个是吃饱了撑的，一个就是想求偶了。也就是我们俗称的“饭饱思淫欲”。可惜这里只有他一个，只能往宅向发展了。不过他应该学会感恩，要不是在我们的陋室里，前两天七级大风刮一晚，气温飙至零下。多少个他向往诗和远方的同类估计都挂了。而他还能在暖房里这么开心的叫。

我都替他感到高兴。
