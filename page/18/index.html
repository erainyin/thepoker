为每一天留下一句(至少)

发表于 2020-02-27 更新于 2022-01-27 分类于 红桃 - 文章

小猴总算睡了。写两笔。

疫情仍在蔓延，这周公司终于结束了长达三周的居家办公，正是到公司复工，但并不是所有人都到公司，一部分是因为有些回京人员还在隔离期。另外公司办公楼的物业要求复工率不能超过50%，也就是说，座位得有一半是空的。为了迎接复工，公司也做了一些准备。每个人发了口罩、手套、抹布、安排了每日负责消毒的员工。然后把员工作为调整成Z字形，意思是你对面和旁边至少都空一个人的距离。还要求尽量全程带口罩上班。当然还有每天测几次体温。

我也不敢掉以轻心，以饱满乐观的热情做尽可能充分的准备。

每天早上洗漱完毕，整理好书包就开始上装备。帽子、口罩、护目镜、手套一应俱全，同时加上一件长款带帽子的雨衣。书包里装上了消毒纸巾、消毒液以及带的饭和水——本着尽量少接触的原则，自带午饭还是最保险的。除此之外通勤方面考虑到路途较远，只能狠心咬牙跺脚打车，还不敢打载客频率较高的出租车，斥巨资打专车，来回就是小150元，几天下来幸好目前打专车的人不是太多，而且路况很好，基本不太堵车。坐在车上的时候开着窗户，一动不动坐一路下车都一身白毛汗，可见我这装备的密封性很好。

到了公司为了喝口热咖啡，还得冒险去趟便利蜂。感谢自动咖啡机和在线支付，保证全程不跟人多说一句话。进写字楼测体温。通过后取电梯口的手纸一张按开关。开始考虑过走楼梯，但带着口罩一口气上九楼还是比较有挑战的。后来采取的方式是早点出门，所以这几天到公司坐电梯基本只是我一个人。

指纹锁是到公司后第一个不可避免地接触。摘手套后开门、进屋。然后开始消毒流程：先开窗通风，然后自喷84消毒液，把雨衣前后和手套书包外部都喷一遍。然后脱下雨衣放进塑料袋——专业上称之为“沤着”。然后脱下外套。用84给工位四周喷一遍。然后拿上水杯和抹布，去洗手间洗手漱口、洗抹布，回来后擦桌子。然后把带的饭放冰箱。这一番工作后，总算可以打开电脑，用湿纸巾擦手后，开始吃早点。

这段时间公司要求上午10点到下午4点办公，4点后都得下班，不得加班。终于让我们这些社畜看到了下午四点的北京的景象。

下班后会有值日生喷消毒液。而我要做的就是再次全副武装，在路人羡慕的目光下钻进一辆专车，捂一身白毛汗。到家后我这个全家最脏的人第一件事就是放下所有东西，全身喷一遍84，摘口罩脱下雨衣后直接进浴室、洗手漱口、洗眼镜，然后洗澡。至此才能亲吻各位并道一句晚上好。

每到此时，感觉有些恍惚。可能是口罩带了一天，有些缺氧。一时间摘了口罩，又有些醉氧。

二月二

在百度搜索日历的时候，你可以看到阴历二月初二这一天居然被称为“龙头节”——就叫龙抬头不好么😂。看到这里我都想当一把百度的产品经理，提出此处应该有一个交互，鼠标悬停今天的时候跳出一个水龙头厂商的促销广告。

全副武装

口罩是硬通货，戴上才显得不失身份，再加上一个护目镜，看到的都知道你是个严谨的人。当然最让路人暗挑大拇指的还是你那一件长款白色套头雨衣，实现了个人防护从玩票到专业的跨越。他们还没有看到你的手套哩——那副外卖附赠的半透明手套像世人展示的是你严谨而不失勤俭的风范，疫情之下仍不忘节约初心的中华男儿的形象。

路上

路上人并不多。但车不少。不限号，不（没人）收过路费，人们如果不是必要都不太出门——此刻光景下如果天气很好，且那个数字没有过分地增加，人们会做什么呢？我不太清楚。不过据说昨天前往香山植物园的路都堵了。

射击

对于枪械的记忆还是高中的军训。记不清教官到底说了啥，大概是一些枪械操作的基础知识，以及拿到枪后你要做的事情。其实也并不需要做什么，打开保险，瞄准，扣动扳机。然后突突突，50米靶子前面冒气一阵土黄色的烟——手里的六四式步枪还没摸热，我就可以起立走人了。0环。我也大可不必去争取进入民兵系统了（据说前几名能够成为光荣的民兵一员）。

所以几十年以后，重新拿起枪的时候，诚实的说我是有些胆怯的。这方面我的经验是0环，一切都得重头开始。人生中的又一个第一次啊。不过同行的团友里，貌似只有一两家人是有过射击经验的。所以和他们相比，我并非是0经验，只是0环的经验而已。

俱乐部的小哥似乎看出了不止我一个人有些紧张。几个女士已经开始争相咨询起来，到底哪个更好打出去，噪音更小。俱乐部的小哥指着桌上一个图片，上面画了各种枪械和子弹的对比图。子弹巨大的那一排基本没人选。其实我并不太担心噪音，主要是担心后坐力——所以我也没多问，跟着女士们选了两只小口径的手枪。然后犹豫了一下，选了一只自动步枪。

没想到小枪后坐力和噪音那么大，就当放鞭炮给大家拜年了。

2020春节小记3

发表于 2020-01-28 更新于 2022-01-27 分类于 红桃 - 文章

清迈小当家（厨艺学校）

今天上午小朋友们去上英语课，家长就只能勉为其难去参观清迈的菜市场，和上厨师学校学习正宗泰式料理。

如果要了解一个地区，菜市场是不错的选择——你要的市井感、烟火气这里都有。虽然我们到的有些早了，但还是看到了忙碌的当地人贩卖着各种见过的和没见过的水果和蔬菜。这个菜市场距离我们要去的厨师学校并不远，所以会先有一个厨师学校的老师教我们认识一些当地的蔬菜。

一个胖胖的皮肤有些黝黑头发卷卷的泰国小哥已在那里等候多时，用略带口音但流利的中文给我们介绍了很多种蔬菜。比如几种香草、几种番茄、几种葱姜、绿色的茄子、柠檬、香茅等等。整体的感觉是泰国本地的蔬菜种类繁多，光柠檬就有好几种，但普遍个头都不是很大。而且泰国人比较喜欢香气。反正到时候都是做汤，只取其味的话，就不用太考虑菜量了。

老师介绍完，留给我们逛的时间很短，我们买了几个莲雾和芒果，询问了柠檬的保质期后小萌买了一袋子准备带回国。结果后来被告知新鲜水果都不能入关。后来只好留在了酒店的冰箱里。

给小象洗澡

清迈旅游特色之一就是大象。因为靠近山区的缘故，这里有大片的森林适合大象生活，所以当地的少数民族也非常擅长驯养大象。有很多旅游项目都是和大象相关的。比如昨天参观的大象粑粑乐园。让我们体会到了泰北有象，其象有屎，屎之大，不知原来可作纸。今天我们来到了一个名为“HUG CHANG”的大象保育营参观。从名字就看得出来，这里是不会有什么骑大象翻山越岭或者大象表演之类的节目的，到这里来，您就是要放下身段儿，规规矩矩地面对他们——当今陆地上的最大的生物，森林和溪流的霸主，站在食物链顶端狮子身旁也不会畏惧的存在，兼具极高智慧和温柔性格的造物们——请表现出你的Respect，秀出你最好的厨艺和搓澡技术，总之，你要让他们感到快乐。


大象能有多聪明？当老师拿出一张大象画的画的时候，所有人都惊呆了。

老师首先给家长和孩子们讲解了亚洲象和非洲象的区别，以及大象的食物。原来大象除了吃草料还吃香蕉等水果。如果大象要吃点咸口儿的，就会整点红土，据说里面有矿物盐和维生素。于是我们每人领了一篮子香蕉，并且要给大象制作营养午餐。

所谓营养午餐，实际上是酸豆角红土泥巴球。考虑到吃土显得没什么身份，给外面包了一层黏黏糊糊的酸豆角果子，制作的时候小猴表示，这玩意儿拿在手里跟大象粑粑没啥区别。

当你费了很大劲做完一顿丰盛的食物以为能够取悦大象的时候，你会有一个惊奇地发现，站在大象面前的时候，食物并不是由你主动喂出的。你甚至不敢主动伸手，同时惊叹于象鼻的灵活。尽管小猴选了一只年幼的小象，小象好奇地伸出鼻子试探，小猴还是怂了，不敢伸手。当我把香蕉递出去的时候，小象马上将香蕉卷进嘴里。后来小象居然把香蕉卷到鼻子下面，要多攒一些才送到嘴里。Amazing!

2020春节小记2

发表于 2020-01-27 更新于 2022-01-27 分类于 红桃 - 文章

清迈（ChiangMai）

清迈是一座美丽的城市。

清迈是泰北第一大城市，泰国第二大的城市，也是清迈府的首都。位于泰国北部靠近山区，昼夜温差较大。纵观整个城市，没有很多的高楼大厦，即使是最繁华的古城周围，大型的建筑也仅仅是医院、大学和一些政府机构。整个城市背靠有着泰国最高峰的英坦昂山，又有宾河流经。风水好到兰纳王选中了这里创建清迈城。不知是不是从北方帝国学来了风水秘术，在城中最著名的大庙里还有一个据说是天神留下的石柱子。以此为基，整个清迈就此绵延开来，到今天，也修了二环路了。

清晨吃完早餐按照行程乘坐大巴出发，一路上满是绿意盎然，道路两旁棕榈树、椰子树林立，还有各种叫不出颜色的植物，深吸一口这南国的空气，湿润，清爽，沁人心脾。导游说现在是清迈的冬天，如果某年冬天云南下雪，冷空气吹到清迈，气温最低能低至0度。如果碰到这种天气，泰国人民会大呼着“我的老天”趋之若鹜，都会来到清迈享受难得的冷空气。

而这一切对于北方远道而来的客人来说，简直是不能更舒服的冬天了。

送孩子上大学

清迈的第一个景点距离酒店5分钟车程。经过一片藕荷色围栏围，一个有着巨大国王照片的路口，便是清迈大学了。刚刚还奇怪看到很多衣着白色上衣紫色裙子和长裤的年轻人在道路两旁，人头攒动。导游说：穿着制服的是清迈的学生，他们是在等二公主驾到。这一切就都说得通了。

行程的前四天上午小朋友都要在清迈大学的语言学校里和老师上英语课，这期间是大人们的快乐时光。虽然8点就开始上课感觉有点早，但是我们还是很开心的将小猴送到了学校。小朋友们就读于清迈大学语言学校，一个白色的三层小楼，上到二层后就看到了他们的教室。整个楼道干净整洁，楼道里有免费的饮水机。厕所也很干净。

经过一番介绍，跟老师SayHi之后，小猴被分到了小班。开始了第一堂清迈大学的英语课。分别的时候小猴很开心，并没有像第一天上幼儿园的时候那样，看来真是长大了。如此一来，我们就在导游的带领下去体验泰式按摩了。

泰式按摩

王宝强游览泰国曾留下一句：“小鱼给我做SPA”，由此可见泰国的养生业之闻名遐迩。这次旅程第一个成年人的活动就安排了正宗泰式按摩。地点就在清迈城里一处名为FahLannaSpa的按摩店内。按摩店装修非常漂亮，一个古朴的小门喷着水雾，进门之后曲径通幽。几个泰国美女袅娜而来，奉上毛巾和玫瑰茄茶饮。之前我还在纠结”我脚怕痒，请不要按摩“的英文怎么说，谁知人家业务水平之高，递上一张纸，上面一个小人儿的图片。需要力道哪里轻哪里重就画出来，不想按摩画个叉叉，免去了语言之不便。于是我在足部正反面画了大叉叉，意思是，这个区域就不需要您费劲了。谁料想——

人家并不是上来就按，泰国妹妹把你领到屋子里排排坐，端上个小盆盆，请脱去鞋袜，双脚放进温热的水中，抓起一把海盐在水中融化，再挤上那半颗丑柠檬，给您先来个冬阴功风格的足浴。这穿过我双足的她的手啊，搞得我想笑又不敢笑，“My feet is very sensitive, even myself, when I touch, I’ll laugh”。我的四级水平的英语超长发挥，化解了中泰两国对于洗脚的尴尬——泰国妹妹刚才看我在小人脚上画叉还满怀不可思议地跟我微笑，我这么一解释，她笑得更不可思议了，哇塞，世界真奇妙啊。

而后的体验就比较顺畅了。进屋换衣服。体验了一把正宗的传统泰式按摩。个人感受这泰式按摩最有特点的就是，你在床上，按摩师也在床上。

我趴好后发现按摩师（♀）竟也上了床，刚想会不会有什么不可描述的动作，人家就一个肘击开始了此次按摩。从背部开始，然后是肩颈、腰、腿一路下行。幸好我选的是轻度的，但劲头也非常劲道，几度要暗地里咬牙吸气攥拳头。为了不给祖国母亲丢脸，我一直憋着没有将那句牢记在心的“抱抱奶”（泰语：轻点儿）说出口，就像长跑，适应力道之后就是享受的过程了。比较印象深刻的是按摩腿部的时候，按摩师会以她的腿支撑，把我的腿放到上面一通推拿，好不舒服。我足底之前画了叉，所以一带而过，然后就翻面儿揉正面了。本来想正面直视按摩师会不会不好意思，人家很专业地用毛巾给你把双眼蒙上再开始。断了我等一饱眼福的非分之想。

整个过程我都非常享受，唯一的遗憾是后半程一直想尿尿，尤其是后来按摩腹部，隔山打牛一般刺激着我的膀胱。导致直到结束都很不放松。这套为时两个小时的VIP传统泰式古法按摩以几个极具特色的拉伸结束。最后按摩师脚顶着我后背，双手拉我手，头部自然而然地往后仰，我真担心先刺啦再噶嘣儿。还好我挺了过来，还能微笑着对泰国妹妹说一句：“卡朋卡”。

按摩之后，泰国妹妹又奉上茶点，玫瑰茄热饮加一种味道清香的米饼。他们怎么知道我按摩完了有些饿呢。

大象粑粑纸张乐园

您没听错。我们就是要一起去组团参观大象的粪便，了解自然界之神奇。用我们小时候常说的套词来说就是：大象全身都是宝，拉出来的粪便都可以为人所用。导游老师讲解说，大象的粪便是没有臭味的，因为他们吃的是草，拉的也是草，您要不信，可以去(尝尝)闻闻。

于是到了目的地后我就四处张望，想先参观一下大象厕所。结果公园风景如画，绿树如茵。讲解员操着泰国口音的中文说道：我们这里没有大象，只有大象的粑粑。我们这里种草，然后把草送给大象保护区，然后他们把大象的粑粑给我们，我们在这里制作成纸张。而今天我们就要告诉大家，大象的粪便是如何变成纸张的。

其实大象粪便制作成纸张不是难事，至少对于来自造纸术发源地的人们来说，老祖宗都告诉我们了：纸张嘛，纤维嘛，水嘛，混合嘛，搅拌嘛，晒干嘛。至于用什么纤维，用什么水，用什么混合，用什么搅拌，那就靠世界各地的朋友发挥想象力了。所以泰国朋友想到用大象粪便做纸张也不稀奇，看到那样的庞然大物，谁都会惊叹造物主之神奇，以及想获得一些自然的馈赠。

这个公园非常适合带孩子一起游玩，跟着老师一起学习如何制作纸张，眼睁睁地看大象粪便变成纸张的过程很有意思。尤其在一口大锅里熬制粪便的过程令人印象深刻，你需要花很大力气搅拌粪便长达一个小时之久——当然是使用一个木棒而非你的手——讲解员说这根木棒在他们专业上称为：“搅屎棍”。

恩，可以称得上是很名副其实了。
