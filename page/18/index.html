发表于 2020-03-10 更新于 2022-01-27 分类于 红桃 - 文章

今天看了一个帖子。作者：一步数学 (来源：知乎)

三角形内角和180度是欧氏几何下的重要结论，90%以上的人认为三角形内角和180度是不可捍动的。实际上，1826年，在俄罗斯的喀山，数学家罗巴切夫斯基发表了一篇＂有违常识＂的演讲，他说平行线可以相交，三角形内角之和不等于180度等古怪的定理。当然，这是当时高斯发现但不敢发表的，这确实太有违普通大众的认知——这就是非欧几何，他的独立称为罗氏几何，在罗氏几何背景下，三角形内角和小于180度的。

与罗氏几何对应的黎曼几何也属于非欧几何，当然黎曼是完全颠覆了欧氏几何的五条公设，在黎曼几何的背景下，三角形内角和是大于180度的。

看完之后我大呼牛逼，朋友们，从今天起不要跟我谈什么欧式几何，哥们只认黎曼几何——虽然我不懂，但我认他！

还看到一个有趣的。说“普罗大众”这个词儿，并不是中文词汇，而是个外来语。且不是外来自类似印度那种历史上的文明古国，而是来自我们社会主义阵营的朋友，源自法语“prolétarien”，而原本的意思更是让人感慨。

您知道谁是普罗大众么，普罗大众实际上指的是咱们无产阶级！

为每一天留下一句(至少)

发表于 2020-02-27 更新于 2022-01-27 分类于 红桃 - 文章

小猴总算睡了。写两笔。

疫情仍在蔓延，这周公司终于结束了长达三周的居家办公，正是到公司复工，但并不是所有人都到公司，一部分是因为有些回京人员还在隔离期。另外公司办公楼的物业要求复工率不能超过50%，也就是说，座位得有一半是空的。为了迎接复工，公司也做了一些准备。每个人发了口罩、手套、抹布、安排了每日负责消毒的员工。然后把员工作为调整成Z字形，意思是你对面和旁边至少都空一个人的距离。还要求尽量全程带口罩上班。当然还有每天测几次体温。

我也不敢掉以轻心，以饱满乐观的热情做尽可能充分的准备。

每天早上洗漱完毕，整理好书包就开始上装备。帽子、口罩、护目镜、手套一应俱全，同时加上一件长款带帽子的雨衣。书包里装上了消毒纸巾、消毒液以及带的饭和水——本着尽量少接触的原则，自带午饭还是最保险的。除此之外通勤方面考虑到路途较远，只能狠心咬牙跺脚打车，还不敢打载客频率较高的出租车，斥巨资打专车，来回就是小150元，几天下来幸好目前打专车的人不是太多，而且路况很好，基本不太堵车。坐在车上的时候开着窗户，一动不动坐一路下车都一身白毛汗，可见我这装备的密封性很好。

到了公司为了喝口热咖啡，还得冒险去趟便利蜂。感谢自动咖啡机和在线支付，保证全程不跟人多说一句话。进写字楼测体温。通过后取电梯口的手纸一张按开关。开始考虑过走楼梯，但带着口罩一口气上九楼还是比较有挑战的。后来采取的方式是早点出门，所以这几天到公司坐电梯基本只是我一个人。

指纹锁是到公司后第一个不可避免地接触。摘手套后开门、进屋。然后开始消毒流程：先开窗通风，然后自喷84消毒液，把雨衣前后和手套书包外部都喷一遍。然后脱下雨衣放进塑料袋——专业上称之为“沤着”。然后脱下外套。用84给工位四周喷一遍。然后拿上水杯和抹布，去洗手间洗手漱口、洗抹布，回来后擦桌子。然后把带的饭放冰箱。这一番工作后，总算可以打开电脑，用湿纸巾擦手后，开始吃早点。

这段时间公司要求上午10点到下午4点办公，4点后都得下班，不得加班。终于让我们这些社畜看到了下午四点的北京的景象。

下班后会有值日生喷消毒液。而我要做的就是再次全副武装，在路人羡慕的目光下钻进一辆专车，捂一身白毛汗。到家后我这个全家最脏的人第一件事就是放下所有东西，全身喷一遍84，摘口罩脱下雨衣后直接进浴室、洗手漱口、洗眼镜，然后洗澡。至此才能亲吻各位并道一句晚上好。

每到此时，感觉有些恍惚。可能是口罩带了一天，有些缺氧。一时间摘了口罩，又有些醉氧。

发表于 2020-02-24 更新于 2022-01-27 分类于 红桃 - 文章

二月二

在百度搜索日历的时候，你可以看到阴历二月初二这一天居然被称为“龙头节”——就叫龙抬头不好么😂。看到这里我都想当一把百度的产品经理，提出此处应该有一个交互，鼠标悬停今天的时候跳出一个水龙头厂商的促销广告。

全副武装

口罩是硬通货，戴上才显得不失身份，再加上一个护目镜，看到的都知道你是个严谨的人。当然最让路人暗挑大拇指的还是你那一件长款白色套头雨衣，实现了个人防护从玩票到专业的跨越。他们还没有看到你的手套哩——那副外卖附赠的半透明手套像世人展示的是你严谨而不失勤俭的风范，疫情之下仍不忘节约初心的中华男儿的形象。

路上

路上人并不多。但车不少。不限号，不（没人）收过路费，人们如果不是必要都不太出门——此刻光景下如果天气很好，且那个数字没有过分地增加，人们会做什么呢？我不太清楚。不过据说昨天前往香山植物园的路都堵了。
