惊蛰

发表于 2024-03-05 更新于 2024-03-12 分类于 红桃 - 文章

惊蛰之前终于把蔷薇种下土了。挑了半天，还是买到了宿迁的土球苗。看来种花草的是逃不过宿迁大学的。所谓土球苗，就是苗的根部被一坨厚实的黄土包裹的花苗。这么做的用途是在运输过程中保持根部水分不流失。而送到手里的时候，土球表面已变得坚硬。别说根了，我用手使劲扣都抠不下来。于是果断上网学习。取桶，接水，花苗扔进水里，泡了二十分钟，拿出来，烂泥融化了一部分。看到一些白色的根系。这下倒是能种了。

土是山东买来的好土，椰糠加泥炭土，保水透气，摸着像海绵。小萌又买了30厘米口径的大盆，花苗放进去，极为宽敞。在室内花盆界，这算是大平层了，以后这花要是不猛猛长，都对不起这么好的盆和土。当然这是爬藤蔷薇。少不了专用的架子。架子也是小萌精选的，顶部站个小松鼠，以后就会被群花环绕。花苗下土后后浇了几遍定根水，直到花盆底部水析出干净。上下通透，就可以慢慢缓苗了。

坐在沙发上，看着有些秃的花茎，盘算了一下，这花成本不低，花盆60，花苗20，土20，还有专门为了促花买的花多多2号肥料，20。里外里100多出去了。难怪网上的专家说，春天种花要注意三件事，一件是温度，一件是浇水，还要注意自己的钱包。

关于2024年春节假期。

公司提前了几天放假，但我从放假前几天就感觉疲惫与不适，放假前后鼻炎终于爆发，右侧鼻腔堵塞，眼眶酸疼。而这一切是能够安静放假的代价。放假前的工作、焦虑的心情，加上为了搬家每天早起晚归，没有休息好。所以身体是诚实的，你不让我休息好，我直接给你上症状。这种情况让我陷入沉思，曾经也有某年过节之前，身体不适。这似乎是一种工作的诅咒，当你的身体放松下来的时候，就开始承受紧张时积累的痛苦吧。

春节是时间的缩影，每当假期结束我们都会感叹，时间怎么过的这么快！

节前一个想法萦绕在我脑海里，我想在假期期间画条龙，但画什么龙，怎么画一直没有想法。直到到了放假那天，我改变了想法，我就是想睡觉，想好好休息，想恢复体力。想赶快过年。

青青最近迷上了玩牌，兜里揣副牌，逮谁跟谁来。说的就是他，这个九岁的小朋友。

春晚至于青青最大的收获是成为了刘谦的粉丝——应该说这是刘谦的收获。

看了电影《养蜂人 Bee keeper》，杰森斯坦森演的动作片，简单直接，没有废话。就图一个痛快。只不过觉得美国人对电诈的理解挺有意思，呼叫中心弄得跟网吧似的，还有个喊麦的主持人。

看了电影《年会不能停》挺好看，吐露打工人的心声。

放假期间还是练了练画画，照着伯里曼的图画了几笔，在公园的时候也写生了一两棵树。

「将走过变为拥有。」

看到一句话「翻过一座山你就高过一座山」这话听起来很狂，够硬。但我马上想到的是高过一座山就会马上向山下走去，你并没有高过一座山，并且永远无法高过一座山。我读到这句话的同时就从山中抽离了出来。而我想到上面的思考的瞬间又从这思考中抽离出来——为什么我会这么去思考？是内心潜在的焦虑感在作祟么？我甚至想改写这句话，或者由此引申，不如说「将走过变为拥有」无论江河还是高山，你走过，途径过，甚至只是眺望过，就把这种感受沉淀在内心里。

龙潭庙会

发表于 2024-02-11 更新于 2024-02-17 分类于 红桃 - 文章

记得上一次来龙潭公园好像是和罗老师，绕着龙潭湖走了一圈，拍了点照片。好像也是冬天，印象中气氛萧瑟且冷清。

多年以后，我们重游龙潭湖，倒也不是为了看湖，也不是为了拍照片。在春节假期期间，到这里无非就是想深入到人民群众当中，融入滚滚人流，感受已经为数不多的节日气氛。是的，这是2024年春节的龙潭庙会。

从家打车前往龙潭公园的路上我就跟孩子吹牛逼。说当年你爸爸我的学校就在龙潭湖附近，每年冬天我们都举办环龙潭湖长跑活动，绕着龙潭湖一圈2.7公里。其实我印象中就跑过一两次。但龙潭湖公园在我的童年分量还是很重的，仅次于东单公园。龙潭湖公园距离我家和学校距离差不多，距离姥姥家距离也差不多，旁边还有北京游乐园。很多次去姥姥家吃完中午饭，下午都可以去龙潭湖公园转上一圈，实在非常惬意。

我还记得在高中的时候有一天和同学中午出去打游戏，那天天气很热，我们还在龙潭湖旁边买了一个西瓜大家一起吃了。记得同行的有老王还有刘超。

龙潭公园附近有一条路叫光明路，另外一条路叫幸福大街。我非常喜欢这两个街的名字，充满了八十年代昂扬向上的感觉。记得从龙潭湖沿幸福大街往东走到头，是一个丁字路口，我爸爸曾经工作的北京水暖器材厂的旧址就在那里。现在应该早被时代的车轮碾碎，消失在人们的视野里了。

还没有到达龙潭公园北门，路就被管制了。还好下车后步行一个路口，就到了龙潭湖公园。能看到慢慢汇聚的人流，但比起网上前两天的照片，今天的人还不算太多。我们也几乎没有排队就买了门票进入了公园。龙潭湖顾名思义，中间是一个人工湖。我们的路线也是绕湖而行，主要目的地是位于南门附近的一个棋牌类的游戏点。之前小萌做功课，说庙会有棋协大师参加的车轮战，就是冲着这个我们来到龙潭公园。

龙潭庙会的装饰也都很传统，无非是各种灯笼和沿路的摊位。湖面被改造成了冰雪世界。除此之外就全是沿途慢走的游人了。青青对周围的景色没有兴趣，一个劲儿往前跑，想赶快参加车轮战。边走边问，终于到了一个湖边小楼房门口，管理人员说下午两点开始。我们看了看门口的海报，是一个名叫沈阳的国际女子特大。

到点后跑上二楼，青青坐在一个空座位上。后来一问才知道要提前报名，但好在有报名没来的棋友，于是青青也可以直接对阵。但不知道是因为刚才跑了一路累了，还是临近中午饿了，总之，青青摆出一副赖洋洋的样子，不用说下棋的斗志，坐都坐不直。下起棋来毫无精神。让人看着揪心。好在这次游乐体验大于练习，而且对手可是特大，输气是情理之中，赢了才新鲜。于是我们就一旁观战。大概三十分钟左右，果然被将杀。签字握手，活动结束。

除了国际象棋，还有中国象棋、围棋和五子棋的活动。可以看出来每个棋类的目标人群不太一样，参加国象的大多是小朋友和小朋友的家长，一边看着棋盘一边皱眉，气氛严肃。而中国象棋那边就好像把公园里的石桌搬到了屋子里，果然是一群中老年人，围着另一群中老年人热烈讨论着每一招。组织者几次跑过去让他们注意保持秩序，不要聊天。无论是下棋的还是看棋的，都参与到了局面之中，好不热闹。

下完了棋走出屋外，依然是人头攒动的庙会。青青被各色游戏吸引，每个都上去看看研究研究。为了让他开心，我们让他玩了一个弹弓打沙包的游戏，还真打中了两个，获得五等奖。还玩了个抽奖的游戏。说白了就是送钱，拿个钥匙链之类的安慰奖。整个庙会就是大型割韭菜现场，不仅游乐项目贵，吃喝更贵。我们走了半天，看到有尹三豆汁店的门脸，便斥资买了个豆汁和涮肚。价格算是适中，还挺好吃。

游览完庙会出来已经四五点钟的光景。我们去锦芳小吃吃了晚餐，小萌刷新了对面茶的印象。

腊月二十九

发表于 2024-02-08 分类于 红桃 - 文章

今天给我的几盆花做了些过年的准备。它们搬进新家大概有一周了。应该也适应了新的屋子的温度和湿度。还有几盆火龙果还在袋子里——长时间的闷着对它们肯定不好。也就趁着这个轻松的早上，我喝完一杯咖啡，拿上一听啤酒，准备开工。

网上有的说啤酒和水的比例是1:100，也有说1:2的，所以我中和了一下，倒了100毫升的啤酒，再加水到500毫升。稀释之后，用柔软的湿纸巾浸泡后擦拭每一个花的叶片。

经过擦拭，绿萝上一层灰尘被清洁干净，叶片显得精神了很多。顺便还擦了富贵竹的叶子，富贵竹有些焦尖儿，我怀疑是有些晒伤，于是挪到了阴凉处。还有泡了一冬天的菠萝头，擦拭之后也显得生机勃勃了许多。最后拿出来几盆火龙果，虽然没法给它们擦叶子，但也用啤酒水给它们浇了水。喝点过年的啤酒。

对了，还有从公司抢救回来的两盆蝴蝶兰。叶子擦拭之后也显得更加饱满。而且其中一盆长出了新叶，有一盆的花剑（还是根？）蹿了好长，真是令人欣喜的变化。

植物以它们自然的状态生长，我们只需要勤加照护就能享受它们美丽的样子。而这一切都是免费的。

记记账

发表于 2023-12-19 更新于 2023-12-20 分类于 红桃 - 文章

年底除了总结就喜欢记记账，尤其在今年。今年的年底给人一种焦灼的感觉，如泥潭跋涉，每一步每一天都不那么痛快。

  1. 为自己的域名续费，129元/年。不为别的，就为在这个移动互联网时代给自己在传统互联网上留一块自留地。就像无论在什么年代，我们都有一个拥有一个院落的梦想。从原始社会到赛博时代，古今无不同。

  2. 给自己和父母买了医疗保险。正所谓未雨绸缪，或者买个心安。北京普惠健康宝，好就好在没有年龄限制，也不用体检。没想到，今年还能直接用医保买了。159元/人。

  3. 故宫年票。300元/人。经过一年的体验，故宫年票最大的好处就是只要你想去，就肯定能约上。300元十次，平均下来30元/次并没有便宜多少，但看到今年夏天故宫的火爆程度就知道人家卖的不是票面价值，卖的是包您成行的体验。“哎呀故宫明天已经约满了，哎？我约上了啊，哦对了，我是年票用户。”（哎呦，有装到😂）故宫年票用户有专门的预约通道，而且孩子还不要钱，太适合我们这种家庭了。

  4. 《万物》杂志300元/年。这个杂志已经定了3年了。正所谓再穷不能穷教育，再苦不能苦孩子。

随笔

发表于 2023-12-14 分类于 红桃 - 文章

在这个时代有什么比读一首诗更不具功利性的事呢？还是一首20年前的诗。还是一首自己写的诗。

这是一段穿越时空的文字，如果有人问这首诗的灵感来源，其实我早有准备。因为我时长回想到那个场景：那是20年前的一天，我走在天坛公园的小路上。

印象中应该是在丹陛桥的西侧。上午还是下午有些记不清了，总之天气很好，阳光也很好。灰色的墙边是大片的绿地，从成片的松柏均匀分布的样子就知道这片树林绝非自然生长，他们种植在这个园林里的时候就被赋予了意义。他们和这片土地的主人一样受到尊敬，享受土地和自然的滋养，也绝不会有人故意伤害他们。只需在特定的时间贡献出一些枝条，让帝国的皇帝在燃烧枝条产生的烟雾中祈求上天的怜悯，赐予他和他的帝国下一年的风调雨顺。而至于帝国的命运到底如何，树木无需关心，也不必负责。

人们精心呵护这些树木。让他们在几百年后长得异常挺拔。张开巨大的枝丫，形成巨大的树荫。几百年后，在严格规划的步道上，人们对这些树木仰望并赞叹。我也是那些游客其中之一。每隔一段时间，草丛中便会喷出水来，水量充足且稳定。而那个下午的阳光穿过枝叶和空气，穿透水帘。我一定是被水光反射的彩虹吸引，站在了那里，愣了很长一段时间神。一个问题忽然闪过我的脑海：那么多的水，浇在它们的身上，究竟是为了什么？
