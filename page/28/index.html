20150727伏日杂记 [星期一 北京 晴]

发表于 2015-07-27 更新于 2022-01-27 分类于 红桃 - 文章

我说过的一些话，或者一些表达方式经常被家里人取笑。比如“用微波炉加热”，我习惯说成“把这个菜用微波炉打一下”。后来据说被我老婆听到笑了好久，觉得打这个动词用在这里特别搞笑。这个细节我自己都没有察觉，因为我妈也这么说，天经地义一般 右侧肩膀脖子感觉不舒服，被风拍 20150725 小猴第2次去露天游泳池。 20150724 小猴长到了十个月大。 20150718 小猴第1次去露天游泳池。上周本来就想带小猴去游泳池，结果他父王我偶感风寒，体弱发烧，没能成行。这周末天气预报说有雷阵雨，白天看天气还凑合，多云，闷热，于是去游泳池看了看。发现没什么人，下水后感觉还真他妈凉。抱着小猴踩了踩水，他很喜欢，弯着腰拍水。结果他妈不小心还让他掉水里了。吓哭了。赶快抱起来哄，后来我们觉得太冷了不适合小猴下水，于是就回去了。虽然人很少，但太冷了，不免有些遗憾。

所以我们管青青叫小猴也不是没有道理的

发表于 2015-07-19 更新于 2022-01-27 分类于 红桃 - 文章

那天晚上，他姥爷正躺在沙发上看电视，忽然“哎～”地叫了一声儿，循声望去，只见他双臂抬起，僵坐在了那里——右手指着左边，头也向左边盯住，就像京剧演员的亮相。不过他可不是为了给乐队一个信号开唱，我们循声望去—— “快，电蚊拍！”他姥爷很小声儿的补充了一句，更印证了我们的猜测。肯定是胳膊上有只蚊子正妄图吸食革命好儿女的鲜血，那怎么能让它得逞呢！就像歌中唱到的那样“朋友来了～哦～有～好酒；若是那蚊子苍蝇豺狼虎豹，招待它的～咦～有哦电蚊拍加猎枪！” 得知此意，更不必多言，老英雄坚毅挺拔的造型那意思就是“尔等退下，祭上电蚊拍，我削死他！”只见那，青青的爸爸（也就是作者我）一个鹞子翻身，蜻蜓点水，将电蚊拍送到手中。老英雄慢慢将电蚊拍靠近手臂，猛然拍下，小蚊咂，就是天王老子也救不了你啦——只见电蚊拍上噼里啪啦闪了一阵火星——那肯定是打着了！肯定还是个吸饱了血的大黑蚊子！ 我接过蚊拍，走到灯下仔细观察，看看它腹中是否有鲜血。谁曾想，电蚊拍上居然没有蚊子的尸体！我正反仔细看了看，还是没有发现，难道刚才不小心掉地上了？还是蚊子没脸见人，已自觉化为了一股子青烟？ “电蚊拍上怎么没看见蚊子啊？您刚才是打着了吧？” “好像是吧，”老英雄道，“我看刚才不是都闪光了吗？” “奇怪，我拍了电蚊拍半天也没看见有，难道…”我问道。 “难道不是蚊子？”老英雄接道，“那怎么会有闪光呢？肯定是电到什么了，当时我胳膊都疼了一下呢。” 一家人陷入了沉默。 许久，只听得老英雄长叹一声，“好吧，既然今天不小心暴露了，我就告诉你们吧——其实，电蚊拍上电光石火的是—— 老夫的一根汗毛。” 原来，相传在神话时代，东胜神州之傲来国图之境内有一个美猴王叫孙悟空，大圣自是从金蝉子习得一身本事之后回到花果山水帘洞，每天教授山中猴子猴孙一些本事，有个山民就在砍柴途中偷看到了大圣传习，忙记下几句，回家一试，发现竟是那汗毛化物的本事，只消捻下一撮汗毛，默念咒语即可化作实物。后来此技被秘传于后世，他姥爷竟是那传人之一！平日里老英雄深藏不露，只不过今日他舅姥爷来看青青，于是他做了一桌大菜有些疲惫，屋里蚊子有太过恼人，于是看电视半睡半醒之际，梦到与蚊子鏖战，心生一计，不如变出个蚊子引来群蚊，将之一网打尽。于是不小心，施了法术，将汗毛化成了一只蚊子。这才有了后来的一幕。 “我先师孙猴子的故事我就不跟你们多讲了，有书也有电视剧。你们自己看去吧。但我时常想起他说的一句话，每当夏天，就是我行事之准则，那话是他虎口救下唐僧时所说，令人念念不忘。师傅说： 老虎要打，苍蝇蚊子也要打！” （所以我们管青青叫小猴也不是没有道理的）

20150701随笔记 [北京 星期三 晴]

发表于 2015-07-14 更新于 2022-01-27 分类于 红桃 - 文章

其实小孩子冷暖不自知还是和家长有很大关系的。比如青青的姥爷，身长八尺，非常魁梧，到了夏天怕热，就总担心青青穿多了热。看青青玩了一阵儿，一摸他后脑勺，忙说：“都出汗了，把背心脱了吧，光膀子凉快点。”—— 而他爸爸我看在眼里，就怕青青被风拍了，因为我身体偏瘦，就怕着凉，稍微有点风吹草动，就想给青青添件衣服。 其实我们觉得孩子冷或者热，不过是自己觉得冷热，爱屋及乌罢了。就像青青的玩具，你给他买的他似乎都不大在意，无论是能出怪声音的电动小狗，还是叮当乱想的塑料球，他真正喜欢玩的是姥姥给他扇风的蒲扇，换尿布时看到的手纸，还有挂着的各种衣服，各种想够又够不着，想摸但被大人抱着的时候故意避开的东西。其实青青的想法很简单，就是拿过来，放到嘴里尝尝，如果味道不好，就吐出来，然后在手里捏来捏去，因为他已经成功地把手纸撕成过很多纸屑，并吃掉它们。所以其他的东西在他眼中只有两类，能吃的，和不好吃的。 青青他姥爷非常擅长挑选蔬菜水果。他能给你很详细地讲如何在一大堆西红柿中间挑选出最优质的那一个，以及轻易地举出两堆长相类似的鸡蛋中哪一个更好。这个手艺很多游弋于菜市场多年的大妈都不能企及，或者他们没有慧根，浸淫多年也不得要领。他姥爷在这方面堪称神助，你要是在那里挑菜时他从身边路过，只消余光一打，就知道你的选择是好是坏——你应该把西红柿翻过来，看它的屁股，选什么歪瓜裂枣烂酸梨的说法都是胡扯，你得看这屁股窝对不对称，深浅合不合适，还得看大小，还得看颜色，总之不是一句两句说得清的，说了也不是你一时半会儿能吃透的，这就是个经验，就是个水平，就是张秉贵的一把抓，总之，你小子选的这个，是个缺哒！ 写下上面这些只是为了说明，青青的茁壮成长是和他姥爷这手眼通天的选菜本事密不可分的。当然他姥爷还有煮饺子不漏的绝技（只有他煮的一个肉玩儿的羊肉馅儿饺子一个都不破，且火候到位，煮完了锅是锅，饺子是饺子，汤是汤，让人叫绝！），说到味道，他姥姥的调味儿那必须再用一篇长文来记叙，总之这一身的绝技，不仅让青青在饮食上保证了质量，还吃出了噔次。 读后感 读了《暗店街》。这本小说在王小波的作品中被提到过，据说王朔也深受这本书的影响。后来看了一些书评，说得云里雾里。梁文道的《开卷8分钟》也讲过此书，说的比较明白——说作者以“记忆”为自己写作的主题，但这个主题在法语世界里是不能轻易触碰的，因为前面有一大哥的作品太过难以逾越，名字叫《追忆似水年华》（A la recherche du temps perdu）。所以作者莫迪亚诺取道清逸，不去写记忆中那些难以忘却的片段，而是去写那些被忘却难以拼凑的如碎片般的记忆。其实和很多名作一样，《暗店街》全书体量很小，却读完让人引发人很多思考。比如里面提到的关于人们记忆的不确定性，说到，每一个人对于他人的记忆都是不完整的，而一个失忆人如果想通过别人的记忆去寻找拼凑自己丢失的记忆的时候，这几乎是不能完成的一个任务。这一切就像通过沙滩上留下的脚印去寻找一个人的踪迹——“沙子只把我们的脚印保留几秒钟。”

20150619 随笔记 [星期五 北京 晴]

发表于 2015-06-19 更新于 2022-01-27 分类于 红桃 - 文章

青青的姥爷夏天最怕的东西就是蚊子。不只是怕，是又恨又怕。如果让他列举出10条蚊子的罪状，他会毫不犹豫地列举出20条。并且指出，蚊子可恨，不仅仅是让他这样的老人夜晚不能安睡，更可恶的是还影响了祖国明天的花朵——也就是青青——尽管现在也就算是个花骨朵儿。青青是小鲜肉中的小鲜肉，蚊子最喜欢，而且青青还不会说话，只能乖乖地被咬，被咬了会不开心地闹，这样就更让他姥爷这样的老人不能安睡。总之，蚊子是天下最可恶的害虫，见到蚊子想也不要想，只要伸出手掌拍过去就成了。 有好几个清晨，我都发现他姥爷早早的就起床了。告诉我他5点就被耳边的嗡嗡声吵醒了（并且发现身上已经被咬了好几个包），于是起来打了两个小时蚊子，战果是灭掉了4只——并通过碾碎尸体来判断是不是咬过人的。如果蚊子尸体上有血，那定是咬过人的自不必说。如果没有，那么感觉更是走到了时间的前面，没有让蚊子得逞，感到大快人心。但两个小时4只蚊子的代价就是要比太阳公公起的还早，打完了再睡回笼觉的话，又觉得耽误了一日之计有些亏。索性一咬牙一跺脚，坐在藤椅上闭目倾听，看还有没有小斯过来受死，再骂他个把小时直到天亮。 这期间青青没准会醒来，坐在床边四处张望——他已经不像前一阵，醒来只能躺着玩自己的脚。现在他已经能够很轻松地坐起来，甚至能扶着床边的栏杆站起来几秒。如果他的妈妈还没醒，青青玩久了会因为有些无聊咿咿呀呀地喊起来，嗓门之大不输给打鸣儿的公鸡。于是他姥爷就站起身把青青抱到他们屋（这样她的妈妈能多睡一会儿），周身翻看一遍，数数身上有没有新包。如果有包，那么就把之前的碎碎念的整理一下，就着刚灭掉的几只蚊子，再痛骂一遍。 其实如果点蚊香或者支蚊帐，估计能很有效的驱虫。但蚊帐太憋气，且他姥姥姥爷不喜欢蚊香的味道，气味儿太重影响睡眠。又觉得蚊香是化学的东西，可能歼敌一万，但不免自损三千。尤其对小孩儿不好。所以目前，我们还是以纯手工灭蚊为主（电蚊拍是个好工具）。每天晚上我们全家在手里没活的时候都会站起身，四处溜达，抬起头望向各个角落，每当看到有一只蚊子落在了墙上，就会激动地通知大家：“快，给我电蚊拍。”这个时候，大家都屏住了呼吸，目光如炬。每当这时，我都觉得自己肩上的担子非常重。 胜负就在一瞬间。 当看到蚊子的尸体被电流吸住，并迸发出耀眼的火星儿时，全家人的脸上都洋溢起了笑容。执蚊拍者像英雄一样受到了大家的赞扬。而发现蚊子的人也被记上一功，就好像成就了一次射门的漂亮助攻。有时候，发现蚊子的愉悦甚至更高，就像海盗发现宝藏。 *** 20150619 今天青青能抱着吸管杯很顺利的喝酸奶了。
