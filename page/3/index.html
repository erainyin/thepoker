花开了

发表于 2025-01-09 更新于 2025-01-27 分类于 红桃 ， 文章

我要把2025年的第一篇文章献给我的花朵。两盆从公司救回来的蝴蝶兰。经过了一年，公司没了。花还在。花还开了。两盆都开了。粉红色的花朵在一月的阳光中给冬日的北京增加了一抹亮色。蝴蝶兰是一种奇妙的植物，这种在自然界里附生在树木枝桠上的植物拥有强大的生命力。甚至不需要土壤。在干燥的环境中，他们的肉质气生根却能长得异常粗壮，生存本能让这些根像四处延伸，寻找可能的水源。这种生存智慧对于养花的人是一种馈赠。和很多其他花草不同，养殖蝴蝶兰最好用透明的塑料花盆，并且扎孔透气，让水分能够充分流淌。在干湿循环的条件下，一条条气质肉生根从花的底部探出头来，根的头部是紫色绿色相接的颜色，有时还会有白色的皮包在上面——这时你要做的是，将花盆端到眼前，仔细端详。这就是「赏根儿」。

赶在最后一天之前

发表于 2024-12-30 更新于 2024-12-31 分类于 红桃 ， 文章

那就是倒数第二天。

新的一年的，倒数第二天。

人们总是有更多的要求，你做好了1，他们说可以再加1。你做好了10，他们说可以再加10，你做到了90他们还说，可以做到100吧，或者，你做到了100，他们说，也许有更好的，是不是可以再这样试一下。人们总是有很多要求，我要做的是剔除这些「以外的」，不仅仅是需求，甚至是声音。对自己说，做到这样就可以了。因为事实上，他们真的在乎结果么？或者说，他们认为加1之后那个结果真的好么？以我的经验，只是多一种选择。多一种对比，多一种可能的错误。仅此而已。

大队长记

发表于 2024-09-27 更新于 2025-01-05 分类于 红桃 - 文章

「青青今天当上大队长了！」——小萌在家庭群里宣布。

前两天青青回家后跟妈妈碎碎念，说我好像是我们班大队长。妈妈听了很吃惊，没听你说过啊，是班里选举的么？青青说，不是，就是听说的。说他们班的班长跟他说：“咱们班有两个大队长”，青青听了好奇地问：“都是谁啊？”班长一拍手说：“就是咱俩！”

听到这里我们都快笑出声音。因为前一阵班里选三好生，他落败于另一个同学。因为青青不太会搞人际关系，只跟几个要好的小伙伴玩。一投票，只有一个人投他。虽然他回来后做出一副看得很淡的样子。问他他就说想不起来了什么的，实际上，这种荣誉对于小孩的吸引力是巨大的。谁不想挺起小胸脯，甩起小胳膊，让红红的杠杠闪耀在班集体中呢。

所以后来青青说大队长是他，我们也没太在意。还以为是小道消息。并且我们对这种名利也没有太追求，没有教过青青为了争取选票讨好同学什么的。只要做到公平公正对待每个人，做自己就可以了。

谁知道这风水轮流转，今天官宣了青青成为了大队长。是老师钦点的，没有选举😄。
