2013-10-06 假期小记A [星期日 北京 霾]

发表于 2013-10-06 更新于 2022-01-27 分类于 红桃 - 文章

)顺义某某山

Day1

睡到大天亮，浑身的疲惫。起来以后慢慢悠悠的开始收拾屋子，吸地，擦地。就像很多个周末开始的那样。外面的天雾霭沉重地像要掉下来——事实上，它已经这样坚持了几天，我在期待某一个时刻天光洞开——这实在不像北京的秋天，不像十月的开始，和秋高气爽什么的根本不沾边，甚至是对着干。漫天的浓雾甚至有了新的名字：霾。多难写的一个字儿啊，只为硬生生的告诉你：这不是雾。人活着真累，呼吸都感到困难还要分辨雾和霾的区别。但所幸的是，这个时候天上似乎开始落下雨滴，像两种溶剂交融，重的那种缓缓流下。

到午后天忽然通透了许多，一阵风（你都不知道什么时候刮起的）一下子把天吹蓝了，干净了，通透了。楼下的人明显多了，互相笑着，指着天，或者抬头仰望，有种奔走相告的意思。

Day2

好天气。跟爸妈约了打算去郊区走走。车一路向南——房山也许人不是很多。虽然也没什么特好的景点，但估计很多人和我们一样——不出去，岂不是浪费了这么好的天气！人就是这么不知足，天气好的时候其实在哪里其实都一样，但人们总是纠结北京的月亮和纽约的月亮哪个更圆这样的问题。好吧，直到真正上了高速，你才知道被骗了——北京高速一日游，各种车辆组成的长队，尾气和鸣笛，被堵在路上的人们发出的谩骂声。人们与他们的目的地遥遥无期，据说河北因为大雾（或者是大霾？）封了路，还有前面一定是出了交通事故——北京的高速路有种烽火台的感觉，只要堵车基本就是两种情况，警察封路或者出事故了。那种交通事故发生后还有一条车道畅通无阻的情况叫做“外国”。据说离我们有十万八千里远。

我们没能坚持到房山，中途找了个出口下了道。去卢沟桥走了一圈，看了看宛平城，永定河居然有水了——记得小时候学校组织拉练从六里桥走到过这里，那时周围还是一片村郊的景象，永定河底一片荒草，似乎有人在下面放羊。那次出游让我懂得了，接受爱国主义教育跟春游是两回事儿，当时怎么就那么傻呢，还背了好多吃的和一本书！我还想坐在车里舒舒服服地阅读一下课外读物，结果到了六里桥集合几个老师钻进了一辆面包开路，引领我们一队学生开始徒步。

如今卢沟桥景色比曾经好了许多，河水湛蓝，还能远眺西山。人也不多。除去需要20块的门票今天算是来着了。

Day3

在家里看了两个电影，有些心神不宁。看着外面的天气不如前两日那么好，开始变得有些灰蒙蒙。

《星际迷航暗黑无界》 (Star Trek into the darkness) 硬派科幻的代表，企业号就是太空版的美国梦，很理想主义的那种，理想主义到主角最后身中辐射还能死而复生。spark船长的发型我一直很欣赏，我要是来那么一个发型一定错不了。

《温暖的尸体》(Warm bodies) 剧情扯，主角帅气。僵尸都能变回人类，爱情真伟大。

《环太平洋》 (Pacific rim) 机器人打打怪兽。制作精良，尤其是怪物的细节。机器人设定有些弱，有武器不早点使出来，多造几门大炮感觉比研究什么同步率更实用。

《Breaking Bad》完结 海森伯格终于挂了，完成了传奇的一生。

Day4

雾霾 清晨6点钟就起了，迎着薄雾前往顺义。第一次坐15号线，地铁里空调开得巨大，吹得人脖子疼。下地铁，包了个黑车，100块25公里。9点抵达目的地，号称“五彩浅山徒步步道”。据说是当地政府修建的便民健身项目。起点有个看门的中年妇女，很友好地给我们介绍了爬山的路线，顺义的山有种很强的丘陵感。属于余脉的余脉，这种山给人的感觉就是，如果有雾霾，即使登上山顶，你也在雾霾之中。

全程据说有90公里，我和同伴的计划是爬三分之一，或者一半。由于起得太早我感觉有些体力不支，幸好山路没有过大的爬升，基本是缓度上升和下降。但这个山的问题是植被太少且单一。一看就是近些年绿化种植的灌木和成片的树林，但规划好的路两旁没有什么树，走起来晒得难受，幸好今天天气不晴朗，偶有小风。我们翻过了两座山，吃完东西继续前行。结果到了第三座山路却断了，只能从灌木之中生生踩出一条路来。我不喜欢走灌木丛，里面尽是带有尖刺的植物和巨大的蜘蛛网。走在其中容易让人有迷失的感觉。但这样的一段下山路增加了爬山的乐趣，幸好视野不错，这段下山路花了我们一个小时有余。

复前行，终又看到了山路。爬了一个多小时逐渐下山，看了地图，于是决定从焦庄户下撤，乘车返京。焦庄户，多么熟悉的名字啊。有一个爱国主义教育基地，焦庄户除了地道个人留下的唯一印象就是窝头加小米粥。那也是我童年经历的一次爱国主义教育活动，老师不让我们带午饭，结果中午一群小伙伴看到村民扁担上的窝头和粥都激动得不成了，只有几个心思缜密的孩子，偷着带了几包干脆面，在角落里咀嚼着，一副先知的德意神色。

从顺义回京感觉头晕，脖子疼。回去揉了揉肩膀，觉得是颈椎病复发。北京城里一片灰色，好像那片被风吹走的雾霾又杀回来了，嚣张得很。

2013-08-19 随笔 [星期一 北京 晴]

发表于 2013-08-19 更新于 2022-01-27 分类于 红桃 - 文章

欣赏美味的技巧之一是不去思考美味的原料是什么，而是去想象你吃到的食材努力呈献给你的是什么。你的想象力越丰富，味道就越美好。我弟一直在这方面给了我很大帮助，比如他端上一盘菜，瞪大眼睛看着我：“尝尝！”我拿起筷子，在他的目光中夹起一块肉（或者是菜？）放到嘴里，这个时候，我看菜的目光和我弟看我的目光是一样的，里面尽是对心里答案印证的期许。

“……是鸡？”

“错。”我弟笑笑，仿佛赌钱的人有了三成的胜算。“再猜。”

“鱼？不过口感不太像……”我弟摇摇头，脸已经如同雨后的篱笆，爬满了得意的神色。 “再猜。” 我只好鼓出最后的耐心：“不会是淀粉的吧？”

我记得有很多次都是这样，我弟像赌桌上的服务员为大家发牌，然后一一听取每个人的答案，然后他以厨师的身份公布答案，在众人的“哦噢喔啊”的回应中得到愉悦。有的时候他甚至等不及众人回答，直接就说出了答案，然后加上一句：“怎么样？没吃出来吧！”

在从业的头几年，他总乐于将各种食材做各种尝试，创造出各种奇怪的形状，口感和味道。但他的这一举动并没太取悦嘴很刁的爸爸，爸一句“你这就是瞎做嘛。”揭穿了他烹饪的本质。一开始爸爸总想让我弟做出“饭馆那样”的味道，让他做饭规矩点，不要老“瞎做”。于是弟弟在他面前逐渐收敛，先对家里的厨具进行了一通改革——买了数不清的锅碗瓢盆，摆满来自东西南北调料，换大铁锅，换大炒勺，换大蒸锅，只恨物业管着不能将灶台改为通氧气加压的。几经折腾的结果是让我这个专职的帮厨在刷碗专业上业务突飞猛进。以至于后来出国去了食品厂搬肉，而不是去饭馆刷盘子刷碗荒废了我一身好手艺而遗憾不已。

经过多年锤炼，弟弟练就了一身好本事，端上来的菜总是汤汁饱满，色泽充裕，一看，不，一闻就知道是厨师专业出身，终于让爸爸吃到了浓浓的“饭馆的味道”。虽然我总感觉他的菜做的口味有些偏重，油多。但能肯定的是用的都是好料，还舍得浪费。饭馆里的做饭风格和毛病他都学了下来，一块姜6刀切个方块，其余的都扔在一边，这样的切菜风格跟老外有一拼，尽可能的去其糟粕。我爸是平常不做饭，只坐在桌边等菜上桌，如果不小心下厨房看到他这么浪费，肯定会觉得这饭馆的味道来之不易，同时还得骂他二儿子花老子的钱不知道节约。

“告诉你吧，是豆腐做的——没吃出来吧！”

《小时代》渣。

《了不起的盖茨比》拍的非常绚烂，拍小时代的郭导应该先看看此片，学学什么叫真正的奢华。但此片的奢华是为了反衬人性的苍白，小时代的所谓奢华呢？反映了人性和时代的双充苍白吧。

《巴黎御膳房》 据说是真人真事儿改的。这说明了什么呢？说明好的国家很多事情是可以搬上银幕的，并且是好的事情。法国主流影片总在不同的角度不同的程度上展示女性的美好、坚韧、自信的一面。而我们这个时代影片中的女性不是流于物语便是身在苦难之中。各中差别如同天壤。

《真爱找冤家/完美计划》 法国人浪的一个侧面的特写。比较夸张，一个“第一段婚姻注定失败”传统的家庭的孩子，为了得到一个稳定的婚姻，要先找一个男的闪婚再闪离。这是怎样的精神？当然如同小萌所点评“再多的心机也战胜不了真情。”

2012-08-16 热 [星期五 北京 闷热]

发表于 2013-08-17 更新于 2022-01-27 分类于 红桃 - 文章

夜幕降临，整个城市和城市里的每一个人一样，吸入黑色的空气，吐出暗蓝色的光芒。这光芒上升得异常缓慢，却很快融入巨大而潮湿的水汽之中。仿佛液体从一种溶剂流向另一种溶剂，仿佛细沙流入细石，仿佛人们的渴望融入巨大而绝望的现实。只有草丛中的虫和树上的蝉还在一刻不停的鸣叫——只要鸣叫，他们就抓住了这个夏天，就如同我坐在这巨大的闷热，头顶上的汗水滑到眼睛的边缘，再划过脸颊，脖颈，在胸前与各路大汗齐聚胸前——无非是形式不同，但却都生出了无可比拟的存在的意义。

每当这个时候我都想起王小波笔下未来的“白银时代”。他说“未来是银色的”，这句话和小说本身都充满意向，显得晦涩难懂却十分迷人。我坐在子夜与凌晨交界的酷热里，在王小波未能触及的未来。

在酷热中想象冰爽是艰难的，无论是呼唤可口可乐，还是冰啤酒的名字，都没用。更别提什么爽口糖了。愚蠢的人类啊，你们还在相信那些广告吗？

要写点关于咖啡的乱七八糟的文字。暂列小表如下：

  1. 如何向人介绍好的咖啡与雀巢咖啡的区别。
  2. 如何向人介绍好的咖啡与星巴克咖啡的区别。
  3. 如何向人蓝山咖啡与雀巢咖啡的区别。
  4. 蓝山与猫屎咖啡哪个更牛逼。
  5. 为什么咖啡那么苦，为什么不甜。
  6. 为什么咖啡有酸味，是不是过期变质了？
  7. 我就喜欢喝星巴克的奶油咖啡，我是不是很有品位啊？
  8. 给我推荐一款最牛逼的咖啡，一定要比星巴克好喝。

2013-08-04 观后感 [星期日 北京 晴]

发表于 2013-08-05 更新于 2022-01-27 分类于 红桃 - 文章

看了汤姆克鲁斯的《遗落战境》(oblivion)。 影片设定很好，有点像《月球》。我一向喜欢这样“独角戏”风格的科幻片，这样的设定往往给演员更大的空间。同时因为角色相对少，背景相对简单，容易对人物心理或者性格做深入的剖析和演绎。

影片选择核战后的地球为背景，核战的对象是来自太空的外星人。“人类赢得了战争，却失去了地球”，在这样一个框架之下，离开地球成为了人类的使命，即使对于主角——汤姆克鲁斯饰演的暂时留在地球检查维护巨大的机器和无人机的工人。影片开始就说明了他是被洗脑的，而头脑之中总出现残留的画面成为了影片最关键的一条线索。

影片里的外星人很高明——将地球人吸引进他们内部，洗脑并变为克隆人（最早的本体估计和月球那位不幸的哥们命运一样，都over了）。外星人利用克隆人反攻地球。并制造了巨大的机器通过吸收海水创造能源。虽然人类使用了核弹控制了外星人的进攻。但外星人仍然建立了监控台利用克隆人维持着能源机器。无人机的设定也很不错——外星人仍然没放弃利用地球并且消灭人类。而最早的本体不是别人，正式飞往太空的科学家。

写这些没用的东西干什么？

影片设定虽好，但导演的节奏控制的不好。因为中间我有一段几欲睡去。而且结尾本应有很震撼的冲突表现——就像《月球》里主人公看到巨大的舱体里满是克隆的自己的时候——整个星球上自己的渺小，以及月球与地球遥远的距离产生的巨大孤独感和心理落差。这些令我期待的感觉《遗落战境》都没有给到我。影片虽然制作宏大精致，但却少了最关键的东西。

2013-07-22 最近 [星期一 北京 晴]

发表于 2013-07-23 更新于 2022-01-27 分类于 红桃 - 文章

最近在数着日子过。是入伏后的第几天，或是立秋前的第几天。总之日子过得很快，也似乎过得很慢。在炎热的光景里，不免让人心生无限慵懒和寂寥。在过去多少个这样类似的时光里，我总有一种天长地久的恐慌之感，这个夏天也不例外。我无所事事，为此心神不宁。

罗老师病了。我们通过网络表示了慰问。看罗老师的微博，情况显得非常严重。据说是国图夜以继日的加班让有着“万年花城桂冠诗人兼丰台星座小王子”之称的罗老师也倒下了，还是上医院检查并要回家养病的那种。罗老师夜观天象，说水星要是再逆行对他极为不利，对于罗老师这样眼里不揉沙子的人来说，甚至有一赌气死了算了的危险。网上遇到罗老师，得知他老人家目前情绪还比较稳定。比起前几日“没有家人陪伴不能下地”的骇人，现在的罗老师也能够聊几句天，并表示继续为我国的民主化进程在网端大声疾呼，奋力转载。 祝愿罗老师康健起来。

游泳是我继滑板后在体育界另一个痛。这个夏天，这个痛点又被狠狠地戳了一下。

我递过一个价值10元的纸条，对歪在躺椅上的一个大叔说：您好，我考深水证。大叔接过纸条，说：准备好了叫我。然后继续躺在那里，他发福的肚子令我印象深刻，白白的，好像里面藏了一个熟透的瓜。我说：我准备好了。而后我跳进冰冷的水中，他的目光开始和我的身体一起在水中慢慢游走。到了泳池中央的时候，他站在岸边朝我喊道：你别测了，你根本不会游泳。你这是在走啊。我点点头，对他说：被你发现了。 我考深水证的游泳馆拥有大概是专业长度的水道，这让整个场馆看起来非常巨大。泳池分为浅水区和深水区。浅水区水深1.2米，所以你能想象将近1米9的我在里面的话会是多大一个笑话——旁边还有无数路都走不利落的小崽子。所以我硬着头皮又多花了10块钱，以考深水证的名义走进了深水区，深水区1.4-1.8米，我很顺利的走了两个来回，以至于自信到觉得可以去考深水证了。但结果告诉我，取得深水证的条件之一并不仅仅是完成全程200米，还有重要一点是“不能靠走的”。

虽然深水证没有考下来，但我失败后他们倒没有马上撵我走。我安安稳稳地在水里泡了两个小时，我觉得这是游泳馆的一个盈利的策略——你不可能跟那些小崽子争着用一个小的可怜的浅水池吧，但没有深水证你就进不来深水区。所以你就多花钱考深水证吧。反正你也考不过，并且——欢迎补考哦亲。

  1. 我发现我仍然对即将到来的倾盆大雨激动不已。
  2. 我拥有的魔法之一是让靠近我的人都无法使用“立领”。因为一开始总不太理解“把领子立起来”这种所谓时尚的衣着方式，如果不是颈部有不能见光的隐秘或是脖子太长，短领不足以防寒，立领总显得出门时过于匆忙而忽略了的细节。所以当我使出“bulandenkuqia”这个魔法的时候，他们的领子都会变得软塌塌，折得服服帖帖。——不用感谢我，这是你们应得的。
  3. 我拥有的魔法之二是他人手机自动读出他们正在看的文字。如果是漫画，就读出漫画中的人物对白，或者旁白，就像俄罗斯那边的电影一样，一个厕所上标注的“MEN”也被用正宗的俄语念出来。当然我的魔法暂时只支持中文。这样当我好奇某些人正在阅读的手机内容的时候，我就说“Liralaconte！”所有人就都能一起听到啦！
  4. 我拥有的魔法之三名字叫“尊贵的陛下”，当然咒语本身是另一个词汇。我生活的奇怪世界里，地铁里总是广播着“让我们一起抵制乞讨卖艺等行为”，这让我很是困惑。因为在我曾经驻足过的另一个国度，乞讨和卖艺是不被禁止的，人们总是（且应该）对他们报以微笑，或者礼貌的走开。而不是鄙夷、怒目而视，或者真像广播里说的那样——“抵制”。所以每当我说出“DonnaRespe，la”的时候，所有人起立对刚走进来的乞讨者行脱帽礼10秒钟。我们不想给予金钱的时候，可以给予他们最为尊贵的尊重。
  5. 与父母玩了一路拍手游戏的学龄女孩，一路上发出单调且干涩的笑声。我惊讶于小孩无比的兴奋和他父母毫无约束的笑容，小孩很轻松地将家庭的欢愉气氛传播到了整个车厢。很远处的一个老太太皱了眉头，显然对小孩子的巨大的笑声表示不可思议。孩子的母亲坐在旁边的作为，脱下鞋子把脚放在对面男人的腿上，以一种晚餐后斜卧在沙发上的疲惫神态享受着眼前的美好时光。公交车上，他们三人占了四个座位，那余下的座位上是今天购物的结果。世俗生活的美好一天。
  6. 在婚礼这种场合我不善言谈，不解风情，属于赔笑型选手。
  7. 我身旁做了一个中年男子。他看着车窗外。有一队骑行的人飞奔而过，胯下的自行车驮着骑手艰难地向山上冲去。他“嚯”了一声说“好多的车啊”后欲言又止，仿佛他的记忆中被唤起了某些共鸣。但看了半晌后只说三个字：“闲的啊。”
  8. “没有之一”将会是下一个被用到令人感到恶心的词语。我的意思是它已经让我感到恶心了。
  9. 当我身处某一个环境的时候，我总想写点什么记录这个环境，反应我自己的感受。而地铁就是一个令我感受非凡的地方。
  10. 我曾有过一种很悲观的想法，也许是科幻小说看多了，对我身为人类这种碳基生命体产生了相当的不信任。任何病毒都能轻而易举的侵蚀这个身体，导致机体的灭亡 ——而其中的不同只在于灭亡的方式。我们的肢体只有有限的局部修复功能，但却不能再生，而即使多加保护，也会因为各种原因患上各种短期的或者慢性的疾病， 而作为人大概只能忍受。所幸我们从这种忍受之中领悟了某些道理，但对减轻痛苦并没有太大帮助。（写于很久之前）
  11. 我的母亲正因为宗教信仰而丧失对生活的兴趣。无论主观上她是快乐的还是悲伤的，我目睹这种现实的发生并且感到无力改变。但我想她应该是不悲伤的，因为宗教信仰告诉她，无论快乐和悲伤都是不应有的。（为此我只能表示她应当坚持她的信仰，只要不影响身体的健康。）

狐狸——“the watch dog”

我们读了那么多书，对那么多的句子表示那么多的感叹。到头来还不过是在感叹自己的生活有多么苍白。

我对我的朋友说：我小的时候炎热的夏日时光里最期盼的是一场瓢泼大雨的到来，中午的时候，天上的云层积得密不透风，好像某扇门被关上隔绝了应该拥有的光明。在忍受这份炎热的时候我们失去了时间的意识，甚至想不起某一阵风从什么时候忽然刮起，而后，天上的世界开始地动山摇，巨大的雷鸣后雨水倾泻而下，似乎是为了和雷音合拍，雨水丝毫不吝惜落下的力量，这一切让孩子们兴奋不已，陶醉于这看似能够毁灭世界的力量随意地玩弄着头顶上的天空。而最美丽的时刻总是在疾风骤雨来临时那样刹那而至，天光忽然洞开，阴云被雨水洗得一尘不染，而此时的夕阳映照出关于童年记忆里，大自然最美的色彩。人们走出屋舍，走到晚霞中，走到和风里，走进充满泥土芬芳的气息中。 我

的朋友说：你小时候是在外地吧。

重读了《鸟，看见我了》，尤其是第一篇《意外杀人事件》和最后一篇《情人节谋杀案》。此时北京正在经历夏天的又一场长达一周的雾霾。《意外杀人事件》我喜欢每个小故事的描写，有条不紊的背景叙述，和最后一刀刀名明快亮丽的终结。而《情人节谋杀案》则是透彻解析了两个绝望的人的心路历程。结尾的那段对话一口气读下来令人异常感动。
