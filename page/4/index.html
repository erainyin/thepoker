大象

犀牛

这世间的动物名字里带「牛」字的似乎都有着很好的脾气。也可能是在这个公园里呆的时间长了安逸了，毕竟有人伺候，有送到嘴边的食物，旁边还有一群长相奇特的鸽子陪着。以至于这样一个庞然巨兽在吃草的时候都能让人摸摸头——只要你不去摸它的屁股——多次旅行我所学到的一点是，对于任何动物，都要尽量站到它的面前或者眼前与它接触或者互动，因为它视野的盲区是危险的，而动物的后腿往往更粗壮一些。对于犀牛我们怀着谨慎的尊重抚摸了他的牛角和宽大的额头——它长着棕色毛的小耳朵忽扇着。令人感到意外的是它额头的温度竟然很高，摸起来热乎乎的，看来它厚厚的皮肤并没有过分阻隔体温。这种温度让额头的皮肤呈现出一种粗糙但柔软的质感。

犀牛(学名：Rhinocerotidae)：是哺乳纲、犀科动物的总称， 有4属5种。是世界上最大的奇蹄目动物

小熊猫

别人撸猫咱撸小熊猫。扫除了对于小熊猫的两个误解：1、小熊猫并不是熊猫；2、小熊猫并不小。第一点明眼人都看得出来。对于第二点，小熊猫趴在你腿上的时候你会觉得这个毛茸茸的家伙个头真不小，脑袋有这么大，爪子有这么宽，尾巴有那么粗。同样的规矩，只要你不摸他屁股，从脑袋一直摸到后背的感觉真是顺滑。也可能是天天在景区值班给喂得太好了，对于和游客互动它已经轻车熟路，甚至会主动伸手抓住你手里的木棍，咬住木棍上插的苹果。

小熊猫(学名：Ailurus fulgens)：是小熊猫科、小熊猫属的一种动物。

为什么这里的梅花鹿没有梅花？用何队的话说：都赖时间。因为时下是冬天，它们换上了冬天的素装。待到山花烂漫时，它们就换上梅花斑点装了。

倭蜂猴

这家伙属于懒猴科，名不虚传。在我们头顶两米不到的树丛里，树下面游人穿梭，绝对算不上安静，甚至可以说是嘈杂。但这位哥就是稳如泰山，将头埋在双腿中间，形成一个毛球，一动不动。有不懂规矩的游客在下面摇树，它也只是伸一下爪子挠挠。据说倭蜂猴在野外更加难以寻找，长时间静止甚至导致身上长满苔藓。如果你在树枝上看到有一团苔藓在慢慢地动，基本可以断定就是这个小家伙。生命在于静止，人家贯彻得好啊。

速写云南-雨林篇

发表于 2023-01-14 更新于 2023-02-05 分类于 红桃 - 文章

ᩈᩥ᩠ᨷᩈ᩠ᩋᨦᨻ᩠ᨶᩢᨶᩣ

西双是傣文“十二”的意思，版纳指的是“千亩田”。

北纬22度46分，东经100度58分。暖湿气流充盈的热带雨林，即使在一月份也绿意盎然。西双版纳东北部20公里，普洱基诺山基诺族乡，从这里进入雨林。和所有的探险一样，没有亲身走过的路都是危险的，队伍以1:4的人员配比配置了当地的村民作为向导和支援，向导都很年轻，肤色发红，身着当地人的服装，话不多，腰间都挎着一柄锋利的开路刀。

芭蕉

我们途径了一大片芭蕉林，相比雨林里的参天巨木，芭蕉植茎粗壮，叶子巨大。贪婪的吸收着上方漏下的有限的阳光。所谓香蕉树其实并不是树，几乎是陆地上最常见的最巨大的草本植物之一。但因为过人的高度以及产出美味的芭蕉果实让人们喜爱。向导说芭蕉是雨林里的宝，除了可食用的芭蕉果实，叶子也可以砍下来卷成容器来盛水或包食物。向导一刀向芭蕉茎中间砍下，掰开取出一根白色的芭蕉心，切成小块分给每个人。切断芭蕉心像藕一样拉丝，放到嘴里咀嚼有水分析出。虽然口感青涩，但是雨林中宝贵的食用水来源。芭蕉生命力顽强，砍断的芭蕉散落在雨林中，不久就会重新长出来，这点倒很有草本的特点。

雨林分层

热带雨林是什么？讲解的老师说这个名字里其实已经包含了其3个特点：热、雨、林。身在雨林之中你才会觉得人的渺小，也会对在这里生存的生物产生敬畏。植被分层是雨林的几大特点之一，那些参天巨木通过高度争取了最大量的阳光，并且会在自己优势的高度上尽可能将枝叶盖满天空；而下方的植物则见缝插针地抢夺着珍贵的阳光，通过巨大的叶子扩张领地。而下方溪流间的花草灌木各显其能，利用空间、土地、环境甚至动物来争取更大的生存机会。另一个有趣的事实是，雨林的植物并不拼寿命，那些巨大的树木可能并没有百年的寿命。他们只是生长迅速。

生存智慧

在雨林里任何奇特的现象都有一个统一的解释，那就是为了生存。见到这棵树的时候人们都会发出巨大的感叹。不仅仅是因为它的高大，因为高大只是结果，成就这样高大的是树皮上那厚厚一层如逆鳞般的树皮。这是应对生存绝境的法宝。没有一种生物愿以自己的躯体成为另一种生物的温床，但在雨林中，严酷的生存环境为那些随风飘落的种子创造了附生于其他植物的机会，你经常能看到在树木躯干上盛开的兰花，或者松萝、榭寄生这样的植物，他们的生长并不完全依赖土壤。而这棵树的树皮逆生，使得种子落到树干上直接滑落而无法扎根，把宝贵的营养留给自己，这就是生存的智慧。

营地

从一片密林中穿过，脚下是一条小溪，潺潺的流水旁是泥泞的红土地。向导说这里是野猪的澡堂，还有这里，你可以看到野猪的脚印和被他踩得稀烂的植物根茎——但在我这样的外行看来，眼前就是一片泥泞的土坑。走过了一片两米高的草丛，下坡之后一棵巨大的折断的树横在我们面前，好在路和树木之间有一个可以穿过的空隙。也许野猪在洗澡之后就从这里走掉。我们低下头一个一个从树干下穿过，再次抬头，眼前豁然开朗。如此巨大的一棵榕树！而那下面的一片空地就是我们的营地。

在雨林之中能够舒服地坐在天幕帐篷下，喝上一杯当地人用不知名花朵酿制的紫色饮料，再吃上一席地方特色的长桌宴，这在十年前我是不可想象的。十年前我和小萌游览云南，拉市海骑马之后在村长用一顿简谱却美味的便餐招待我们，形成了我们对云南美食最美好的印象之一。而今天我们能够带着孩子，在这样的大山深处享受热气腾腾的美食，只能感恩自然和社会的馈赠。当然我们并不仅仅是为这一餐而来，更重要的是体会和感受这个充满植被的空间。以及……爬上那颗大树。

树降


所谓树降就是从这棵巨大的榕树上降下来，但要下降就必须先上升，如何上升呢？就必须徒手攀爬。当然这个活动不是冒险，更不是拼命。我们以最大的尊重面对这雨林里的庞然大物，通过定滑轮，缆绳、头盔、护具以及两个身强力壮的领队以身体做秤砣，用尽全力把孩子一点一点连拉带拽，送到标记的高度，完成一次我们自己都不太相信的攀爬。更令人啧啧称奇的是领队不仅护送了孩子，还护送了我们这群大人，爬到半途我的胳膊已经没有了力气，就像王小波说的，到了这个年纪，最大的体力活都是在床上做的。以这样的肉体凡胎冲击这样的高度真是巨大的挑战。虽然顺利登顶，但我承认下来以后腿有些软，脸有些白，身体有些虚。可能是过度紧张导致的低血糖——我给自己这样诊断——中午的饭白吃了。

年终总结

发表于 2023-01-10 更新于 2023-01-27 分类于 红桃 - 文章

2022年如期而尽。

在新的一年的朝阳里，是时候做一下总结了。如同曾经说过的话写过的字，此刻正是公历新年与农历新年之间的一段时间，一个容易把日期还写成2022的过渡阶段，生理上已经进入2023但仍在农历腊月末尾，心理上仍然属于旧历年的百无聊赖的时刻——所以有些不可避免地流水账，就像这开头的一段，已经在我的脑海里流转了几日，终于还是要写出来，作为一个开始，为一个结束起一个头。

工作

2022年我的工作虽然努力，但没有什么亮点。可以归咎于大环境、疫情，以及诸多不可抗力。除了本职工作，副业的项目有所减少，不可否认疫情对我的客户都有影响。而设计似乎永远只能锦上添花，却无法做到扭转乾坤。为此我只能在做好自己手头的事情的同时祈求疫情早日结束，我的客户们都生意兴隆，新的一年需求增加，大家共同富裕。

本职工作方面，22年是我到美科的第五年。五年弹指一挥间，周围的同事换了又换不免让人唏嘘。最大的感悟是工作这个事儿要向内观望，扪心自问，看自己想要什么再评判利益与得失。今年比较值得一记的是我和公司签了第三次合同，按照法律规定，这次签的是不限期的合同。虽然说明不了什么，但也体现了一个打工人的忠诚，和我的客户们一样，我也希望公司越来越好，大家共同富裕。

健康

「一切往好的方向看」 是我这一年的领悟与行事准则，所以虽然副业减少，但好处是留给我个人更多的时间分配给家人和自己。对于健康这至关重要。因为21年底和22年初因为肉眼可见的过劳导致了颈椎病复发，经常头晕，身体免疫力下降，还经历过两次偏头痛发作（我自己诊断的）22年春季随着花粉季的到来，经历了我个人历史上最严重的一次过敏性鼻炎。这一切都是「身体在发信号」——哥们你不年轻了，得注意注意了。所以我审时度势，顺其自然。

先是调整了作息，坚决不再熬夜，子午觉先安排上。然后对颈椎进行以理疗为主的恢复性训练，多抬头，画米字，按摩，换枕头。这么一训练才感觉自己其实并不简单的是颈椎，整个颈肩腰一串都别扭，都得养，都得慢慢恢复。除了修养，另外就是多运动。还专门买了根跳绳，没事儿跳跳。大概到了夏天，整体颈椎有了比较大的恢复。又斥巨资购置了升降办公桌，没事站着办公，别有一番趣味。

今年整个一年我给家里打电话，妈妈都会亲切关心我二子的健康状况，所谓“二子”，指的是脖子和鼻子。脖子刚才说了，鼻子也是持久战，经常性复发鼻炎。现在还多了一个季节性的过敏性鼻炎。氯雷他定也成了我的常备药物。而对于鼻子，今年找到了一个好物：洗鼻器。洗鼻器其实就是带了一根管子的瓶子，里面灌上专用的海盐水。捅到鼻孔里，以水倒灌，将鼻腔鼻窦中的液体冲出，洗鼻后双管感觉明快了很多。并且颅内各窍相通，还可以顺道杀一杀口腔眼部的细菌。我一般是睡前洗一洗，晚上鼻子会好受很多。

对于全家人的健康，今年最终要的议题就是新冠了。政策的一波三折，从近乎整年的严防死守，到年底随着寒冬一道而来的全国解封。我们从适应了健康宝和核酸的生活变成直面病毒的健康责任人，谁都躲不开逃不过。并且在解封的最初一个月，我家五口人就完成了中招到杨康的过程。这个过程有专门写了日记就不再赘述。到写此文的时刻，我们的身体已经从奥密克戎的伤害中慢慢恢复，除了偶有咳嗽和乏力，一切如冬日即将过去的太阳，都在向好的方向发展。我们只希望病毒变异再慢点，群体免疫再快点。生活恢复正常。

旅行

虽然受到疫情影响，但生活还要继续。我们还在今年完成了两次重要的旅程。一次是年初的三亚之行。三亚之行的主题是帆船，小猴跟着教练学习了OP帆船的驾驶知识，我们也跟着体验了一下龙骨帆船的刺激。总而言之，那是一次舒适惬意的旅程，让我们深刻感受到了与寒冬截然相反的自然气候。也深刻理解了为什么那么多北方人要去三亚过冬，世界上最宝贵的东西阳光、空气和风都在那里，都不需要你付出任何东西就可以得到。

另一个重要的旅程是6月底7月初的新疆之行，历时十三天，跨越北疆千里。从伊宁到吐鲁番，从乌鲁木齐到喀纳斯。说实话现在我都无法复述整个行程。就连游记都只写到第二天赛里木湖。新疆之美已留在了我们的记忆中，传说中的西域让我们体会了中国的幅员辽阔。能够带青青体会如此的大山大河让我们倍感满足。

新疆之行花费了巨额的成本，但这一切付出都值得，因为那里已经印上了我们的脚印。我们的记忆里也印上了那些感受。

山岛愁思记2

发表于 2022-12-17 分类于 红桃 - 文章

大概十二年前我还在法国的时候写过一篇《山岛愁思记》，也是记录了病榻之间的一些想法。

有很多个生日都在记忆的长河中满满流淌而去，而今年的生日值得留下一些文字。不仅仅在于年龄上的增加，更因为此时国家的政策巨变，我们当下的生活瞬间也有了一些反应。

12月9日 周五

下午，本来计划前往青青姥姥家吃饭，按照娘家的规矩，有人过生日的话，前一晚是吃饺子，生日当天吃面条。这规矩雷打不动。加上蛋糕，所以晚上肯定有一顿好吃好喝等着青青。怎奈计划赶不上变化，下午4点多接到小萌的电话，说青青发烧了，晚上我只能自己去他姥姥家，吃完给他们带回去。青青可能是前一日跟同学玩得时候没有注意保暖，出了一身汗后被风拍了。结果今天就发烧了。

晚上下班我早早地走了，到家后发现是青青给我开的门！“你不是发烧了么？”青青说“已经不烧了”，“青青说想来姥姥家吃面条”妈妈补充道。哈哈，我的生日终于大家能聚到一起。还吃了巧克力蛋糕。开心。

12月10日 周六

周六的时候青青体温趋于稳定，但还有些咳嗽。小萌说青青周五发烧的时候说浑身疼，脖子也疼身上也疼，然后就发烧了。幸好小猴底子还不错，我们也是直接给他上了连花清瘟+橙子套餐，补充维生素。然后饮食清淡多喝水。控制住了体温。

12月11日 周日

两天后，小萌开始嗓子疼。后来回忆，小萌说其实周五周六的时候就觉得嗓子不太舒服了，当时没有在意。而到了周日发作了出来，嗓子开始巨痛，与此同时身体开始发热。这个时候我们的朋友圈里也有很多人自曝抗原阳性，也就是夺冠了。昨天我们还在刷着朋友圈看乐，今天小萌就开始体验了。家里没有抗原试剂盒，现在也没有核酸点了，现在的状态是薛定谔的新冠。不过用小萌同事的话说：“和攻略里写的一样。” 最典型的是浑身疼痛加上嗓子巨痛和发热。

幸好家里还有我这么个劳力，能收拾收拾屋子，下午的时候小萌躺了一下午，起来吃饭的时候也精神极差。而这一天另一个事实也证实了新闻里说的北京正经历一波噢密克荣的感染——我们叫了几个外卖居然一直没有人接单——那意思就是好多外卖员也都阳了呗。快到晚上7点还没有叫到餐，我最后只能顶着风去外面买了大包带回来。

这一天天气极差，整天多云，没有半点阳光。

12月12日 周一

我申请了居家办公。

因为考虑到小萌的病情和青青的情况，在家里能收拾一下屋子，干点活。经过了一上午的工作，到了下午，我开始感觉浑身发冷，以我对自己身体的了解，这不是什么好兆头。果然，临近傍晚的时候我试了个表，也发烧了，38度9。不过我的症状和他们都不太一样。伴随发烧我主要的感觉是畏寒、肢体酸疼，嗓子并没有巨痛，也没有他们的全身疼痛的感觉，就是头有些疼，浑身酸软，精神不能集中。坐在电脑前办公的时候，腰感觉巨酸，支撑不住。于是我也安排上了连花清瘟，下午也躺了半天。

12月13日 周二

又申请了一天居家办公。

这一天青青基本没什么事儿了，就是偶尔有些咳嗽。小萌的体温有些反复，因为昨晚因为嗓子疼得没有睡好觉。我经历了一个痛苦的夜晚，一直没有太睡着，开始是浑身发冷，盖了三层被子都没出汗，脑子里也不放松，直到快清晨了，才迷迷糊糊睡着，清晨出了点汗，感觉才好些。但是起来之后浑身无力，鼻子嗓子都不痛快。看了看攻略，大差不差时这个症状。问了问群里的朋友和同事，也都有好多人中招。

这一天的精神都很萎靡，头晕脑胀，小萌的嗓子还是很疼，并且开始鼻塞。她把四个手指握成一个球状，球的中心对着自己的下巴——“现在就集中在这里，特别堵，特别疼。”所以她说话都张不开嘴，只能发气声，就这样也几乎是大声地跟青青喊了几回——从调皮捣蛋的情况来看，青青好了至少八成了。

其中一次就是自己在木地板上来回地滑，最后把自己凭空滑了个跟头，这才躺在地上哼唧——本来想骂他，却发现地上居然有一个汗手印儿，看来这下摔的不轻，弄的我们一下又都笑了，他自己看到我们笑，也笑了，眼睛边还挂着刚才哭出来的几滴眼泪。

所以这一天我和小萌几乎是两个半个人支持着整天的生活。

12月14-16日 周三-周五

周三到周五我索性请了3天病假。

实在因为腰肢酸疼在电脑前坐不住。我连手机都懒得看，更别提电脑了。一个细节是周三的下午我头晕晕沉沉，浑身发冷，于是躺在床上打算睡一会儿，躺着的时候带着耳机听播客，这样可以分散一些注意力。但后来感觉耳机的声音有些大，我都一直没有力气把耳机拿下去。一直处于迷迷糊糊的状态。

朔风吹雪满横湖,众鸟归栖日欲晡，看来老天爷让我歇歇，咱就听话做个好孩子。不过好消息是从周三开始小萌逐渐恢复，精神头也好了一些。嗓子虽然疼，但已经比前两天好多了，目前最难受的感觉是鼻子不通气。就希望能鼻子通气好好睡一觉。所以晚上我们三人都洗了澡，热水加成病魔退散，一晚上过后，休息的不错，吐出的痰的颜色都显的清澈一些了。

另外从周三开始天气变得好了很多，外面开始刮大风，刮风意味着吹散乌云，吹散乌云意味着晴朗重现，于是我们能在白天将窗户打开，足足地通上他个吧小时风，让寒冷的新鲜空气在屋里周旋，再请紫外线充分穿透晾晒的衣物。再把我的花花草草加上水，让屋子充满生机。

总结

流水账记录了这几天的感受，虽然我们没有进行核酸检测或者抗原检测。但大概率我们是中招了。而且不光我们，就连青青的姥姥姥爷也都相继发烧，症状都类似。所以应该不是普通的流感。问了一圈朋友们也都相继中招，北京这一波感染实在是厉害。后来据小萌分析，青青上周和同学玩的时候，同学的家长见面的时候就说她家楼里有两个病例。小萌听了心里一紧，您这是带毒来的啊😂。结果他们还请小萌用他们手机给他们录了足足40多秒的视频！——这只是我们可能中招的路径之一，但自从开放以来，病毒防不胜防。哪里都可能有病毒。
