2010-06-12 慢2 [星期六 巴黎 多云]

发表于 2010-06-12 更新于 2022-01-27 分类于 红桃 - 文章

　　年初的时候和小萌去Parc floral de paris玩儿，中午在附近一个日本料理店里吃了顿午饭。我要了一份寿司，小萌则要了一份刺身。这是我们第一次在巴黎吃日本料理。结果上来以后发现量小得可怜，看菜单上图片还以为盘子有案板那么大，结果端上来一看，就一个板擦，上面几个不多的寿司在板擦上倒是显得挺充实，一放到嘴里就没了。再加上这日本子的料理本来就中看不中吃，导致的结果就是吃完了也觉得看都不中看了。

　　于是我们决定自己做刺身，吃个痛快。

　　为此小萌专门跑到了中国超市买了专门的日式酱油和被我称作“傻逼西”而实际名字叫“瓦萨必”(wasabi)的绿芥末，又去法国超市搜索了一番，最后在欧尚买了一盒新鲜上好的三文鱼。以我的经验和所见所闻，三文鱼(Saumon)是人们很喜欢吃的一种鱼，基本上各大超市的水产品区都有卖。而且法国临海，海产品非常丰富，日常的消耗量也很大。很多法国人都很喜欢吃日本料理，没什么油烟，低糖低脂肪低卡路里却营养丰富，符合西方人的饮食要求，而且看起来还干净——尽管现在巴黎大多数的日本料理是温州老板的天下，不过这也没什么嘛，你知道现在法国卖啤酒和烟草的酒吧已经不是奥佛涅人的市场了，还不都是我们中国老板！

　　我们在超市里买的三文鱼大概十二欧，有600克左右。我们在饭馆吃的刺身一份十多块钱，真正的鱼肉少得可怜。所以面对着这么大块的鱼肉我们兴奋不已，忙回了家大快朵颐。这三文鱼果然不赖，做起来简单方便（日本料理也没什么嘛～），吃起来新鲜刺激，这里顺便普及一下日语“刺身”的说法叫“傻西米”，我吃刺身的感受就是：“蘸瓦萨必吃傻西米吃得我傻逼啊 ToT”（苦列娃绕口令得似？）。不过说笑归说笑，我还是在吃的时候想到了两个问题。

　　问题一：为什么鱼肉里没有血而我们吃的其他肉则有很多血呢？

　　问题二：羊肉吃多了身上会有羊肉的腥膻味到，而鱼本身有腥味，日本人吃的很多，似乎也没有把鱼腥味传染到人身上，Why？

　　想了半天还是不得其解，还是说笑吧。

2010-06-10 慢1 [星期四 巴黎 雨]

发表于 2010-06-10 更新于 2022-01-27 分类于 红桃 - 文章

　　昨天终于闲来无事，打开电脑之后玩了一下很多天前下载的一个游戏《土豆怪兽》。这是一个画面卡通、休闲娱乐类的塔防类游戏。小萌问我：“什么叫“塔防”游戏呢？”我解释了半天她还是没明白，最后只好说：“就是《植物大战僵尸》那样的游戏！”她就明白了。明白了这样一个游戏类别的结果是她替换了我继续奋战，直到——次日凌晨五点。

　　要是我妈在场肯定会说：“哎呀，跟你们说了多少次了——不要熬夜，对身体不好！”这种情形让我想起高中时买了电脑，天天玩到很晚。后来工作了，也经常面对着电脑工作到深夜。那个时候我最不爱听的一句话，就是我妈说的：“你还不睡觉？别老玩电脑耗那么晚！”我对这个出于爱心的责备之词意见颇多：一则我不是在“玩儿”电脑，二来我也不是在“耗”。不过到现在自己也完全体会了父母操的心，快三十的人了也有些养生的意识，绝不会无端熬夜了。不过内人小萌却是例外，此女和我同睡一床却用着不同的时差，且向来精力充沛，还特容易兴奋。多年积累的经验告诉我，如果按我妈的标准，爱她就别给她推荐什么好游戏。

　　所以我有些后悔，唉，要怪也只能怪我自己——

　　我也是个闲不住的人。虽然耐心打穿游戏的心越来越少，却还有很多对新游戏的好奇心。这就好比一个上了岁数的男人，虽然不是每个女人都干得动了，但还是不死心。所以对于很多人，女人和电脑游戏一样，“玩儿”在某种意义上是要对自己说：哥们还不老，还干得过！如今的情况就是这样：费劲巴拉地下了一个游戏，安装之后玩不了多久就失去了兴趣，于是一删了之，也号称“玩儿”过了。但具体细节已无从想起，历时心下空空荡荡，此种感觉如同手淫。

　　话说回来，小萌鏖战一宿，奋战通宵，一觉醒来之后兴奋地告诉我：“我终于打穿了！最后那两关还真难！”看到她兴奋的样子也让我感到些许释然。忽然想到如今同龄人之中大有三十不惑之恐，没等到不举就开始怀旧了，这种态度实在让人看着生气。在这点上，我和内人小萌认同一致，小萌对三十岁之后的生活十分向往，认为那才是一个人活力迸发的时期。我则认为虽然人不可避免有时伤感岁月如梭无可厚非，但现在怀旧实在是有些早了，若沉湎于此道真真令我不齿，实在是心智尚未成熟就已衰老的表现。

　　现在的人容易怀旧，拿怀旧当事儿，很重要的一个原因就是没有得到满足。这里的满足既有外在的，也有内在的。之所以觉得以前好，就是因为觉得现在不好。虽然小时候其实生活水平比现在差很多，但仍然觉得当时快乐——其实问题就在这儿，重点就在“觉得”二字。这个行为是外在的，但是原因是心理上的。所以要我来看，与其怅然若失的怀旧，不如想方设法去满足一下自己。觉得小时候游戏好玩，就找点游戏真正玩玩。我就不相信牛逼的技术不会给你带来所谓真正的游戏快感——你说以前的游戏多好玩配置也不高，那是因为当时的技术是真达不到。我们怀旧的时候总容易走入一个误区觉得旧的好，殊不知你怀的那个旧也许是当时最新的东西呢。

2010-06-07 走失的主人 [星期一 巴黎 晴]

发表于 2010-06-07 更新于 2022-01-27 分类于 红桃 - 文章

记件小事儿。

　　今天早上出门前往打工的地方，在路上看到一条白色的小狗。他跟前面的人距离落得很远，前面的人不紧不慢地走，似乎很放心地走在前面。他也没有把狗拴起来，实在是悠闲且自在的清晨散步。我走过去的时候，小狗慢慢在后面走着，循着空气里的气味找寻着自己的地盘，不时在某棵树旁转半天。我奇怪他的主人似乎并不在意，不过跟着走了一段也并不在意。因为之前路上也看到过几个路人和他们的狗，同样没有拴住，保持着距离前行。终于，前面的主人在路口左转，小狗却没有跟上，在路旁转了两圈后跑向了路中央。这时对面开过一辆车，我和司机都看到了这一幕，都放缓了脚步，等着小狗跑过马路，盼着他的主人呼唤着跑来。但这两个景象都没有发生，他的主人径直走了过去，而小狗则在路中央呆了一会儿后返回了路口。这时我已经走上前去，看到安全返回的小狗舒了口气，而此时的小狗似乎有些迷惑，在路口旁边的树脚下闻了起来。终于到了交通灯变绿，我抬脚走过了并不很宽的马路。我仍然有些不放心白色的小狗，于是回头望去，结果发现他竟然跟我走了过来。

　　我这才意识到，前面的那人并不是他的主人，也许他是一条走失的小狗。

　　于是我放慢脚步，因为我前面还有三个路口，我仍然抱着他主人会来呼唤他的幻想，尽管面前路上没有车，但我还是等到红灯变绿，走过路口。如我所料，小狗果然仍跟在我的身后。我要在下一个路口右转，将要去面对一条很宽的街道，这是一条连接着高速路的街道路口，很多下道的汽车从这里走向辅路。我不免有些担心，不知不觉之间我成为了刚才的那个路人，别人也会把我认为成他的主人，而我则在不知不觉之间要开始对他负责了，至少他的任何闪失已经不是他自己的事儿。

　　我和小狗安全走过了第二个路口，这期间我仔细看了小狗：白色，圆圆的脑袋，卷毛但并不长。也许是比熊犬但我不能确定。我对狗没有更多的认识，但这只狗很会令人产生好感。不知道为什么，法国的狗普遍有着很好的“素质”，不会随便朝人乱叫。而这只小狗则表现得更聪明——知道跟着人可以安全地过马路，所以显然，问题出在他的主人。他仍然是一条忠实的狗，走失的是他的主人。

　　面对最后一个路口巨大的车流量，我显得比等待绿灯的司机们更有耐心。因为我不得不一直盯着小狗，但又不想表现得太像他的主人。我想在别人看来这一定是一副很有趣的景象：一个亚洲人带着一条小狗，他和小狗都显得心事重重。看哪，他真不像个称职的主人，甚至都没有栓住他的小狗，万一他的狗跑掉了怎么办？我想说你们的担心很合理，因为我也在担心着。人不要总担心某些事情，因为很多时候这些担心都会成真。在等待红灯的时候，小狗居然要跑下台阶，还好这时候车堵地很厉害，司机们都不敢开动，看着我。我赶忙伸出手，招呼“我的”小狗。回来，小家伙儿，这儿，到你主人这儿来！小狗在路边呆了一会儿，终于看到众多视线之中我的目光——那个最为关切的目光。然后走回到我的身边。

　　我喘了口气，忽然心里有了一个想法：要是下一个路口之后你还跟着我的话，那我就把你带回家，我成为你真正的主人。

　　终于，我们等到了红灯结束，我慢慢走过路口，又走过了另一个路口，小家伙跟着我顺利通过了马路。我就要转向左边一条小路，这是一条没有路口的小路，他要是一直跟着我到了打工的地方，我就给小萌打电话接他回家。但我仍然表现的和一个路人无异，人们只能通过我们之间的距离猜测我们的关系。忽然一种责任而生出的权利感让我心中有些忐忑，虽然没有了车流如注的路口，但小狗会不会走到我的身边继续跟着我前行则并不得而知，我的心理自然而然地产生了一个距离，这个距离用来考验作为一只狗对于新主人的忠诚。

　　我向前走了几十米，当我再回过头却见到小狗并没有跟着我，他在远处的草丛中继续寻找着味道留下的踪迹，显然他并没有如我所期待那样向我跑来。他以独行的状态保持了他的尊严，他仍然是一条聪明的狗，依靠我走过几条汽车川流不息的路口，而后继续去寻找他的主人。

2010-06-05 科隆大教堂到底造了多少年？ [星期六 巴黎 晴]

发表于 2010-06-05 更新于 2022-01-27 分类于 红桃 - 文章

最近听了若干个节目，都提到了欧洲的教堂，且都不约而同地以德国的科隆大教堂为例，但是几个人说克隆教堂的建筑时间都有不同。袁腾飞说造了600多年，《冬吴相对论》里说科隆造了200多年，还有一个节目说造了300的多年，真是“各有千秋”。其实对于这个教堂我还不算陌生，去年夏天跟团欧洲四国游的时候还去参观过。不过当时全然没有注意导游对科隆大教堂的讲解，印象中她好象提了教堂的高度，但也许是一路下来看了太多的教堂，有些审美疲劳了，所以实在没留下什么印象。后来觉得这种跟图旅游真有些令人望而生畏，不仅什么都看不仔细，记不清楚，还豁逼累。正所谓“上车睡觉，下车看庙，车停撒尿，吃饭拍照，回来一看照片，啥也不知道”。

这几乎是我最不喜欢的旅游方式，但作为生活水平初级阶段的我们，也没什么选择的余地。毕竟这种打包旅游的好处就是能节省出大笔的交通和住宿费用，当然，代价就是——跑吧，哥们儿！

我查了一下，正确答案如下：

科隆主教座堂（Kölner Dom，全名Hohe Domkirche St. Peter und Maria），是位於德國科隆的一座天主教主教座堂，它是科隆市毫无争议的标志性建筑物。157米高的钟楼使得它成为德国第二（仅次于乌尔姆市的烏爾姆主教座堂）、世界第三高的教堂，另外也是世界上第三大的哥特式教堂（前两位是塞维利亚主教座堂和米兰主教座堂）。它從13世紀中起建，工程時斷時續，至1880年才由德皇威廉一世宣告完工，耗时超過600年，直到今日仍然修繕工程不斷。艺术史专家认为它完美地结合了所有中世纪哥特式建筑和装饰元素，因此聯合國教科文組織于1996年将其列為世界文化遺產（2004年科隆主教座堂被列入瀕危世界遺產名录，2006年脱离该名录）。

游戏推荐 / 策略《远古帝国》

发表于 2010-06-04 更新于 2022-01-27 分类于 方块 - 声色犬马

最近玩了一款策略游戏，感觉很不错。虽然画面一般，但比一般打打杀杀的游戏更耐玩儿。《远古帝国》 是一款以远古文明为主题的模拟策略游戏，从苏美尔到古埃及，从印度到中国，从希腊到罗马，远古帝国将带你进行一次全面的旅程穿越早期的古代历史。

  • 画面： 简洁的2D画面以多种颜色划分区域，对占领的国家数量，士兵人数等具有清晰的列表显示。在不同的战争中不仅地图变化丰富，还刻画有战役标志表现主题。
  • 声音： 每场战役中的背景音乐具有不同的民族风情，游戏中虽然没有惊心动魄的战斗画面，但战斗时的兵器碰撞音效也营造出激烈的战争氛围。
  • 上手度： 蓝色代表的玩家所控制的部队，在每个回合中你可以在不同区域扩充士兵数量，选择进攻或者防守，最终以占领全部领土为目标。查看菜单中的说明选项可获得更多帮助。
  • 创意： 《远古帝国》和《美国战史》同为Sillysoft Games所开发策略战争游戏，采用了类似玩法的规则。在本作中设计了更多的对手国家参战提高游戏难度，并加入了奖励和道具卡片等元素丰富游戏内容。
  • 可玩性： 游戏中包括58个远古国家供你支配，12场战役将被逐渐解锁。选择士兵、指挥官、王者等4种游戏难度，运用巧妙的策略获得战役胜利，沉浸于 Ancient Empires Lux将带给你持续的乐趣。

2010-06-02 随听随想 [星期三 巴黎 晴]

发表于 2010-06-02 更新于 2022-01-27 分类于 红桃 - 文章

在听一个名为《东吴相对论》的广播节目。由于节目涉及了经济管理的内容，算是我的老本行，所以有些兴趣。节目做得很有意思，其实很多观点都很基础，但很贴近生活。比如讲到中国的房市之热导致的一系列问题。令我印象深刻的观点是对于中国整个商业创新能力的负面影响。

相比起其他产业，在中国目前最盈利的产业大概就是股票和房地产了。圈地卖钱和屯房置业是最快和最保值的获利方式，但房地产经济过热所带来的影响则是巨大的。节目提到一个观点，就是一个国家经济产业的生命力在于创造和创新。美国的经济之所以健康，因为很多人在很年轻的时候就开始创业，并且政府和整个社会的环境都支持鼓励创新。而目前的中国，一切创新都面临巨大的风险，何况相比房地产，自主创新的效益和利润又小的可怜。

并且，就产业本身来讲，一个创新的行业可能创造更多的利润点和就业机会，对于整个社会的循环都是良性的。而一座房子在建好后除了土地本身的价值，则不会有什么创造就业机会和价值可言。土地之所以永远涨价，是因为土地本身的稀缺性造成的。而这种稀缺性并不能造成经济的良性循环。并且房地产过热这种情况会使越来越多的商人将资金投入进去——对于一个已经事业有成的商人，很可能撤出一部分资金，只保留核心业务，将中心移到房地产上而非对新技术的创新和风险投资上。因为比起管理和提升一个企业的价值，经营房地产既轻松又更容易获利。但这对于整个经济的循环所造成的后果将是恶性的。

我对这些观点深表同意。

___另一个观点___

节目有很多集讲创业、领导和如何管理一个团队。这勾起我一些旧事。

应该有某些东西可以超越时间。

人们在创造了“时间”这个概念之后便自动地成为了它的奴隶，一切都在被这种拥有我们称为“永恒”的特质的东西所笼罩。但事实上，时间本身甚至不具备任何形式。于是人们在这个概念被创造之后的数千年时间里，都在寻找着某一样东西试图打破时间的包围。无论社会如何向前推进、发展，人们的速度如何加快，人们寻找的脚步从未停止。从哲学到科学，从有形到无形，从定理到推理，从大地到天空。人们从试图寻找一种永恒逐渐走向试图创造一种永恒，但似乎没有任何结果。最后，人们终于意识到“超越时间”这个想法本身就是一个无法达到的目标，或者说，也许只有通过创造另一个概念才能实现。于是这个工作落到了艺术家的身上，而在他们努力工作的同时，这样东西变得越来越清晰可见。艺术家通过他们的工作创造了另一个具有超越性的概念“美”，而就在这个概念建立的同时，时间变得空洞失去了本身所有的意义。

我陷入一个奇妙的思维循环，从一开始的那幅画，到紫色的薰衣草，到梵高笔下的向日葵。事实上我要回到最初的主题“普罗旺斯”。

我的印象之中法国在地理上被一分为六，整个东北部都是巴黎及巴黎周围，东部则是阿尔卑斯山，西南和西班牙接壤的比利牛斯山，中央高原，南部就是普罗旺斯和蓝色海岸了。但普罗旺斯到底有多大也许没人说得清楚，也许所谓“普罗旺斯”就是你手指向南方，那片被紫色覆盖的土地，如果你还不确定，就抬头看看天空，阳光毫不懈怠地倾洒下来证明这一切都无需证明。事实上普罗旺斯在地域行政上包含了五个行省：

  • 隆河口省Bouches du Rhone，省会为马赛
  • 沃克吕兹省 Vaucluse，省会为亚威农 Avignon
  • 上普罗旺斯阿尔卑斯省 Alpes de Haute Provence，省会为第能雷半Digne les Bains
  • 阿尔卑斯滨海省 Alpes Maritimes，省会为尼斯 Nice
  • 瓦尔省 Var，省会为杜隆Toulon

我们旅行的起点就是Avignon。

南方之前01

发表于 2010-06-01 更新于 2022-01-27 分类于 红桃 - 文章

从今天起，我准备开始写一些关于旅行的文字，这些文字是摘抄，是一些感受，或者一些幻想，也是对即将的行程做一个更加充分的准备。因为这个夏天到来的时候我即将前往法国南部，开始一段旅行。所以我要记录一些文字在南方到来之前。南方！南方！你知道我的意思——那意味着“普罗旺斯”。

　　先从一张图片开始吧，当我在网络上搜索“普罗旺斯”这四个字，或者法语“Provence”的时候，最先映入眼帘的便是这样的一些图片：

　　广袤的法国南部平原上，盛开着大片紫色花朵的植物整齐地铺排在平坦的田地里，金黄色的阳光从某一个角度斜射下来，使这一大片田地显得更加清晰。你可以看到远处连绵的山脉和散落在一片片树丛之间的房舍。在田地的中央一座简单到极点的石头房子因为反射了太多的阳光而显得越发耀眼，你不由得重新看到周围的花田，这时你才意识到弥漫在空气之中的清香——但事实上，当你踏上这块土地的时候，你已在这种味道之中，而这时的感觉更像是加强或者说：唤醒。

　　薰衣草（Lavender）作为普罗旺斯的标志几乎掩盖了一切成为了“唯一”的特点，因为她的独特、靓丽和视觉上无法抵抗的巨大。面对这样的景观，人们如同面对巨大的珊瑚，这些完全由微小个体组成的庞大事物对“大”与“小”的概念本身就是一个冲击。而薰衣草则更具有超越自然的魅力，这是人工造成的景观、只是利用了大地、土壤、空气、流水以及太阳。人们只是做了妥当的安排，让自然的生命焕发出不可抗拒的美。但事实上对于我，普罗旺斯的意象并不形成于薰衣草独特的紫色。在普罗旺斯只是我头脑中若干个梦一般的目的地的时候，它有着更加浓烈的色彩和压迫感。

　　“明年春天，大概是2月吧，我将动身前往法国南部，听说那里的光线充满变化，景色常常呈现灰蓝的色调。”──文森特·梵高。

　　事实上并不是这样的。或者说，也许那种景色只属于二月。比起那些深入大陆的国家，法国的春天短暂并且难以令人察觉。在几次不大的春雨过后，阳光会迫不及待地冲破云层。这个时候来自地中海的暖湿气流在最快的时间里滋润这片法国南部的土地，这是和寒冷潮湿的大西洋气流经常光顾的法国北部以及荷兰如此不同的感受。而当他来到这里的时候才发现自己当初的猜想有一半正确——灰蓝的色调是土地、是紫色的薰衣草，或者某些时候的天空，但这绝对不是这里的主色。还好他另一半猜测不错：这里光线充满变化。对于一个印象派画家，还有什么比光本身更重要和更有魅力呢？而这里简直就是为光线本身所造，土地平坦但并不僵硬，视野开阔但仍有远山阻隔，除此之外，就是他需要的一切。于是他走进了更加浓烈的色调之中，金黄的土地，一望无际的向日葵。这一切都拜一样东西所赐，太阳！太阳！

　　对于我，普罗旺斯应该是金黄色，热烈到令人痛苦。有人说梵高之所以发疯正是因为这浓烈的太阳，长时间在烈日下作画对谁都是一种折磨，何况还有肆无忌惮干燥的风。但这里成就了梵高艺术追求的最高境界，他用画笔表现出了光线所刻画出的各种颜色，并从中悟出了生命的意义。我想普罗旺斯之所以令人如此印象深刻，梵高功不可没。想想一下梵高笔下普罗旺斯深邃却躁动不安的星空吧，是怎样一片土地才能支撑起这样的天空呢？
