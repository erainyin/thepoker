2009-11-08 A firend of NE [星期日 巴黎 阴]

发表于 2009-11-08 更新于 2022-01-27 分类于 红桃 - 文章

我的朋友必定要陆续悉数登场，那么，现在轮到你了。

　　我在巴黎打工的地方认识了一些朋友，这些朋友都和食品有关，比如包包子的大庆，包飞碟虾的东亮，包春卷的小胖，做包装的小马，以及和面的阿Z。事实上 他们都是留学生，来自五湖四海，正所谓“年轻的朋友们巴黎来相会”，我们相聚在了同一个食品加工厂，做着不同的工作，为巴黎市民生产着他们心爱的食品。这 件事情本身就很有趣，也很荒谬，荒谬之处就在于虽然我们都是打工仔，都是留学生，但我们对其他人的了解和对自身状态的认知是不同的。这是留学生和那些已经 在巴黎扎根的亚洲移民的区别，但也是留学生与留学生之间的区别。

　　这样的好处是你可以接触到很多人，很多不同的人——包括一些你并不喜欢的角色。举例来说比如在工厂吃完午饭，那些潮州人温州人组成的老烟民俱乐部就在 食堂里开始抽烟，而留学生们也单独坐一桌，这两桌人年龄不同，面目表情不同，谈话的内容不同，想法不同。根本上就是两个世界的人，如果我们真属于所谓第三 世界的话，我想这其中也有人不会介意再划分出一个第四或者第五世界来。我就是拥有这样想法的人，因为我发现我无法融入他们的谈话，也不想融入，我在这样的 环境里除了有这样的主张以外，就是默默地坐在旁边看着他们，用这种方式表达一个简单的意思：“我不想理你们，你们也不要理我”。

　　在这些人当中，阿Z是和我交往最多的人。此人姓Z名Z，东北人士，籍贯锦州。用柬埔寨华侨的命名规则来说，忽略姓氏冠以“阿”字，故名“阿Z”。从相 貌上来看他正是标准“身高八尺”的东北大汉，其可以加到他身上的形容词还可以有“方海阔口，通贯鼻梁、大耳朝怀”等等。所以在一派由低矮蹉跎的南方人组成 的人群之中，他的形象算得上玉树凌风，算得上给工厂拔份的。通过他，我有机会了解一个东北人，了解一个赵本山小沈阳之外的东北青年。

　　总的来说，他是那种热情的人，喜欢和别人打成一片。我觉得这是东北人性格之中比较突出的特点，他们善于并乐于交际，因为这样更容易形成和扩大自己的朋 友圈。他们不像上海人和北京人那样高傲和排外（至少表面上给人这种感觉），他们可以以一个最平民的姿态进入到别人的视野，表示出热情，传达“你好我好大家 好”的信息。因此他们可以在很短的时间内熟识并获得别人的认知。阿Z代给我的另一个感觉是直率，和上海人的计较、北京人喜欢耍心眼儿相比，东北人更喜欢直 接表达出自己的意思，尽管并不是所有人都乐于接受这种表达方式。他们喜欢在别人面前发表夸张的言论，就像赵本山的小品里所表现的那样，但我发现这种大大咧 咧往往是经过深思熟虑的，换一个角度考虑你就会发现，这是一种掩饰，以便更好的接近别人，当然更多的时候这个行为方式已经成为了条件反射，无论你觉得多么 做作，但他们已习惯成自然意识不到了。

　　其实上述分析都无碍于与一个东北人成为朋友。他们外向的性格特征并不能说明人品好坏，但这更容易为彼此了解创造一个相对轻松的氛围。以我的性格来说， 我并不容易轻易和某人成为朋友，但表面来讲我会尽可能的平和，这让人看来我并不难接近，但并不容易相处，我尽可能地倾听意见而尽可能不发表意见。所以对于 一个喜欢自来熟的人来说，我会让他们觉得有些城府。（这个分析真有意思，我得意思是说，啊哈，我们都错了！）事实上我并没有什么兴趣了解什么人，除非我对 他感兴趣。而我之所以在这里长篇大论写一些关于阿Z的评述，原因也是我通过他了解了一些有趣的事情。这个开始也是平淡无奇的，因为打工的人群里，阿Z和所 有人都显得很熟，我和所有人都显得很容易相处，而他又是唯一一个可以和我身高比肩的年轻人，无论我戴眼镜而他不戴眼镜，无论他是否明显比我显得健壮，无论 我的发型和他的发型有多么的不同，在那些越南人泰国人柬埔寨人温州人潮州人浙江人广东人甚至非洲人眼里，我俩就是一对儿好兄弟。当然，比起其他留学生，我 们俩在同一个车间工作，可以经常见面，有更多的机会可以聊天。

　　我忘记了我们到底从哪天开始谈话，但是后来事情就变成了这样：我们成了工厂里所有人眼里的亲兄弟好朋友。这是有趣的地方，也是荒谬之所在。我要用一句 离异夫妻常说的话来为自己澄清：“事实上，我们还并不了解对方。”我对他了解的情况是这样的：他有女朋友，也是家乡人儿，并且不久就会结婚了。他们初中就 相互认识，并一起结伴来到巴黎。同甘共苦到了今天。除此之外，我还知道一些细枝末节，但并没有写出来的必要。因为无论是否写出，都无法增加了解的深入程 度。我只能从外貌以及外貌能延伸到的部分进行描述，但幸好，我还看出一些表层以外的东西。

　　阿Z是一个性格开朗的人，但这并不能掩饰他内心的虚弱。他虽然有不错的外表，但在法国立足对于他来说并不容易——他的女友是个能力很强的人，客观上给 了他很大的帮助；而他的外语和其他方面能力并不出众，但在我看来，他有些过于依赖他的女友，这种依赖是正常的，但也因此他缺乏一种独立的精神，在某些时候 他会表现地不那么大胆。

　　起初我总有这样一种担心，我能够想象到别人看到这篇文章后的感受。但后来还是写了出来，我有这样的想法，不怕也不应该怕别人怎么想。我对自己的记忆力保持着始终如一的无信心，我需要文字帮我重新建立那些印象，包括我的朋友们。

(注：这篇文章还没有完成，因为在写作过程中我不断受到其他想法的干扰。我会一点点写下去，直到我满意为止。)

2009-11-05 入手 [星期四 巴黎 雨]

发表于 2009-11-05 更新于 2022-01-27 分类于 红桃 - 文章

用句潮人的话说：“今天入手了一枚法国2009年限量发行的10欧元硬币。”

先放图

前两天看报纸，说法国2009年发行了10欧元的硬币，现在可以通过各大邮局去咨询购买了。于是查询了一下邮局的网站LaPoste.fr，果然看到了相关的介绍。由于今年我开始收集欧元硬币（详见“+PlanE欧元硬币收集计划”），便有了去换一枚10欧硬币的想法。何况10欧的硬币还是限量限期发行，全法国只有1000家邮局有售。

——为什么是“有售”而不是“有换”呢？仔细看了一下介绍才知道。这硬币并不是直接兑换可以获得的。而是需要在邮局购买物品后以找还零钱的方式取得。这就 意味着10欧的硬币你需要准备更多的钱才能换到。我想这么做的原因是为了抵消铸造新币所产生的成本——为了收集就只能花钱，钱虽然不是花在硬币本身，但是 由此带来的利润可以一定程度上提高收益。

10欧硬币需要去Bureaux de Poste换取，就是大一些的邮局。我前几天去香榭丽舍大街上的一个小邮局都没有，于是查了一下网站，看到了 1000家邮局的列表，发现有一家离我家只有一站地之遥。于是今天就前往邮局。我花了1欧多买了一个含50g法国境内邮资的信封（没想到居然这么贵），给 了邮局工作人员一张20欧的纸币，于是得到了一个10欧的银币(:))，不过遗憾的是我被告知，5欧元的硬币他们那里没有——因为是去年发行的。如果想收 集，就要去专门发行硬币的单位去碰碰运气了。尽管如此，我还是很满意地拿着10欧硬币回家了。此行目的已达，10欧银币入手！

回家以后仔细看了一下，发现其实10欧硬币做工并不是很精细。虽然是银质，但看起来显得很单薄。币面直径倒是不小，比2欧硬币大出两圈。图案也并没有新 意，还是法国的标志性的“播种者”形象。正面RF(法兰西共和国)缩写的字母中间有2009的字样，背面的面值周围是法国国家的口号：自由博爱平等。其实 总的来说，感觉有些失望，也许是只有一枚的缘故。法国计划发行的大面值纪念币这才刚刚开始，从5欧到500欧还有很多值得收集的，不过如此看来，这后期的 开销可太大了，500欧一个金币，谁敢随便拿着到处溜达啊。那可是5000人民币啊！

2009-11-04 不能忍受 [星期三 巴黎 多云间晴]

发表于 2009-11-04 更新于 2022-01-27 分类于 红桃 - 文章

网上有很多朋友发结婚照片，我们都会抱有极大热情地去审查，但令评委们满意的目前还没有超过10张照片。不客气但客观地说大多数照片都很俗气没有新意，少有惊艳或自然生动之作，那些图便宜追求性价比的婚纱照更是落得一个不可目视的结果。

　　一圈图片看下来，觉得水平这东西真是落差很大，很有名的婚纱店拍出来的照片却问题很多，让人很怀疑这名气是怎么来的。难不成是靠卖便宜货靠薄利多销？ 总结起来，有一些问题是技术方面的，一些是审美角度方面的，不过归根结底问题是理念上的。有一些照片差到让我们觉得照片敢发到网络之上任人评说本身就很能 说明问题，到了这里我们就不太好从技术角度评判，只是感叹其勇气之可嘉了。

　　小萌不能忍受那些糟糕的相片，和我不能忍受糟糕的网站一样。在看到一幅构图、配色或者化妆造型失败的结婚照的时候，她都会表现得义愤填膺，设身处地地 把自己当成了受害者控诉影楼差劲儿的技术。就如同我不喜欢一些设计糟糕的网站一样，无论人气多高，都不能掩盖糟糕的设计的事实。所以我不喜欢“战斗在法 国”这样在旅法华人之间有极高知名度的网站，并不是因为它不成功，而是因为它太成功了而给人的感觉却不是专业，而是乱糟糟。

　　我和小萌对自己领域都有追求完美的审视和需求，这也是为什么我们能够走到一起。很多人觉得这样较真并没什么意义，但我想我们的意义是通过否定而给与了 他们更客观的意见，让他们能够（但愿能够）意识到不足从而有所进步。这看起来残酷一些，但是却是个走向成功的法门。有些东西之所以牛逼，是环境逼出来的。 就像芭蕾舞和交响乐或者京剧——欣赏者的水平对这个行业的水平是有着指导作用的。所以我们不能忍受这些不完美，正是给完美创造了一个机会。再比如我们的足 球，正是因为太多的球迷的宽容，容忍了足球行业的从业者们糟糕透顶的行为和态度，才有了今天中国足球的局面。我并不是想说明中国足球没有腾飞错在球迷，但 我认为如果球迷更理性一些，更严酷一些，那么也可能给所谓中国足球的明天一个机会。不是吗？

2009-10-26 中国大使馆见闻 [星期一 巴黎 晴]

发表于 2009-10-26 更新于 2022-01-27 分类于 红桃 - 文章

因为小萌护照十二月要到期了，所以今天有机会陪同前往中国驻法兰西国大使馆办理护照延期。使馆很好找，就在香街。夫香街者，香谢丽舍大街是也。

　　我天朝大国气派之处就在于能够在巴黎最贵的地段买一套房子作为大使馆，然后找一个带红箍的老头在门口看门儿，红箍上写二字：“执勤”。

　　虽然作落在最繁华的街区，有钱买大房子，但钱毕竟还是有限的，这时候我天朝大国的公务员会想到纳税人的钱不是好收的，国家的钱更应当合理利用。于是他 们并没有把门口并不宽的街道也买下来，这导致的结果就是门口的队伍甩得老长，且是无秩序的长。无所谓，有了那么有中国特色的公务员看门大爷，还用担心什么 “有失国体”的问题呢！

　　进门之后就更像进了国门，有传达室，也有咨询台，有呈怒目金刚相的门卫。但没有清晰的指示，没有明确的引导，没有队形，没有顺序，有的是叽叽喳喳的大 人和叽叽喳喳的孩子。四下望去“收费”的大牌子倒是惹眼。跟着队伍盲目转了半天才发现一角落里有台机器，上写：领号。于是拿到一张纸条，上中法文俱全，便 开始排队。

　　若说这国民素质问题之重要也许独自行走的时候你难于察觉，必是到了人物聚集之地，因为数量之变化个人成为了“群众”，你才会发现蔽国民之真实面目。虽 一墙之隔是他法兰西领土，但此地是为中国。终于没有语言障碍的人们，也如盛夏的伏天跋涉到家，可以尽可能的舒展，做自由之姿态了（其实咱们大家都是东规矩 的人，不过谁让这儿没贴着“禁止喧哗”、“禁止小孩追跑打闹”的牌子呢？）。但我想这是上行下效的问题，国民是没有错的，看到诸番景象你会想到那些大部头 的问题：保险、就业、医疗、养老、教育，等等等等。你会感到“我们”和“他们”的差距之巨大，甚至在今生都难以企及，“发达国家”似乎就是他们造给自己用 来鄙视所谓“发展中国家”的，但其间的区别有目共睹，让你也不禁自愧汗颜。一个简单的例子：都穿黑丝袜，人家外国妞怎么就那么顺眼呢！

　　终于排到了自己，办事的姑娘态度还是很不错的，张口闭口都是“您”，和颜悦色的。拿出准备好的材料，结果被告之护照最后一页是不需用复印的，而告诉我 们要复印的也是使馆里的咨询台的小姐。法国人一个人办事一个样的风格他们倒是学了个八九不离十，要不然就是这两人私底下结了梁子，故意搅乱办公秩序？我想 不会，看看旁边的几个工作人员，那手底下的活叫一个利落。毕竟，多干一个就多挣一分钱，多给国家创一分汇。说到这里，小萌又要大发雷霆：“在国内办个护照 才200块，这里居然要45欧（约450人民币），妈了个巴子的！创汇也没有这么创的啊！”虽然咬牙切齿，但毕竟是花了钱能办事儿，十几天就可以取证，不 像法国人办事，一个一年的居留证可以拖半年才给你办好。

　　“不过人家不收钱啊！”小萌精辟地分析道，“我宁愿多花点钱让他们早点给我发证件，要不多耽误事儿！”

　　三天后。

　　我收到了法国警局来的信，说我可以去取更新的身份证了。今年只用了不到两个月就可以去取了，没想到法国人的办事效率有了质的飞跃。刚要欢喜赞叹一下法 兰西这个民族还有指望的时候，忽然看到通知书上一行字写道：“取证时请交纳30欧元的居留证件税……”——看来效率不是白飞跃的，法国人学精明了，居然今 年开始向留学生更新居留证收费了！小萌啊，你真不该放那句狠话！

2009-10-25 邻居2 [星期日 巴黎 多云]

发表于 2009-10-25 更新于 2022-01-27 分类于 红桃 - 文章

　　我们住的屋子旁边住了另一对夫妇，他们都是博士。和曾经做邻居的小屁孩们相比，无论是从学问还是从生活素质上他们的表现都是上佳的。毕竟到了而立之年的他们 工作生活已经步入正轨，家里也是有老有小的了，自然做事情都比较靠谱。但其实最令我们佩服的不是他们的学问，而是他们在家闲不住的精神。

　　之前我们旁边房间里住的一个邻居是个学通信的留学生，85后。抛开此人诸多不良的生活习惯不谈，单说此人在家曾一度闭门不出，甚至和作为邻居的我们都 抬头不见低头也不件。曾有几天我们怀疑他出门了，结果次日清晨发现他门口多了一个待扔的垃圾袋，才知道此人其实哪里都没去，竟然无声无息地在我们身边活了 好几天。这对博士和他相比，简直是天壤之别。

　　博士自从成为我们邻居之后，他们两口子表现出对巴黎极大的热情。那时博士的老婆刚到巴黎，这种好奇心驱使的热情也是可以理解的。但他们能将这种好奇心 保持到今天实在是令人感到佩服。在这段时间里，他们每个周末都要出门，每个月的第一个周日巴黎的博物馆免费，他们自然就要去博物馆。而周六也要去其他景点 游览，当然这并不因为周一到周六他们休息，他们在巴黎的身份是博士，而博士是不需要上课的——但他们需要工作。这就是我们的邻居的充实的生活，一周工作五 天，周末还要到处走走。即使不去景点，也要去逛商场或者超市。

　　这种生活节奏是令人佩服的，他们也都是三十好几的人了，咋就这么有精神头呢？终于在一个星期天，博士的老婆病了，巴黎的秋天让她不得休息的身体感到不 适，感冒令她痛苦不堪。早上看到她的时候，问之：今天还出去么？她摇头道：今天可能不出去了，太累了，我觉得好久没有好好休息了。啊！——我们的博士夫妇 也知道累的！我们甚至有些好奇地呆在家里，等着“见证奇迹的时刻”——这是博士老婆来巴黎后的第一个呆在家里不出门的星期天！多么的不可思议！

　　下午四点，我们看到博士一行整装待发，似乎准备出行。我们好奇地问：你们不是不出门了么？这是去哪儿啊？博士老婆回答说：“我们去卢浮宫。哎呀我忘了，今天是十月份第一个礼拜天，博物馆免费的！”

2009-10-23 Mélancolie, La [星期五 巴黎 阴]

发表于 2009-10-23 更新于 2022-01-27 分类于 红桃 - 文章

　　我甚至还没有选好一幅图片，就像以往那样，告诉自己“看，这代表十月。”我怀念那些在山岗上的日子，掐指算来，已经是三年前的光景了。我害怕某一天我想到那些日子，看着手指愣住，一句话在头脑里回旋但难以启齿。那一句话是铁的事实，被时间长河清洗，清晰可见无法逃避。

　　我在老王的身上看到了这铁一样的影子，在罗小的身上也看到了，还有你、你，……还有你。在我身边所有的朋友和不是朋友的人的身上，这影子同光一样巨大无边。我变成了水缸中的一条鱼，看到一滴墨水从天而降，我将毫无躲避的可能融入这即将来临的黑暗。

　　我重新读那些喜欢的文字，思考着那些渗透着忧郁伤感的文字以及它们的作者们。他们的才华令我羡慕甚至嫉妒，我想某一天也拥有那样的文笔，随心所欲地抒 发而不是绞尽脑汁寻找辞藻做一项所谓“描写”的工作。这就像英俊的王子与性感的公主偶遇，二人欲火焚身——干柴烈火，无需多言，一切就那么自然而然地开始 了。但王子的英俊和公主的性感则需要长时间以及大量的准备工作，但这一切并不会写入剧情。我希望别人只看到我文字的高潮，因此觉得自己积累的还不够多，那 些真情还需要很长一段距离才能流露在我的字里行间。

　　18岁的时候还能写出万余字的小说，时间、地点、人物、起因、经过、结果——六要素清清楚楚、明明白白，虽然情节并不曲折，但也是中规中矩。25岁的 时候只能写一些散文，所谓散碎的文字而已。但这几年的趋势是文字愈加散碎，而神亦散了。我写着这些文字的时候却在想，等我老了，就告诉我的孙子：爷爷写的 那叫“诗”。但我孙子那句问话每当回想起来却都如同芒刺在背，“爷爷，你写的这是什么啊？”

　　后来我想到了对策，我可以这样回答：“孙子，爷爷不是在写，是在创作。”
再后来我想，要点是要憋足了劲说出“孙子”最好拖长一些尾音，造成“孙贼”的音效为最佳。
