发表于 2009-06-24 更新于 2022-01-27 分类于 红桃 - 文章

法国高考以及法国人看待中国高考

从《费加罗》报上看到的09BAC的专题里一篇文章而引发的思考。《Dix millions de Chinois
dans la bataille du bac》（在“大学战斗”中的一千万中国学生）,还有一些法国人的反应，对比起来比较有趣。

　　每年的这个时候正是国内高考结束，学生紧张填报志愿的时候。对于法国来说，放暑假之前的这段时间对很多人来说也很紧张。因为一年一度的大学入学考试(B.A.C.)也在这个时候到来了。在这个时候法国的高中生将会参加考试，争取考进理想的大学。

维基百科上对B.A.C.词条是这样定义的：
在法国，学士学位(B.A.C.)程度是至关重要的进入中高等教育的一关。这是一个必要条件而不是充分条件：部分高校，包括预备班，选择从大学生根据自己的标准。候选人主要是高中学生完成中学教育。这被认为是第一个大学学位。

　　但需要注意的是，所谓的法国的学士学位（BAC）相当于我们的高中毕业文凭。这是一个很容易和我们的教育体制混淆的概念，因为和国内不同的是，国内大 学经过４年学习如果成功毕业将会获得一个学士学位，这个学位并不以年份为标准；但法国的BAC文凭则有年份之分BAC +1至5不等，基本上申请硕士 （Master）以上高等教育需要BAC+3以上的文凭，这就意味着至少上过大学三年才能申请硕士学位，一些专业则需要BAC+4或者BAC+5的文凭。 但对于大多数中国大学毕业的学生来说，直接申请Master文凭上按标准应该不成问题，但实际上却并不那么简单。这种不同是由于教育理念和制度不同所导致 的，一来教学内容和理念不同，二来评定考核标准不同，第三加上语言问题，所以很多大学毕业的中国学生到法国来都面临着读语言预科的情况，有一些还需要专门 重新考核和相关的专业知识审核才有可能上专业。毕竟是在国外，入乡随俗是不可避免的，而且决定权和标准都是由人家来定，所以尽管在国内某些专业学习的内容 比法国同级别的学生要深很多，但还是需要降级入学，不能不说是很无奈的。

　　再说回来法国的“高考”BAC考试，从费加罗报网站上的信息显 示，这个考试从18日持续到24日根据专业不同考试科目略有不同。大学分类分为BAC S（科技类）Bac L（文学 类）Bac STG CGRH（）Bac STG CFE（）Bac ST2S（）Bac STI (genie civil, mecanique, energetique, materiaux) （机 械、能源以及材料等）Bac STI (genie electronique)（电子）Bac STI (genie optique)（眼 科）Bac STI (arts appliques) （实用艺术）Bac STL (biochimie)（生 化）Bac STL (physique de laboratoire) （实验物 理）Bac STL (chimie de laboratoire) （实验化学）以及Bac serie hotellerie。可以看出，专业分得 很细，因为考试结果是获得BAC文凭，所以从这点来看，性质和中国高考并不相同。因为高考属于入学考试，而BAC考试则是文凭考核。

今日夏至。

　　这个周日总算干了点正事儿。首先是把小花盆里的西红柿苗移栽到了大花盆里，顺便把剩下的一些向日葵种子种到了小花盆里。我发现在花盆里的植物发芽都很 快，也许是放在阳台上，阳光比较充足的缘故，而且再加上我精心呵护，长势都很不错。不过院子里的植物似乎就没那么茁壮，前一阵下地的向日葵和菊花一直没有 发芽，我一直怀疑地里有虫子，今天还看到一只死去的蛞蝓的尸体，想来我的担心还是有根据的。为了保险起见，我还是决定在花盆里种西红柿。

2009-06-18 邻居 [星期四 巴黎 晴]

发表于 2009-06-18 更新于 2022-01-27 分类于 红桃 - 文章

　　我们住所的对面住的是一家法国夫妇，老两口。老头和我们的房东关系不错，于是爱屋及乌，老头也和他邻居的房客关系不错。我们搬到这里之后，打开窗户就 可以看到他家，是正对面，于是也经常能看到他进进出出，便也经常隔着街打招呼。不过鉴于法语水平有限，尽管有时候看到老头有千言万语在胸间，到头来也只还 是打招呼那几句。尽管如此，并没有影响中法两国之间的友谊，毕竟只隔着一条街，低头不见抬头见，正所谓一衣带水嘛。

　　事实上老头人很好，而且很热情。从相貌来看他并不是传统法国人，据我们猜测他可能是阿拉伯人，不过我们也没有证实过，但他也没有大胡子，为此这个问题 还有待商榷。不过老爷子虽然年纪不小了，但还是很勤劳的，总看他在干活，在自家后院敲敲打打。不仅喜欢做自己的事情，还喜欢给左邻右里帮点小忙。

　　去年秋天的时候。我们清理院子的时候，发现有几株玫瑰长疯了，于是打算把树枝锯下来。老头看到我们用小剪刀奋力切割树枝的时候就跑了过来，用他家专用 修剪树木枝叶的刀割下了那些花枝。最近一次是他帮我们收了家里寄来了包裹，为了表示感谢，我们送了他几个自己包的粽子，不过当时想了半天不知道粽子法语怎 么说，于是只好说这是“中国寿司”。

　　老头子总喜欢“三瓜换俩枣”，喜欢和邻居礼尚往来，以前有一次看到对面邻居一帮中国小孩把酒尽欢，就拿着瓶红酒跑过来与民同乐。据说前年还给我们之前 的房客送过他家自己栽的樱桃，为此老爷子美名传为佳话。这不，眼看着又到了水果纷纷收获的季节，昨天我们忽听见有人击扈，开门一看原来是邻居老爷爷，手里 拿着一棵大树枝，笑吟吟地说：“晚上好！”于是我也拿熟练地法语应对：“啊~您晚上好！”然后我就看着他手里的树枝──那是一棵樱桃树枝，上面挂满了红彤 彤的樱桃，可是，可是……这樱桃的法语是什么来得？我张口想说：“C’est….(这是……)”不过想了半天还是没结果，老头接了话说 道：“C’est cerise.”（这是樱 桃。）“Ah Oui, C’est ca! Merci beaucoup! c’est tres gentil!”（啊！对了，就是这个！太感谢 了！）

　　我的手头有一瓶酒，但是我不敢喝。它就那么安静地放在桌上，消解着我的欲望。我想我可以喝掉它，整瓶地喝掉。但我也许将以我的清醒为代价，换取这一刻的清爽，或者苦涩。但我终究是薄饮寡酒之人，想到啤酒花的味道便醉了一半。这一瓶酒岂不要了我的命？

　　溃疡：哪怕水一样柔软之物，也可如利刃一般，从口至喉，留下比辣更辣的感觉。那些体火所蔓延的末端，在潮湿的皮肉之上呈现出异样的色彩。和一切鲜活的生命一样，它们舞蹈，它们歌唱：燃烧吧，燃烧吧。

和王老师聊天。
看到王老师的海南照片。评说。生活感叹。解疑。沉默。

和罗老师聊天。
罗老师刚看了一部关于红色高棉的电影《S21》，于是展开讨论。之前我对“红色高棉”的了解还是从法国国际广播电台的中文网站上看到的相关消息。说今 年1月柬埔寨人民开展了对于红色高棉三十周年纪念活动。当时看了一些资料，但最后就记住一个词儿：杀人机器。看到这个词的惊讶程度不亚于当年第一次听说 “红色绞肉机”，何况两个都极端血腥的词语还是血亲呢。于是和罗老师从红色高棉说到极端GC主义，看来这是这种社会制度的产物（是否应该称之为毒物呢？） 无论初衷多么高尚，多么理想化，但终究是对声明残忍的杀戮。歪曲了“人人平等”之后的社会主义在一种人为的平等环境之下所造就的是对一切差异的打压和灭 绝。维基百科上的一段描述令人触目惊心：

红色高棉废除了货币，废除了私有制，废除了家庭，废除了婚姻。整个国家就是一个共产主义大家庭，所有产品都由国家统一分配。…当权者想尽利用诸多手段清洗不 纯洁的“新人”。其中有留法经历、会说法语、会写法语一律刑决（曾有西方留学经历者，最后只有15人幸存），其他知识分子无论是使用本国文字或者其他，也 难逃诸多形式的迫害。其中迫害对象甚至蔓延到一些戴眼镜的人。

　　这种独裁式的极端GC主义造成的结果就是种族灭绝的屠杀，因为到后来眼里不揉沙子的赤柬（柬埔寨共产党）已经达到了疯狂的地步，居然发明了一种自残式 的内部清洗方式——即使是党内高官也被逐步屠杀。其中对于废除家庭的做法令人颤栗，而为了提高工作效率而制造工具的行为也成了“资产阶级的腐蚀伎俩”我的 妈，这是赤裸裸的暴行啊。最后让我们看这样一个评定标准，就可以知道能在红色高棉统治下的柬埔寨幸存的人是多么牛逼了：“悲伤的情绪是对党的路线的不满，要被消灭；高兴则是小资产阶级的腐败堕落，也要被消灭……

　　大清洗，文革，红色高棉，到8964。GC主义真是可怕，而极端GC主义简直就是恐怖了。可悲的是在红色高棉的高压红色恐怖统治之下的柬埔寨人民居然 都不知道统治者是谁，只知道管理这个国家的是“安卡”，也就是柬埔寨语“组织”的意思。估计当年马克思老师研究共产主义的时候没有想到这些，多么高尚的理 想和抱负，到了人类手里就变成了屠害生灵的工具——意识形态真是一种可怕的东西，无形的东西可以致人于死地。真是“人类一思考，上帝就发笑”，还是罗老师 总结的好：“估计马克思思考的时候，上帝哭了。”

　　想来这此充满反动言论的对话或许被监听了，于是末了高喊几句“共产主义万岁”“全世界无产阶级万岁！”“人民万岁”……“人民币万岁！”说到人民币， 我又想起了《三峡好人》，真是个好电影啊！说到那个造型奇特的楼，罗老师还有幸亲眼目睹过，居然是个真家伙。三峡真是伤国本的大工程，当年人家李冰搞得好 好的，若要看到几千年之后的子孙们这么不道法自然地大动干戈，伤了国脉，变了风水，他老人家若有灵在天还不得气得活转过来！

后来在我的独家造型师Kandesn大人的笔下，我找到了答案。其实我的脸只比领导长一点、瘦一点；头发比领导长一点、黑一点；眉毛比领导细一点、挑一点；眼睛比领导小一点；鼻子比领导高一点、窄一点；嘴比领导小一点，嘴唇厚一点——仅此而已。
当这个问题解决了之后，另一个问题又来了：我到底是像哪一位国家领导人呢？

　　答案

　　由于Kandesn的工作需要，我奉命前往她学校当模特。她们学校要求学生完成以历史人物为主题的化妆作业，找一个历史人物不难，但找一个像历史人物 的模特就不容易了，何况身在巴黎朋友本就不多，找起来就更困难了。于是只能先从身边周围的人下手——我自然是第一个入围选手。经过一番寻找，最后 Kandesn决定让我作为她的模特，原因很简单：好用又便宜。

　　说实话做模特是个并不轻松的活，可没有照片上那么光彩。你需要做的虽然并不难，只是坐在那里，但时间长了便浑身不自在，也不能起来四处闲逛。而且面对 着化妆台上明晃晃的灯光，和镜子里自己太过熟悉的面容，疲惫和无聊会很快包围你，就像狂野上的风一样在你周身盘旋。何况我还是一个特别容易困的人，坐在那 里如果不干点什么肯定就会犯困，而且还是打哈气摇头外加了流口水的那种，外相十分的不雅。所以为了完成上面的那个工作，我还是做了一些准备的。比如带了一 大瓶咖啡，并且告诉造型师“我要是稍有困意，你就掐我，别怕我疼。”但我这职业道德并没有起到多大作用，工作期间几次摇头晃脑险些让Kandesn的工作 半途而废。不过经过几个小时的工作，走进摄影棚里的时候一看，还是很像那么回事的！

2009-06-10 中国与法国的差别之管中窥豹16 [星期三 巴黎 阵雨]

发表于 2009-06-10 更新于 2022-01-27 分类于 红桃 - 文章

我们这次从避孕套说起。


购买于家乐福的避孕套

　　我来巴黎之前曾经担心过未来的性生活难以得到保障，因为该国使用欧元货币，所有东西的价格等于我们的价格乘以十，这几乎就意味着我们消费的能力除以了十，而我并不想把我的性生活几率从小康降低到温饱，但我不能不考虑的一个问题是：避孕套是不是非常贵呢？

　　以我在国内的经验来看，避孕套的价格并不便宜。虽然有很多国内品牌诸如“男子汉”“杰士邦”，但我一般还都是慕名购买“杜蕾斯”。倒不是崇洋媚外，实 在是对其他品牌不大放心，弄出人命就不好了。不过事实上我并没有用遍所有的避孕套品牌，没有资格进行评判，但我的总体感觉是，在国内，便宜的避孕套用起来 感觉也就值那个价钱，怎么说呢？您总不愿意买避孕套的时候再花钱买一瓶润滑剂吧。所以后来我还是用杜蕾斯比较多，其实“杰士邦”或者“第六感”之类的也不 错，但另我惊奇的是有的时候别的品牌的价格却比杜蕾斯还贵！所以一番博弈之后，我还是倾向于杜蕾斯。

　　我出国之前那时杜蕾斯的价格是30～50元一盒不等，因为有不同的款式可以选择，比如各种香型，各种质感。对于一般上班族也还可以接受，不过如果换算 成欧元，乘以十倍的价格，那么可就比较吓人了。50块的一盒避孕套到了法国摇身一变身价倍增，这个价钱在国内可以做一次人工流产了。就拿法国人来说50欧 元也不是小数目，他们的钱包里都不一定有一张50的大钞，所以我得担心是必要的，至少在不了解情况之前。

　　不过到了法国一看，情况并没有像想的那么糟糕。因为在这里，留学生可不是最穷的阶层。我们不是常说么，世界上还有三分之二的劳苦大众还等着我们去解放 呢不是。所以这里也有便宜的避孕套。另一个比较令人振奋的消息是，杜蕾斯的价格也没有想象中贵。这里先插一句，法国一般中大型超市都可以购买到避孕套，你 完全可以闲庭信步地在货架旁浏览选择，当然如果你愿意，也可以到药店或者性用品商店购买。而上面的照片中就是我购买过的两款避孕套。左边的那个包装朴实的 避孕套是家乐福里最便宜的，右边的不用我说你也知道，至于价格6只装的杜蕾斯大概3欧多，也就是30多块人民币，5块多钱１个；而左边最便宜的12只装的 才1.8欧，18块人民币，相当于一个也就1块5人民币！你看看，这贫富差距一下就体现出来了。

　　其实说到这里我想说的是一个生活水平基础的问题。因为在我提心吊胆地使用过一次这最便宜的避孕套之后，我发现感觉还不错。正所谓，号大量又足，也不用另花钱买润滑剂。

　　从这里你可以看出一个区别，这个区别就是发达国家和发展中国家的区别。除了避孕套，每一个超市都会有自己的“1号产品”，这些产品都有特别的标志，但 都是“同类物品中最便宜的”意思。而这些产品质量虽然低，但并不伪劣，可以放心大胆的使用。而且据我观察，并不只是黑人、阿拉伯人和亚洲人购买这些1号产 品，也有很多法国人是这些产品的稳定用户群。所以，这里就体现出了这类产品在价格上的优势，在法国这样的国家，你可以在有正常供货渠道保证的超市买到这些 产品，而不用担心他们的质量问题。对比我国，虽然我们的物价低于法国，但我们即使购买的最高价格的产品的时候也会担心其质量问题。而且更可怕的是，就目前 来看，这些质量问题可能普遍存在于一些生活得必需品之中。我们如果购买最便宜的产品，付出的除了低廉的价格之外还有承担商品质量风险的可能性。

　　由此我得到了一个并不成熟的评判标准，这个标准并不以社会体制为基础，而是关注社会的发达程度。虽然我们传统教育对西方社会有一定的微词，但就普通大 众的生活水平而言，差距还是很大的。但其中的原因就比较复杂了，比如由于社会发展的程度不同，发达国家自然吸引了更好地技术和管理体制，比起作为发达国家 产品工厂的发展中国家，他们对产品有更大的选择权，他们可能最先得到新的商品和技术，从而保证了整个国家的生活水平。我在这里并不深入讨论原因和解决方案，只希望若干年后通过这个标准判定的差距能够更小。
