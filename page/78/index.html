2009-11-26 Cold [星期四 巴黎 阴雨]

发表于 2009-11-26 更新于 2022-01-27 分类于 红桃 - 文章

冷，非常冷。
我只有通过遍历路上的年轻女人来增加热量。

最近在看《六人行》，有感想一二，略记。
一、此片是94年开始拍的，距今一十有五年矣。十五年前的片子拿给现在的国人看还能赢得无数笑声，我总在想这不是他们在搞笑，而是我们在笑我们自己，笑两个社会的差距，且这差距远远不止十五年。

二、此片又名“老友记”，实际上比“六人行”更符合影片主题和英文原名。看此片让人深感朋友对于生活之重要，更具体点，是对生活之有趣的重要性。朋友 无疑是满足作为一个人交流的需求的重要组成，很难想象一个没有朋友的人的生活是丰富多彩的。“活在一个人的小世界”这种虚构出来的理想状态往往是无奈之 举，一个人很难坚强到在一个社会里自我孤立，这种去社会化的行为到后来很可能导致一个人的崩溃。如果需要一个例子，我能想到的是：海子。

三、我仍然对“朋友”这个词语产生了很多想法。这个代表人际关系的词语比起同类其他词汇更富有弹性，比起“亲戚”，朋友可以脱离血缘关系，比起“同 学、战友”，朋友也可以脱离共同经历的局限。这是一个可以达到很超脱的境界的关系，所谓“君子之交”“高山流水”，当然这同时也意味着，朋友关系的定义在 某种程度上更容易变得十分庸俗、贫乏和泛滥。

四、我审视我周围所有人，判断他们可否与我成为朋友。得到的结论是，在我的词典里“朋友”的定义级别很高，你们都不合格。

五、我希望能与我的朋友讨论，甚至争吵，除此之外就是心照不宣。一个眼神一个动作就彼此知晓，无需多言。这种默契令我痴迷。

2009-11-22 悲喜交加的一天 [星期天 巴黎 大风]

发表于 2009-11-22 更新于 2022-01-27 分类于 红桃 - 文章

今天是悲喜交加的一天，所以我想写下这些和大家分享。


Much clear,aha?

　　“早上八点钟醒来，外面仍旧是一片阴霾。”——你知道十月以后巴黎的天气就是这样，我甚至已经厌倦了总用“阴霾”这样的形容词开始一篇日记，但事实就是外面的天空——你只有抱怨的份儿却不能改变它。好吧，既然这是悲喜交加的一天，至少从一开始的天气上就已经点题了。

　　起床以后发现小萌已经去实习了，今天是星期天，她却要去实习，6点起床7点出发，这就是为什么我醒来之后屋子里就剩下了我一个人，并且我将有半天时间 见不到她。这意味着什么？意味着……我将半天见不到她。对，就是这样。于是我爬起身，按亮电脑屏幕，坐在电脑面前，像一个穆斯林醒来之后想到的第一个人就 是他的真主一样，我想着我的电脑。因为昨天我下午我重装了我的电脑系统，我要为我日后的工作重新安装一系列软件，让它看起来更像一台工作用的电脑——为此 我至少要给电脑换一套苹果机的图标。

　　事实上昨天晚上电脑系统就安装完了，但是这并不意味着工作结束。我还有驱动和软件没有安装，还有一系列繁琐的设置。尽管在安装winXP系统这件事情 上我基本上已经可以算得上一个二手的专家，但我仍然要强调“还有一系列繁琐的设置”！好吧，为了一个干净利落并且显得专业的电脑系统，我一起床就坐在了电 脑前——谁让小萌没有在家呢，要是她在，至少会有一个声音让我去先把牙刷了，否则这个动作将会不定期延迟。从坐在电脑前的那一刻起，我就开始感到郁闷，因 为我知道自己在做什么。我知道自己其实也不想这样，我理想的状态应该是“起床后喝杯水然后洗脸刷牙哼着歌曲去做早点”，但我看到外面的天气和空荡荡的屋子 之后就改变了想法，何况我还看见了昨天晚上剩下没有刷的碗。

　　两个小时后，我站起身，看着外面阴晴不定的天空。想到其实今天还有很多事情要做——至少在小萌回来之前得把被子叠起来并且刷刷牙什么的。对了，我还得 去买菜，偏偏这时刮起了大风。但电脑还没有装完软件，期间重启了若干次，Windows就是这么一个让人费心的东西——我只能通过不断重装而让自己在使用 它的时候感觉更好一些。我站起身，用很短的时间收拾完了屋子，吃了昨天剩下的半个棍儿面包，小萌还剩了半碗方便面，一顿早餐就有了。吃完饭已经快十一点 了，这时感到腰疼，这就是为什么我不喜欢一起床就坐在电脑前的原因。我找出护腰带上——我并不喜欢护腰，因为我腰太细了，这东西带在我身上很难说它到底护 了哪里。

　　穿上衣服拿了钱包出门，我需要在小萌回来之前去超市采购。走到地铁才发现竟然没有带车票，于是只好回家取，一路上用尽了毕生所学用各种语言抱怨了这倒 霉的一天，可惜地上都是落叶，哪里有狗屎啊，来吧，快出现一块让我踩上去！我顶着风回到家，拿了票，确认了没有落下任何东西，然后前往超市。超市位于13 区中国城，整个购买过程还算顺利。但整个路程令我痛苦不堪。因为我一直带着护腰，因为着急跑回家拿车票，结果到地铁的时候护腰从腰部跑到了肚子上，正好压 着我的肺下部和腹腔，于是我一路保持着低幅度高频率的呼吸，就像一个缺氧的人在大风里——靠，其实我就是缺氧，并且在大风里！

__记事__

10:00-11:00 吃了昨天剩下的半根Baguette，喝了剩汤。收拾了屋子，准备出门买东西。腰疼，带了护腰上街。
11:00 出门前往超市，外面大风，到了地铁发现没带地铁票（FML），于是回家取地铁票。坐在车上的时候护腰跑到了肚子上，让我一路都呼吸不好。（FML）

14:00-17:00 一边装软件一边看《老友记》，这事儿发展到后来就变成了“看《老友记》”。

　　到家之后解下护腰感觉这个星球的空气至少增加了两倍，我和我的肺叶都感到无比欢畅，我甚至在这样的情绪之下走进了厨房把碗刷了，然后放上锅给自己煎了 一个荷包蛋。这时候太阳居然从云端探出头来，妈的，刚才在路上的时候怎么一点太阳都没有！冲上一杯咖啡，就着昨天买的4欧多的科西嘉羊奶酪吃下荷包蛋三明 治，这个世界一下子变得美好了许多，一路上积攒的怨气随之消散。各位读者，我终于感觉快乐一些了，尽管小萌还没有回家，我的意思是，这难道不是另一次点题 么？

　　电脑终于装完了，干净利落。之前一切乱七八糟的软件都清空了，现在我宣布，这是一台工作用的电脑！唯一的娱乐只有扫雷！（和开心网……）。并且搞定了 本地服务器IIS，之前的系统里我的网站无法在自己的电脑里预览，IIS总是显示错误，这样就是我为什么要重装系统的重要原因！不过……后来发现，这个问 题似乎可以通过修改一个文件夹的权限属性来搞定，至于重装系统似乎有些小题大作了……好吧，不管怎么样，这也挺好，谁让这时悲喜交集的一天呢！于是我打开 小萌的电脑，开始看《老友记》，为什么要用她的电脑呢？我不是已经说了“这是一台工作的电脑了”么！

　　有两台电脑就是好！

债务美国 观后感

发表于 2009-11-19 更新于 2022-01-27 分类于 梅花 - 影视

实事扫盲，关键词：美国 金融危机 次贷危机 债务问题

　　我喜欢看这样一部纪录片，面对现实存在的问题，让当事人说话，展示问题所在，探讨根源，寻找解决方案。当然更多的时候一部影片能够做到的最多只是将问 题描述清楚，给予出诸多可能达到最终答案的途径和线索，直到影片结束也不能给予一个答案。到这里影片的目的就已经达到，人们意识到了问题所在——很多时候 这比解决问题的方法更加重要。

　　作为世界中心的美国，它的债务问题似乎并不值得我们关注和引起兴趣。但这部影片实际上透视出的原因并不仅仅针对美国，事实上当人们把“金融危机”挂在 嘴边作为很多重大问题的原因和说辞的时候，这部影片的价值就可以体现出来——你需要知道罪魁祸首么？那么就看一下这部影片，对经济、财政、税收、美联储、 次贷危机、房市、股市、政府预算以及金融危机有一个客观而全面的了解。

　　这个免费的教材的主讲人是美国总审计局的最高负责人，历数了美国历史上的债务危机以及债务的组成和影响。通观此片，你会发现问题是多么的严重和难以逆 转，当然对于我们这个生活在另一个国度的国民，看到这个历史上被我们无数次批判为资本帝国主义的魁首的敌国正走在行将崩溃的边缘的时候，也许心情会多一份 幸灾乐祸的畅快。影片的中后部分的重点转向了美国的贸易逆差和交易负债，提到了今天在贸易领域顺差排首位的国家：中国。但如果你仅仅看到今天的美国需要通 过谨慎地对待和中国、日本等国的贸易来维持本国的债务平衡的话，如果你仅仅欣喜于他们美国人自己都承认自己傻逼了的话，如果你仅仅满足于看到这些令美国全 国上下都震惊的数字的话，那么你需要警惕起来了，因为原因很明确但并不简单。

　　美国担心中国会变成未来的美国，像他们当年利用英国的债券牵制了英法两国获得了苏伊士运河权利的争夺战那样，经济上的威胁让他们非常恐慌。在经过了经 济危机和伊拉克战争之后的美国，赤字已经成了家常便饭，负债已经成了生活习惯。然而美国大众对这个问题的关注程度却被发达国家的生活水平所掩盖。完善的社 会保险和信贷制度曾成就了这个国家，但今天却成了问题的关键。这些问题是美国今天的问题，换句话说，也将是中国未来的问题。我们在饱尝近乎疯狂的经济增长 带来的富足感和基础设施的进步的同时，也将面临保险、信贷、股市和房市各种问题，承受作为一个经济中心将承受的各种风险。但我们应对风险的水平远远不及 GDP增长的速度，而另一个不幸是，我们的政府能否也如美国一样将这些问题摆在表面，告知所有国民即将面临的问题，通过各种途径寻找解决方案么？我们有理 由和经验对我们政府的行为能力表示怀疑，并应该对问题有一个客观的认知。或许我们的政府继续沉醉在高增长率上，用一副发达的表情掩盖不发达的体制所引起的 国民实际生活中所面对的痛苦？

2009-11-15 巴黎夜游记 [星期天 巴黎 晴朗]

发表于 2009-11-15 更新于 2022-01-27 分类于 红桃 - 文章

小萌的一个朋友邀请我们吃饭，于是我们就冒着周六晚上的寒风出门了。

　　X先生这个朋友是柬埔寨的华侨，来到巴黎已经有几十年了。已结婚生子，有车有房，生活安逸。据我的猜测，他和很多柬埔寨人一样，是当年红色高棉时期逃难来到法国的。经历过激情燃烧的奋斗岁月到今天终于稳定下来，开了店置了房子，可以享受安康的生活了。

　　他先开车带我们先去接了他女儿Sophie，因为他和他太太都工作，所以把孩子寄养在了一个朋友家。因为法国法律规定小孩子是不能单独在家的——无论 去哪里父母都要把孩子带在身边，《小鬼当家》是不大可能在法国拍的，不过由于法国已经算得上一个老龄化国家，很多赋闲在家的老头老太太独守空房，所以《老 鬼当家》还是可以一拍的。

　　Sophie小朋友看到爸爸来接她很高兴，于是我们从巴黎市郊驱车前往十三区的饭馆。据X先生介绍，在巴黎买车并不贵，一万多块欧元（10万左右人民 币）买一辆五六年的二手奔驰是很合算的。其实很多人来到巴黎都会觉得失望，看看路上并没有名车排队的景观——后来这些人才知道，该景观可以到一个名为“瑞 士”的地方看到。街上更多的还是雷诺、雪铁龙或者标致，支持国货嘛。车走在路上，一派秋风扫落叶的景象，从94省开到巴黎路上竟然很顺，周五晚上本来应该 车多人多。“经济危机影响的缘故啊——”X先生说，“前几年每到周末这条街根本开不动的。”不多久我们开到了13区，这里就是巴黎著名的中国城。虽然是中 国城，但并没有一个大门上写“ChinaTown”，只是这个区域住了很多华裔移民。事实上这里的中国人并不是最多的，真正的中国人聚集区塞纳河以北，也 就是传说中的美丽城(Belleville)。十三区的中国城有很多餐馆，我们要去一家名为Jardin d’Asie的餐馆，是东南亚风味的自助餐 (Resto de Libre Service)。

　　在饭馆看到了X太太，东北人，在巴黎也有十年了。这家饭馆的老板是X先生的朋友，也是金边人。一进门就看到了饭馆中央桌子上的各种美食，于是废话少 说，洗洗手马上进入状态。总的来说吃的还不错，吃到一些很怪的菜，属于东南亚风味儿大集合，柬埔寨、越南、泰国、日本什么的。菜品走的是海鲜水果轻口味路 线，基本没有过重过刺激的味道，这一点很受法国人欢迎。我们吃到了柬埔寨风味的鱼汤米线、日本三文鱼寿司、越南的蒸虾饺以及泰国的需要蘸特别的酱料才有味 道的大春卷——也就是说，春卷本身没有任何味道。自助餐的好处就在于你肯定能吃饱，并且很可能吃撑。比如这一餐，我问小萌：“你吃饱了么？”她说：“当然 了，撑死我了。我吃了好多冰淇凌呢！”——从这句回答你可以看出来，让人记住自助餐上的一道菜名还是很难的。

　　饭后百步走走看风景是个不错的主意，如果能坐在车上看风景，那就更不错了。X先生提议带我们看看巴黎夜景，我们自然欣然接受。从巴黎南边的中国城一路 北行，很快就到了塞纳河畔，两岸一派灯红酒绿，但人影有些稀疏，弥漫着一股萧条之意。远远看到GrandePalais挂着国旗的蓝色房顶，穿过塞纳河上 最漂亮的一座桥，随着路线一转，车开到了16区，埃菲尔铁塔进入视线的时候，我们眼前一亮——铁塔正在闪耀着金黄色的光芒。Sophie小朋友想下车看铁 塔，于是我们下车步行。沙约宫博物馆门口的空地上站了很多人，背对着一条繁华的街道，面对着更加繁华的巴黎市景。这里拥有相对较高的地势，可以看到万家灯 火的作为背景的宏伟的埃菲尔铁塔，塔身被金黄色的光笼罩，塔顶还有一道探照灯射向天空。这时忽然人群中一阵骚动和欢呼之声，抬眼望去，原来铁塔全身闪耀着 繁星般的灯光——这是为了庆祝埃菲尔铁塔修建120周年的特别礼物，每天晚八、九、十、十一点每隔5分钟都会有12分钟的绚丽灯光效果。所有人都被这灯光 带来绚烂的效果所震慑，这个时候你会被眼前这个巨大的建筑所感动，当然不只因为她的巨大。

　　站定观赏了十几分钟，在炫丽的光芒停止，铁塔重新被金黄的灯光笼罩的时候，我们驱车前往香榭丽舍大街。从著名的乔治五世大街上直行，路过五星级酒店乔 治五世饭店，前面就是凯旋门了。凯旋门坐落的广场称为星形广场，此地四通八达，乔治五世大街旁边就是那条著名的蒙恬大道 (Avenue de Montagne)这才是真正名品大牌汇集的街区。只可远观不可亵玩的气场贯穿其间，那些代表着“时尚”、“豪华”、“奢侈”的店 面毫无香街上那些哗众取宠的灯光，一派超大玻璃橱窗，极富质感的窗帘装饰，酷到不成的灯光，一水儿黑人大哥看门的保安。这一切都在无声地告知所有访客：您 花的不是钱，花的是眼光；您买的不是衣服，买的是品位。

　　和所有城市的第一街道一样，香街上永远一派车水马龙之相，不知是哪个阿拉伯世界国家的元首到访，街道两旁都挂上了法国和另一个国家的国旗——一如长安 街，一如北京。但令我们惊讶的是路旁有很多警车和警察，能看到街道上有许多阿尔及利亚的青年从车里探出身子挥舞着国旗，但警察并没有阻止，任凭他们欢呼雀 跃。X先生说也许今晚有游行示威，所以巴黎大规模布警。见前路堵塞便转头开往另一个方向，一脚油门便到了巴黎最有名的百货商场老佛爷百货了 （Lafayette）。老佛爷百货每年圣诞前都会布置橱窗，里面有非常漂亮的装置艺术品，虽然主题往往很卡通，但吸引的绝对不仅仅是小孩。旁边的春天百 货（le Printemps）也有很漂亮的橱窗装饰。下车步行，每个橱窗逐一观看，真是非常非常漂亮，虽然商场已经关门，但是橱窗还持续展览。透过商店 玻璃大门，还可以看到商场中心巨大的圣诞树，虽然窗外寒冷异常，但屋内一派暖意融融。这门里门外就是两个世界，一个无家可归的乞讨者坐在门口，他要踏进其 内又是何等漫长的距离…………我们能从中学到什么呢？这就是所谓资本主义的丑恶本质吧。但要注意的是：这里虽然丑恶，但是公平。
