碎碎记

发表于 2022-09-29 更新于 2022-10-12 分类于 红桃 - 文章

最近感觉画画还是很解压的。青青淘汰了一批奥特曼卡，我把卡片分了分，挑出怪物的卡片。可以拿奥特曼卡当画画练手的模特。相比起怪物，奥特曼的皮套子更贴近人体曲线。头部又有比较明确的几何造型。关键是画完了小猴还会过来看几眼，说，爸爸你为什么画这个奥特曼？——他不厉害啊。

现在家里有几盆花，其中一盆被我称之为“花中不死草”的绿萝是从公司救回来的——就是公司里淘汰了一批绿萝，放到地上准备让保洁处理掉。于是大家就纷纷拿回家养，结果在我家窗口下面经过了一个夏天，现在长势居然很好。虽然入夏的一段时间长了几片黄叶子，但我还是通过网络学习，掌握了啤酒养殖大法。就是将喝剩的啤酒兑水，擦拭叶片并浇灌，没有几天就长得特别好了。科学道理是啤酒里有大量二氧化碳，叶子能直接吸收。并且啤酒中养分丰富，喝不完浇花是最好不过了。听完我讲述后，青青经常偷着拿我兑的啤酒水擦他养的绿萝。

最近养的一盆是公司活动赠送的小杂花，里面有雏菊和牵牛花。最紧长势喜人。

八岁的第一天

发表于 2022-09-25 更新于 2022-09-29 分类于 红桃 - 文章

青青说八岁的第一天他要下载奥特曼游戏，因为下载游戏的界面上明确写着游戏的年龄是“8岁以上”。所谓八岁以上，那就是包括八岁啊。所以在生日前一天晚上就跟我说，爸爸，能给我下载一下奥特曼游戏吗，我明天就可以玩了。因为之前答应过他，并且对于他玩游戏，只要是保证用眼健康的基础上控制时间以外，唯一的要求就是他玩的游戏都要我们审核过，以免有什么少儿不宜的游戏对他产生不好的影响。于是我说，可以啊，但今天还不能玩，只能大概看一下。

于是打开Pad，下载，安装，顺利进入游戏，进入游戏封面就是几个帅气的奥特曼“嚯嚯嚯嚯”出场画面，登录界面上赫然写着游戏年龄“16+”…什么叫做无商不奸啊，可算给我逮着了，一方面我立马按照实事求是的原则，跟小猴说：青青你看，这个游戏明确写了，是16岁以上才可以玩哦。

看到货不对版的封面，青青喃喃自语，那他为什么下载之前说是八岁以上可以玩啊。嘿嘿，这就撞到你爸爸我的专业上了，这就是广告啊我的孩子，广告追求的是什么？不是美誉度，而是知名度，你看到了，被震惊了，记住了，它就成功了，你还点击了，下载了，安装了，注册了，它转化率提升了，完成KPI了，至于是否欺骗了人，他就不管了，你要删除游戏，随您便。反正他的目的达到了。几句阐述下来，青青说，这就是电信诈骗吧。

孩子，你理解的没错，这两个事儿大差不差的。

当然作为一个理性的父母，我还是把游戏下载了，并且用我的身份证登录进入了游戏——青青你看，这可是用爸爸的身份证才能进游戏玩的哦，咱们就看一下吧。结果发现这个游戏并不是简单的对打或者卡牌对战游戏，也不是植物僵尸那种塔防游戏。就是个网游嘛，别说孩子，这种类型的游戏我都没玩过。于是直接给结论——青青，这个游戏你不能玩。

大觉寺野采

发表于 2022-09-04 更新于 2022-09-06 分类于 红桃 - 文章

上次去大觉寺回家的路上小萌就说，等秋天咱们一定还得再来一次，到时候肯定有大货。

这不一转眼，秋天到了。

虽然周六下了半天雨，到了周日早上天还是有些云，让人担心中午会下雨，我们只好带上了雨伞——什么叫做未雨绸缪啊。我们不仅带了雨伞，还带了登山杖，登山杖，又叫打狗棒，因为之前走香山八大处的时候被野狗尾随出了心理阴影，所以这次我们有备而来。山，在哪里。我们来了。

从家打车前往大觉寺，路虽然不近但很顺，下了六环路走一会儿就到了。除了费用有些小贵（算上高速费100来元），但正如那些奢侈品广告说的一样——大觉寺除了贵，没别的问题。而且大觉寺是真有货啊，上次来顺利抓到几只鹿角花金龟，引来不少路人艳羡的目光。这次我们决定重走上次的路线，非常好，这个路线除了一段不是很难的爬升，到了消防通道就是一马平川了。

说一下最后的收获，这个季节是螳螂和蝗虫爆发的季节。我们也捉到了4只大螳螂，一只非常肥硕的蝗虫，棉蝗还是稻蝗我分不太清楚，但是真肥。而且我们还发现了大量的蜥蜴，很多刚出生的漂亮的蓝尾石龙子在墙上爬来爬去，享受冬眠之前的温暖和舒爽，还有足量的食物。以及足量的游人。
