2009-04-21 IT梦续 [星期二 巴黎 晴]

发表于 2009-04-21 更新于 2022-01-27 分类于 红桃 - 文章

继续发梦。

我的 Ubuntu 8.10 桌面

　　之前不久写过一篇《IT梦》，今天打算再写几笔，因为这几日已经在玩Ubuntu了！

　　其实对于操作系统这个东西，选Windows或者Linux其实是个有趣的博弈，二者都各有优缺点。如果你想拥有傻瓜式的操作，友好简单的界面以及完 善的硬件支持的话，Windows系统似乎是不二之选；不过如果衡量价格、安全性以及性能要求比较高的话，Linux就更胜一筹了。一开始接触Linux 的时候，最吸引我的是它能够在地配置的电脑上流畅的运行——这是在强调硬件的Win系系统里绝对的亮点。后来尝试了几个发行版Linux发现，事实上，所 谓美观和漂亮，Linux绝对不逊色于Windows。第一次看到Linux系统展示的3D桌面的时候，那种惊艳之感绝对不亚于第一次看到苹果漂亮的操作 界面——由此你能够深刻感觉到Windows是一个多么取巧的操作系统，它希望承袭（或者我们也可以略带贬义地说：抄袭）了Mac系统的用户界面风格以及 Linux的菜单特效。但无论如何，它无法抄袭这两个系统的精髓，因此，势必会造成今天这种三分天下的格局。

　　我感觉对于Mac、Linux和Win三个系统来说，一个比较恰当的比喻是三个台阶。Mac是拥有Unix内核的系统，对于如我一般的电脑用户接触或 者使用Unix系统的机会并不多，但是它的稳定性和商业应用上的占有率足以令电脑爱好者仰止。不过Mac的出现让我们和Unix走的更近了一些，更令从事 视频影像制作的IT人事奉为专业操作系统的不二之选。如果拥有Unix之心昂贵的Mac仍不能令你亲近的话，那么可以向下走一级台阶，投入Linux的怀 抱。因为Linux和Unix有很多相似的理念，承袭了Unix思想的Linux在开源的基础上扬起免费的大旗，成为所有不愿意花前去购买系统和升级的人 们的福音。在安全性上以绝对优势胜过了Ｗin系统。这是微软的软肋，也是Win系统与其他二者不能比拟之处。在限制了用户选择和使用的自由之后，下一步就 是被市场限制住自身发展的自由。但盖茨遇上了好时代，Windows系统也很大程度上满足了人们的需求，并在易用性和用户友好度方面做得不赖，所以现在的 情况就是这样，很多人站在第一级台阶上眺望，想向前一步，但已经被微软Win系统形成的习惯所束缚。

　　Linux不能普及原因有很多，因为开放源代码，所以任何人都可能并可以对这个系统进行再编译，创作出一个完全个性化的操作系统。据说世界上目前有超 过500个发行版的Linux，想象一下Window到今天才一共经历了多少次版本升级呢？不过从这500个系统中选择真正适合自己的却并不容易。何况直 到今天我才使用上了Ubuntu这个由南非IT高手们打造的发行版Linux，如果你对它并不了解，那么Google一下，你就会看到这样的介绍：“目前 世界上最流行的Linux操作系统。”而当你体验了Ubuntu的时候，我想信你和我一样有两个感觉，你会对自己说：“对！这就是我想要的操作系统!”然 后你又会接着说：“靠！我怎么现在才发现！”

　　这就是所谓一见如故和相见恨晚。

　　对于Ubuntu系统的特性我并不多说什么了，只有亲自体验一番才最为妥贴。不过即使通过它的名字，你就能感受到这个系统的气度非 凡：ubuntu（乌班图）是一个古非洲语，意思是：人道待人(对他人仁慈)。ubuntu 也意谓着：我们都是同源一体，这和开放源代码的精神似乎不谋 而合。在微软推出Windows7的时候，Ubuntu也即将退出最新版的9.04，其实从名字里我们就能看出二者的趋势，有人戏称Windows是“吻 到死”，而Ubuntu则是“有奔头”!

2009-04-20 听听听 [星期一 巴黎 晴]

发表于 2009-04-20 更新于 2022-01-27 分类于 红桃 - 文章

　　继续跟着金老爷《鹿鼎记》游历杜撰的历史，“通吃岛”最后居然改名“钓鱼岛”，韦小宝远征雅克萨，竟然参与了《尼布楚条约》的签订——虽然韦爵爷只签了个 “小”字，但还很切合实际地给罗刹人的纸张上写了个很伟岸的小字，至于其中含义确实泼皮无赖的想法罢了。虽然条约签订之前韦小宝学周瑜戏蒋干把罗刹将军玩 于掌股，不过我觉得很精彩的是韦小宝和罗刹俘虏投色子赌生死的那一段对话：

他又叫了一名罗刹兵出来。那兵倒也精细，先要问个明白：“按照中国规矩，这一次是点子大的赢，还是点子小的赢？”韦小宝道：“按照中国规矩，是中国人赢。中国人的点子大，就算大的赢；中国人点子小，就算小的赢。”

　　这近乎无赖的理论道出一个跨越了历史和文明的定理：“历史是强者所书写的。”正所谓赢者王侯败者寇，谁强大，谁就是规则的指定者和话语权的掌握者。在 康熙大帝和大清强大的军事保证前提下，在这样一种精神指引之下，韦小宝又完成了一次不可思议的任务——走进了历史事件的中心，看到这里我不免想到阿甘，不 过韦小宝更加主动——还送了自己的裸体雕像给罗刹女摄政王以缓解其相思之苦，可谓识大体顾大局，弘扬了我大邦的威仪。

__歌曲__

　　写东西的时候总想听些歌曲，不过听来听去却觉得自己电脑里那些歌曲不适合。为什么不适合呢，因为电脑里大都是摇滚歌曲，且大都是金属。这些歌有太强的 意识形态，无论是音乐还是歌词都太容易让人进入歌曲所塑造的氛围。对于需要某中心情或者不需要心情的写作活动来说，这些歌去不免是一种干扰。我琢磨了很长 时间，觉得应该听些轻音乐，比如卡农之类的似乎是更好的选择。但后来选了一些轻音乐，听了几天就放弃了。

　　也许应该听广播，让别人安排我的播放列表，一切随机，不由自己选择。这就是我开始使用Last.fm的开始。找一个自己喜欢的歌手，然后让网站自动推 荐类似的歌曲，然后就一首接一首自动听就可以了。配合上Ubuntu的媒体播放器Banshee可以很方便的连接到Last.fm进行歌曲的收听，也不必 再另行下载客户端软件了。不过目前的问题有两个，一个是必须联网——通过软件登录到last.fm；另一个是这些歌曲只能收听不能下载——不过我还是发现 了相关软件可以截取保存last.fm的流媒体并保存成mp3，比如FreeMusicZilla。

在家乐福买了一个新桌子，花了25欧。

猜猜桌子上盘子里是什么东西？

A、杨梅
B、荔枝
C、牛肉
D、蟹肉

答案：按Ctrl+A看答案：C、牛肉球

不知道写什么的时候就随便贴个图，然后发一个问题——有时候无关紧要的问题也可以起到吸引人的作用，这至少比堂而皇之的标题+空洞的内容来的诚恳一 些。比如之前不久看到一个博客，这个博客挑战了一个语文上的极限。你能想想到博客的最少字数限制是多少么？如果我告诉你这个博客的内容只有两个字的话，你 会不会表示惊讶呢？当然你可以说：这没什么，至少一个句子的成分够了，最简单的主谓结构并不必须有第三方的宾语到场，所以也还是可能的。不过我看的这个帖 子更气人，完全没有主语，只用了一个词，如果这两个字的词语拆开了看居然只有谓语和宾语——还是个省略主语的用法。所以看完这个帖子你就会明白一个道理： 有的帖子（博客）本身并无价值，或者说其价值仅仅是个标题而已——这就是标题党的所谓。而我看到的那个帖子做得更绝，它不过是把这个价值强化了一下，做了 一个简短而精辟的感叹，让人看了好奇心顿消只感觉那一下鼠标点的特别地冤枉。

　　其实这种感觉就像上面得哪个问题，实际上我把答案已经些在了帖子里（在引号的灰色区域），只不过用了灰色，你需要按住Ctrl+A全选就可以看到反白 的答案。不过对比我看到那个帖子，这个小把戏还是相对有趣的。一来我是本着互动的精神来做，而不是只对浏览者进行单方面挑逗；并且这个问题还是有一定趣味 性，至少图文并茂，而我看到的那个帖子表达的只是重复。当时看到我还想起了海明威写过得一句话：“毁灭——她说。”极少的字构成一种冲击力很强的语境，在 我面对这四个字的时候，好奇感和求知欲只能有增无减。相对于海明威，那个帖子就是个反例，那看起来很唬人的标题之下只有两个若不惊风的小字：如题。

　　好比一个人擦完了屁股后告诉别人说：“我擦完屁股了！”然后一指纸篓：“不信你自己看。”

另一个自毁长城的例子。在法国有很多中餐馆和日餐馆，老板大都是中国人（温州和潮州的老板很多）。很多餐馆招留学生打工，同样的不需要工作经验的杂工，中 国学生的工资4欧每小时，而日本学生的工资则是SIMC（法国最低工资标准）7欧多一小时。请注意，是同样的工作。

2009-04-16 我和丫 [星期四 巴黎 雨转多云]

发表于 2009-04-16 更新于 2022-01-27 分类于 红桃 - 文章

在我的记忆力有过几次牛逼的记忆，而在这些记忆当中，初中时代留给我的只 是一个名字，这便是我的母校汇文中学。

　　这所学校之所以牛逼，因为它是那个时候崇文区唯一一所市重点中学。记得当时上小学的时候，汇文中学巨大金字招牌在每一个崇文区小学生及其父母的心里闪 闪发光，每个家长都期望自己的子女能够考上这所学校以光宗耀祖光耀门楣。不过汇文的兴盛也从侧面反应了崇文区教育水平，相比起名校如云的海淀区，如我一样 的崇文学子无不都默默地承受着地理所带来的教学水平的不平等和师资力量的相对落后。整个崇文区只有一所市重点的窘境导致了学子们没有更多选择，我幸运地从 小学被保送到了汇文中学初中，而后又幸运地没有考上这个学校的高中，上了一个区重点高中。前一个幸运足以让我爹娘在他人面前脸上有光，后一个幸运则让我自 己体会两个重点中学之间的区别和差异。我有一个三段论的认知：作为那个时代的专业学生（也就是朝着上大学一门心思走到黑的如我一般的学生），最痛苦的并不 是没有学上，而是上了一个区重点中学。因为市重点的学生一般学得都很好，不担心考不上大学，他们的目标都是重点大学，即使学习不好的能上重点高中那家里必 定条件不差，也总是有出路的；而上一般的高中的学生毕业后目标大都是职高技校，运气好能上个大专或者三流大学，所以也无忧无虑；而最要命的就属我们这些上 区重点中学的学生了——既不能做到像普通高中学生那样放下一切虚妄率性而活，又没有硬实力跟市重点学生拼成绩。所以无论老师、学生还是家长，都在一种期望 与绝望的氛围中过活。不过幸运的是，经历了三流的小学二流的高中和一流的初中的教育经历后，你就会看出中国教育体制的问题所在，以及觉悟在这样一个体制下 接受教育是个多么艰辛的过程。

　　不过后来我发现，汇文中学虽好，但留给我的记忆并不多：我能记得曾经每个早晨我都骑车上学，路上会经过一条名为幸福大街的街道，我仍能回忆起清晨美好 的阳光，和煦的风照到一个少年的脸上。那时对我和这条大街的名字一样，幸福到令我忘记了一切细节，脑海中只隐约记得有高大的教学楼，巨大的操场、跑道还有 就是和我一样天天打球的学生。我记得当初在汇文的时候课余生活很丰富，上课也并不枯燥，三年的时光很快就度过了。而高中时代我感觉度日如年，老师教得和我 们学得都异常辛苦，考试成绩却越来越不理想。我以自己证明了我的三段论，不过经过一番挣扎我还是考上了个二流大学，也算给爹娘有个交代。我想汇文中学是我 学生生涯的黄金时代，属于一个甚至连梦想都未曾来得及建立的童年。

　　毕业之后我再没有回到过汇文，也不愿意在文字之中写什么。相比起高中和大学时代，我字里行间以汇文为题这似乎还是第一次。因为无意之间，在网上看到了 一张同学发的照片（就是你看到的），一下子将我内心深处的记忆的河流激起，顿生无限涟漪。这个正在表演快板书的孩子表情夸张，背后的拉花和黑板上的板报一 看就知道是一年一度的新春联欢会。而在背景的正中偏下的部分一个盒子上清晰地印着“汇文”两字——就这样，这个我并不熟识的少年和我产生了联系，而这个属 于十五年前的联系竟然直到今天才通过这样现代的方式让我重新获得，如同一个故事的大倒笔，不免让人回过神来之后一声感叹。

　　其实“校友”是一个很有趣的关系，这个以教学机构为基础的人际关系可以跨越性别、年龄、地域甚至时代，并且衍生出无限可能和很多有趣的效果。比如我清 楚的记得，我上初中的时候汇文曾经举办过一次规模很大的校庆——很多社会各界的名流和知名人士都到了。其中一个比较重要的活动是给汇文一个重要的校友彭雪 枫将军的雕像揭幕，为此，有很多无产阶级革命先辈也都纷纷到场，一些头发和胡子都白了的老爷爷在一群黑发小朋友们簇拥着走向教学楼，每个人脸上都洋溢着幸 福的笑容，正所谓黄发垂髫并怡然自乐。不过还有更多不知名的校友也都返校参加活动，其中有一个有趣的活动是让这些校友回到他们曾经的教室，以及一些知名校 友命名的教室里和小朋友们一起谈人生谈理想。于是你可以想象，彭雪枫将军曾经生活和战斗过得教室是怎样一副场景。不过经老师们介绍到场嘉宾里真正的名人其 实并不多，来宾们一张口就是：“我和那谁谁谁是同学，那时候我们……”原来他们和我们这些小辈们一样，和那些真正的同校名流的关系都是“校友”罢了。不过 这次聚会让我看到了人生的不同，从同一个校门走出去之后到底会变成什么样子谁也说不好。比如我可爱的大学竟能走出一位摇滚歌手确实令我感到很振奋。由此推 想，也许学校门口那个衣着破烂形容枯槁的看车老头也许也是我的校友呢，只不过没走上无产阶级道路，去混丐帮了而已。

　　最后回到这张照片，图片中的孩子已经长大成人，此人一对明眸，一副嘹亮的嗓子，一手王高李练得呱呱叫，相声快板两门爆。从2006年随着天桥德云社也 让广大老百姓熟知，越来越多的人知道了这个“北京丐帮少帮主”——对了，这就是我今天的发现，我和你们一样感到不可思议，感到世界真小。他叫李菁，我和丫 居然是校友！

　　——太刺激了！

我发现日下竟到了如此一个境地：端坐于屋宇之下，望窗外绿意盎然，满目春华，心中却 没有一丝激荡。看日月穿梭，光影流离，地上那些树影从斑驳被拉长，然后慢慢散去，我感到怅然若失，不过到底失掉了什么，却不得而知。这是人生的大可悲之 一，是众多不可知之中也许最可知的一部分。这其中包含了空虚、寂寥以及无所事事，心情在激动澎湃转向低沉无语。我清醒地感受动力的消逝，就像看到太阳西 斜，大地的温度随之慢慢消逝。如果写下一些什么之前需要某些情绪的话，我想，现在是时候了。

　　那天她问：“你想搞一夜情么？”我想了想，说：“现在不会想了。”然后补充道，“如果再过去，没有交朋友的时候，也许会……不，不会。”然后我又（虚 伪地）否定了自己，而后反问：“你呢？”她说：“我想啊。”我说：“你也许很想，但如果真的去做肯定会思考很久才做决定。”她点点头说：“也许我也跟你一 样，如果没交朋友肯定会搞一夜情。”其实搞不搞一夜情，从生理角度没有对错之分，对错的判定都是伦理和道德范畴的——如果从客观的生理需求来说，一夜情无 非是为了满足性欲。而由此衍生的矛盾和问题实际上都是把一夜情复杂化的结果，或者说，是失败的一夜情。在我看来，一夜情和婚前性行为都是可以接受的行为， 在安全合法的前提下，这些行为都是被理解的。何况在性启蒙和教育落后的中国，婚前性行为在我看来是婚后和谐的必要前提——这两个问题都曾是我和小萌饭后谈 论过并且达成共识的。

　　听《鹿鼎记》的时候一直在思考如何写听书笔记，这部小说看起来太庞大，不大好写。这种感觉在听射雕三部曲的时候也有，所以听完之后没有马上写。不过二 者相比，《鹿鼎记》更好写一些，故事虽然庞大，但是线索清晰，以康熙智擒鳌拜和平定三蕃为背景，二十四章经是一条线。现在听到了韦小宝奉康熙之命前往东海 剿灭神龙教，结果被擒，后来从鹿鼎山阴差阳错跑到了莫斯科。有趣的是书中所言那个时候俄罗斯被清人称为“罗刹国”，想来也挺有道理的，因为英语 The Russian读音跟“罗刹”着实很接近。另外对书中所代言的一个观点印象很深：韦小宝对康熙说：“汉人虽然骂满人，想反清复明的人很多；但更多 人骂吴三贵的人更多，在他们看来，汉奸比满人更可恨”——问题出来了：汉奸和侵略者谁更可恨呢？

　　在无聊的时候琢磨解决无聊的法门是一个很矛盾的事情，后来总结出一点：创作，或者阅读，或者思考。罗老师看后评论：“怎么没有女人呢？尹老师有点走王 老师的路线啊。”我于是补充上，同意女人的重要性。后来又想到一点，其实还有一个更简单的方法可以缓解无聊，那就是进食。于是在下午的某个时候从冰箱里拿 出一盒冰淇淋，和小萌吃将起来。这冰淇淋是家乐福牌的，口味名曰“Tiramisu(提拉米苏)”，提拉米苏（Tiramisu）是一种带咖啡酒味儿的意 大利蛋糕，由鲜奶油、可可粉、巧克力、面粉制成，最上面是薄薄的一层可可粉，下面是浓浓的奶油制品，而奶油中间是类似巧克力蛋糕般的慕司。吃到嘴里香、 滑、甜、腻，柔和中带有质感的变化，据说和卡布奇诺咖啡是绝配。我们虽然没有喝咖啡，虽然吃得不是蛋糕而是冰淇淋，但味道口感也相当不错，一勺入口，精神 为之一振，无聊之感顿消。以此证明了进食可以缓解无聊……以及嘴馋。

　　关于北京人，来到巴黎之后。和别人聊天的时候都会被问道是哪里人。答曰：“北京。”有很多人说：“你不像啊！”于是问：“何以见得呢？”答曰：“你没 有北京人那么重的口音。”然后我自言自语了一路，边说边听，确实说得儿话音和老北京的词儿不多。不过我同样为自己找到了说法，之所以不像北京人那样说北京 话，并不是母们不会说，而是为了让别人听得明白容易交流——就如同咱说英语，不是讲究伦敦腔；讲法语讲究巴黎音一样嘛——咱也得说普通话不是？

　　前两天发现了一部名为《Braid》的游戏，下载后一玩才发现这个小游戏真不简单。无论从画面和游戏性都相当有水平，看了一下相关介绍才知道这是一款 获奖游戏。开始玩的时候以为是一款普通的横版过关游戏，但后来玩了两关过不去了，看了攻略才发现这游戏是给高级玩家准备的，不仅需要有很高的横版过关类游 戏操作能力；还要有解密游戏的联想能力——看看攻略你就知道这个游戏到底有多难。这个游戏最吸引我的一点是音乐和画面，完全手绘风格的画面配合大气灵动的 背景音乐玩起来感觉相当好。

　　据说这是一款向此类游戏鼻祖任天堂《超级玛丽》致敬的作品，从故事和某些关卡的设计可以清楚的感觉到。但玩起来又是感觉大不相同。如果你想挑战自己，那么就尝试一下《Braid》。

由Jonathan Blow开发的小游戏《Braid》PC版延期到4月10日发售。千万别小看这款小游戏！这款游戏曾经在2006年IGF独立游戏节 （Independent Games Festival）获得过最佳设计的奖项，本作的Xbox360版已经于2008年8月6日在Xbox-Live 市场上市，获得GameSpot.com 的9.5分极高评价。

《Braid》是一款传统的2D卷轴式动作冒险游戏，游戏的剧情设定也是相当老套的“拯救公主”路线，但利用时间流动方式解迷的奇妙设定却是其与众不同的 最大亮点所在。在游戏的全部7个世界中，分布着各种希奇古怪的时空模式，玩家要充分掌握时间运行的规律配合灵活多变的思维才能顺利过关。
