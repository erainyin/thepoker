2008-12-20 中国与法国的差别之管中窥豹07 [星期六 巴黎 多云间晴]

发表于 2008-12-20 更新于 2022-01-27 分类于 红桃 - 文章

从圣诞说一说法国家庭。 拍摄于巴黎老佛爷百货 马上就要过圣诞了，对于法国人，圣诞节意味着什么呢？其实这个节日并没有我们想象中那么复杂。虽然在圣诞节前各个大商场、社区都会装饰一新，到处一派 节日气氛喜气洋洋。商家们也在同期大举促销，搞活动；各个媒体也在这个时候推出了与圣诞有关的节目，比如现在法国最有名的选秀节目《Star Academy》就会在圣诞节举行总决赛。无论大人孩子都期待着圣诞节的到来，这一切也都为圣诞节增添了更多的欢乐和色彩。但是真正圣诞节那天法国人会去干什么呢？ 答案有些令人失望——如果你问一个法国人，多半他会告诉你他们会在那天和家属在一起，吃吃喝喝，说说笑笑。对于国人来说，圣诞这个舶来的节日倒是个不 错的出行借口，约上三五好友，或者红颜知己出去high一晚，尤其是在圣诞前夜，大家一起努力把这个SlientNight搞得轰轰烈烈。但对于法国人 （甚至所有传统意义的法国家庭来说），圣诞夜是个万人空巷的时刻，至于巷为何空，那是因为大家都回家了。“圣诞节仍然是富有宗教意义的节日，人们仍然保持 着这个传统。在这一天，人们返回家里，和家人一起度过这一天。” 事实上，传统的法国人的家庭意识还是很重的，如果有人在圣诞那天不回家陪家人，是会被人鄙视的，即使现在法国人已经很开放家庭观念已经很淡漠，人们还是会有这样的想法的。圣诞之夜在街上游走或无家可归是悲惨的——即使是真的无家可归者，他们也可以到 爱心餐厅Resto du coeur 吃上一顿像样的免费圣诞晚餐的。 有一组05年的数据：法国目前有7.8%的单亲家庭，有超过50%的儿童是非婚生育，并且一般法国家庭都有2个或两个以上儿童，但这并不违法。有超过 1/3的人以Couple（二人生活）的形式生活，但这种形式并不单指以婚姻为基础的家庭。也包括了同居、 PACS（Le pacte civil de solidarité,这种形式起初为了同姓同居而建立，而目前有了更丰富的内涵）等形式。由于婚姻这个 概念对于法国人越来越淡薄，随着思想的开放以及对于人权、民主的认识的提高，法国人似乎比其他国家更能接受不同形式的家庭形态。目前来讲，当法国人说到家 庭Famille，更多的是指Couple，而非传统意义的一家三口或多口的家庭。 当然，由于越来越多的孩子是父母在未婚的时候所生，所以，如何处理这些孩子和家庭、社会之间的关系就变得越来越令人关注。对于我来说，一个家庭父母和 孩子形成了一个三角形的稳定状态，单亲家庭对于孩子必定会造成消极影响，导致孩子性格或者为人处事方面的一些方面有所改变。尊重人权的法国人更是处处把孩 子放在一个很高的位置——因为他们年幼、无知、缺乏判断和选择能力，所以他们是无辜的，保护他们成为了整个社会义不容辞的义务。 一般情况下，如果父母离婚，孩子必定由其一方抚养（一般情况下是母亲）。而另一方就需要承担一定的（甚至是大笔的）抚养费，并且保证定期和孩子见面， 照顾孩子。这样孩子就可以保持得到原有父母的关爱，不至于因为离婚这种家庭悲剧导致更恶劣的影响。说到这里又不得不说一下法国人的开放。由于越来越多的人 不再受婚姻的束缚，于是越来越多的人也把离婚看的很平常。在法国人看来，离婚甚至可以是不需要理由的——一个人如果和另一个人过腻了，提出离婚也不是非礼 的要求。在法国人看来（并非很传统的法国老人们），人们既有选择结婚的权利，那么选择离婚也是他们的权利，法国人常把一句 “Si tu veut, tu peut”（你想干，就可以干）挂在嘴边，对于结婚这件事，你可以对这点有个比较感性的认识了。如果还觉得不够感性，那 么推荐你看个电影《Je reste!》(我留下！)——你会惊讶于法国女性婚恋的自由和对男女关系的开放。 如果觉得电影离生活太远，那么就看看现在的法国总统萨克奇吧——上任之后就离婚，还坦坦荡荡地若无其事——这也就是在法国，换到别的国家早已经成为了 反对党攻击的把柄。而老萨总统宝座仍然坐得安稳，带着孩子和新第一夫人不亦乐乎。这种案例莫要说是在尚未如此开放的中国，就是放到已经很开放的美国，也定 会大书特书一番，弄一个身败名裂家破人亡的下场。而在法国，无论政坛还是民众都以一种达观的态度对待，对于法国人，这很正常，因为“这就是生活”嘛～～ 相关资料： 介绍PACS的官方网站：http://vosdroits.service-public.fr/particuliers/N144.xhtml?&n=Famille&l=N10

听书笔记 《鸳鸯刀》

发表于 2008-12-16 更新于 2022-01-27 分类于 红桃 - 文章

　　《鸳鸯刀》是一个笑话，并且这个笑话几乎适合任何一个所谓“江湖”。

　　传说有这么两把刀引起了江湖上轩然大波，各路豪杰都突然现身，搞得风生水起。于是也就有了一场多人参与的追逐混战。镖局向来是江湖人士的搞笑对象，尽 管他们也自称身在武林，却不如真正的豪侠来的潇洒倜傥，为钱而生，为钱而死。这一次更是着了江湖的道儿，引来了杀身之祸。最搞笑的是镖师周威信自以为深谙 江湖道理，左一个“江湖上有言道”，右一个“江湖上有言道”，说起江湖上的规矩头头是道，但动起手来却总用孙子兵法。在金老爷的书里，为朝廷卖命总是不得 好死的，纵使有一身武功也难逃江湖人士的谴责或诛杀，何况一个三分动手七分动口的镖头把式。此为笑点之一。

　　武林之中有过无数神仙眷侣，或侠骨柔肠，或情深万丈，而此书中的林玉龙任飞燕夫妇却大失风度，满面市井之色、满口泼皮直言。而恰恰是这二位却识得阴阳 双修的“夫妻刀法”。不过二人虽泼但也至情至性，不失得一口豪气。但这番气质也只能算得上是二三流的刀客，在这笑话里倒也恰如其分。此为笑点之二。

　　江湖上有言道：“三个臭皮匠，顶个诸葛亮”。而四个臭皮匠至少也能组成一个能叫出名字的小团体了。何况江湖帮派从来不计较人数多少，红花会十四个兄弟 已为数不少，但只要小弟卖力大哥高兴，也可以提拔上第十五把金交椅。而四人团体里，自少不了“太岳四侠”了。如果评选金老爷武侠小说中最搞笑武侠团体的 话，这太岳四侠必定荣登排行榜之列。不仅武功平平，且想象力异常丰富。尤其是做出了一系列很阿Q的举动楞好意思说是“行侠仗义”，由此四人的智商可见一 斑，还好他们自诩一个“侠”字，倒没有多少坏心眼儿。看在还能明辨是非的份上，他四人还不至于被和傻姑归为武侠智障一类。不过江湖上有言道：“傻人有傻 福。”书终还让四人闪耀了一下，捉到了奸贼卓天雄。此为笑点之三。

　　其实这部小说有个中规中矩的结局，英雄聚会，恶人被擒，英雄气短，儿女情长，并且澄清了历史真相，顺便鞭笞一下罪恶的封建统治者。而小说最后落笔于一个谜底，揭开了鸳鸯刀所藏的一个惊天秘密——其实也很俗套了，“得之者无敌于天下”的秘密。

　　“袁夫人将鸳鸯双刀拿在手中，叹道∶「满清皇帝听说这双刀之中，有一个能无敌於天下的大秘密，这果然不错，可是他便知道了这秘密，有能依著行麽？各位请看！」众人凑近看时，只见鸳刀的刀刃上刻著「仁者」，鸯刀上刻著「无敌」两字。
「仁者无敌」！这便是无敌於天下的大秘密。 ”

　　唉，还是那句江湖老话：“以德服人，以德服人啊……”

2008-12-16 阴霾的天空下 [星期二 巴黎 多云]

发表于 2008-12-16 更新于 2022-01-27 分类于 红桃 - 文章

看了自己以前写的两句话，觉得居然写的很酷。

一曰：“晚上睡觉，外面雷雨交加，我在梦里落荒而逃。”

另一曰：“我要在这下雨天描绘一个晴朗的故事，这太难了。仿佛凭空憋一个屁和强挤一个笑容——这不是能力问题，而是时间问题。”

这两句话是在一个阴雨天里写下的，那一天感觉事事不顺心烦意乱。 但无论如何，那一天的某段时间里，太阳还是探出了头。走在阳光之下，让人感觉心情舒畅了很多。我一直以为一个城市的天气决定了一个城市人的性格，像北京这 样一个北方城市、四季分明、风来则是狂风，雨到则是倾盆，一切痛痛快快，甚至简单粗暴。北方大陆性季风气候的天气培养了北京人爱憎分明，虽油滑而又不失局 气的特点。 而在巴黎，虽然夏天有着充足的长时间的日照，但一到冬天，这里便沿袭了北方寒流带来的冷酷。整整一周都看不到太阳，从早到晚都是灰色的天空，时不时会 有雨落下，或者是一些细碎的冰雪。但触地即化，丝毫形不成积雪。人们就在这一片潮湿低靡的气氛中行走着，打伞也不是，不打伞也不是，一切都被浇湿但又不够 淋漓。这样的天空下人们只能想到一些消极的词汇，如果将这些词汇汇总，形成一段文字，或者一段音乐，或者一幅图画，他们都有一个共同的名字：忧郁。 这不是我喜欢的风格，尽管我喜欢蒙克，但我喜欢他因为他的作品中贯穿了一种弥久的情感。但我所追寻的恰是相反感觉，就像站在烈日照耀下的麦田之中的文森特，不怕强烈，只怕不够强烈。

发表于 2008-12-16 更新于 2022-01-27 分类于 红桃 - 文章

咖啡

咖啡并不是提神的良品。我困的不成了，斜卧在沙发上。“瞧瞧你，又犯困了。”她说。“……”我心里说我也不想困，但就是困，就像我现在想说，就是张不开 嘴。“要不你喝点咖啡？”她说。“……”没用的，我心里说。吃完饭以后我就开始犯困。个中缘由我自己都能分析地清清楚楚——由于吃得太饱，导致饭后胃部剧 烈活动，于是血液向胃部集中，于是头脑因为缺血（或者是缺氧）就开始犯困。这是久病成医的哲学，在她看来不过一口空谈而已，还不如不说。“你要睡觉就先刷 牙去，别又这么睡着了！”她发出命令。

综艺节目

主持人向观众大吼一声：大家一起来！而后人们纷纷起立跟着唱了起来，有一个大哥表情甚是害羞，估计自知五音不全，完全是在对口型。他看到大炮一样的摄像机 从面前晃来晃去，这才意识到自己是在上电视节目，于是精神为之一震，不仅唱的声音大了起来还挥起了双手。摄像机在他面前停留了两秒钟，于是电视机前的观众 看到了一个十分尽兴的热心观众。这时大哥低下头，从兜里掏出手机看了一眼，是一条短信，上面写着：爸爸，我在电视上看见你了！大哥笑着把手机塞进兜里，再 抬起头显得自信了十倍。歌曲还没完，他更加尽力地挥舞了起来——摄像机又从那边转了过来。大哥似乎获得了启示一样，在摄像机对准自己的一刹那猛摇了几下 手，还做了一个飞吻的动作。几分钟后，手机又振动起来，大哥地低下头，怀着自我实现的兴奋感打开手机。又是一条短信：爸爸，你秋裤露出来了！

2008-12-10 不是一回事儿 [星期三 巴黎 晴间多云]

发表于 2008-12-10 更新于 2022-01-27 分类于 红桃 - 文章

今天上课的时候聊到了金融危机，经过老师一番解释才得知我们轻易地混淆了两个重要的概念。原来 经济危机 和 金融危机 不是一回事儿。 百度上面有这样的定义： 【金融危机】又称金融风暴，是指一个国家或几个国家与地区的全部或大部分金融指标(如：短期利率、货币资产、证券、房地产、土地价格、商业破产数和金 融机构倒闭数)的急剧、短暂和超周期的恶化。金融危机可以分为货币危机、债务危机、银行危机等类型。近年来的金融危机越来越呈现出某种混合形式的危机。 【经济危机】（Economic Crisis）指的是一个或多个国民经济或整个世界经济在一段比较长的时间内不断收缩（负的经济增长率）。是资本主义经济发展过程中周期爆发的生产过剩的危机。 有一个经典的例子可以说明经济危机，就是资本主义国家生产了过盛的牛奶，为了保持获得最大利益，奶商宁愿把牛奶倒进河里也不会降价销售——这样一来人 们便不能买到足够的牛奶了。但从表面来看，似乎是因为市场需求饱和了所以才有了很多“卖不出去的”牛奶，而实际上，牛奶卖不出去是因为人们买不起而非不需 要。由此可见，资本主义的生产过剩并不是绝对的过剩，而是相对的过剩。如果牛奶卖不出去那就会出现什么情况呢？自然就是少生产了，而事实上人们就会陷入一 个或长或短的时期内使生产和社会生活陷于严重的苦难和危机之中。这种危机的特征是生产严重不足，而资本主义的经济危机则是生产过剩。 相比经济危机，财政危机就难解释一些了。但可以通过米国金融危机形成来看一下：从美国次贷危机引起的华尔街风暴，现在已经演变为全球性的金融危机。大 体上说，可以划成三个阶段：一是债务危机，借了住房贷款人，不能按时还本付息引起的问题。第二个阶段是流动性的危机。这些金融机构由于债务危机导致的一些 有关金融机构不能够及时有一个足够的流动性对付债权人变现的要求。第三个阶段，信用危机。就是说，人们建立在信用基础上的金融活动产生怀疑，造成这样的危 机。 由此可见，经济危机是从资本主义基本矛盾中产生的，只要资本主义制度存在，危机就是不可避免的。举一个极端的例子，就是说经济危机对于封闭的计划经济 几乎是没有什么影响的，所以对于一个经济自给自足封闭的国家来说，甚至是可以避免经济危机的。而金融危机波及的范围则更加广泛，而且涉及的领域也更为全 面。尤其在当代，一个国家不可能没有对外贸易，不可能不考虑利率、资产、土地以及证券业务。而这些方面又不可能存在于一个相对独立的市场之中——必定会与 其他国家形成更广泛的市场。这就意味着金融危机具有更大的“传染性”，甚至可以说金融危机比经济危机可怕，而更可怕的是这两种危机往往先后同时发生，愈演 愈烈。 为什么说到经济呢，因为现在法国人、法国媒体、政府整天都在谈论La Crise Financiale（金融危机），对于资本主义世界的人们来说， 他们更清楚的了解并感受到了危机给他们的生活所带来的变化——这种感觉也许是像中国这样一个经济并未完全市场化的国民所无法体味的。其实也很好理解，问题 是体制的问题，我们没有使用相同的体制，所以也便没有遇到相同的问题，即使金融危机具有传染性，但影响也是相对小很多的，对于一些生产本地产品的行业和企 业，以及国家控制的行业更容易对危机作出积极的反应。而对于一个经济更加自由，更加市场化的国家，更加依赖于信用体系、依赖于证券的国家来说，危机所带来 的伤害是根本性的。 所以一个比较黑色幽默的事实是，如果一个国家在这样一个巨大的金融危机中并没有收到很大冲击，那么说明这个国家的金融体系还不够发达。

2008-12-09 生日快乐 [星期二 巴黎 雪]

发表于 2008-12-09 更新于 2022-01-27 分类于 红桃 - 文章

今天我过生日，很快乐。

白天如往常一样上班的上班，上学的上学。唯一不寻常的是巴黎下雪了，下得很急。巴黎的雪和北京的有些不同，也许是空气本身就很湿润，所以这里的雪片都不甚 大，但风又不小，吹落到地上便化为了雪水。于是天空中白色的雪花飞舞，而脚下则是一片湿漉漉的道路。上课的时候老师也感叹这雪下的如此之大，说2008年 的这第一场雪比以往时候来的早了一些。下课的时候，雪已经停了，空气里便变得异常寒冷和干净。 回家收拾屋子准备做饭，看了看冰箱，拿出来上周末买的鸡翅，这时顿感疲惫，不想做什么复杂的菜肴（实际是自己能力有限，容易把菜肴复杂化）最后想到炖 鸡翅最为简单不过，且味道差不到哪里去。小萌七点下课，到家大概要四十分钟，等到八点还没有回来。于是我开始做饭，这样她回来就可以吃现成的了。 正在这时小萌回来了，递给我一个袋子，同时说：“生日快乐！”我亲了亲她的脸，她进屋后忽然把门反锁，我站在门外不知她做什么。门开后，桌子上摆上了 她买的一个漂亮的水果蛋糕，还有一瓶很精致漂亮的酒。于是她去洗澡，我做饭，她洗完澡后，鸡翅也炖熟了，青菜也炒好了。可以开吃喽：））！端菜的时候却不 小心将一个杯子打碎了。我赶忙收拾，小萌说：“哎，你把杯子打碎啦？”我点点头，她却说：“今天把杯子打碎了，可真够巧的！”我不知道为什么“巧”，也许 是应了那句吉祥话“碎碎（岁岁）平安”吧。于是说了两句“岁岁平安，岁岁平安。” 蛋糕很好吃，酒很甜但入口辛辣，饭菜都很可口，最好的是她陪在我身边，真高兴！ 最近发现我吃完饭之后必定犯困，且困得摸不着炕头。今天很是高兴，又喝了两口甜酒，不多时就困的不成了，于是歪在了沙发上犯迷噔。后来被小萌叫起来， 说：“你要睡觉先把牙刷了再睡。”我这时已经渐入佳境，浑然不知，也想趁机以“无知觉”的状态哄过小萌直接钻被窝睡觉，于是歪歪扭扭地挪到了床上，嘴里说 着：“我…这就去……刷牙……”然后就睡着了，睡了大概十几分钟，看小萌没叫我，大概她看我睡着了不好意思再叫我起床刷牙了。我也不想起来，要不必定有一 番训斥要听。于是准备调整一下姿势，躺到枕头上睡去。正在这时，忽然头下一硌，感觉是压倒了被子下面什么硬东西上面。一惊之下头脑清醒了一半，赶忙翻身查 看，别是什么重要的东西。 翻开被子一看，竟然是一个棕色的纸袋。上面一条丝带。“啊！生日礼物！！！”我叫道，“是什么啊？”“你自己看”她笑着说。打开袋子，里面是一套咖啡 杯：一个碟子、一个杯子和一个勺子。白色的，上面有黑色小猫的图案很是漂亮。我以前也说过多次要有个咖啡杯就好了，今天这个愿望小萌帮我实现了！啊！—— 我这才恍然大悟，为什么她看到我打碎了杯子，说“真巧”了！ 不错！这杯子摔得好！摔得应景！ ___ 补发几张图片 蛋糕和红酒： 生日礼物：
